　　洪荒之中，人族不断降下功德，自然是惊动了洪荒中的无数大神通者，他们纷纷猜测，这人族到底做了什么，为何会有如此多功德降下？
<br/>　　他们之中，甚至有人想将注意打到人族身上。但，一想到人族的背后是女娲圣人，他们顿时就息了那个心思。
<br/>　　笑话，女娲那可是洪荒中第三位圣人啊，谁敢得罪？若是不小心惹恼了她，那时才是真正的悲催。
<br/>　　就这样，在女娲圣人的无形威慑下，人类并没有发生什么太大的灾难。数百年时间就此过去了，人族在缓慢发展着。
<br/>　　但实际上人族过得并不好，洪荒之中，危险实在是太多了，不管是天灾，还是人祸，他们都不能抵挡。人族天生羸弱，无法与自然或者其他种族对抗。也幸好如今其他种族还没有注意到人族身上，使得人族得以发展，若是有一天当其他种族发现了人族的巨大潜力后，定然不会放过人族。是以，增强人族的实力，势在必行。
<br/>　　但，人族中又没有什么传承留下，人族也不会修炼。其他种族的修炼方法也不适合人族，人族为此很苦恼。通过漫长时间的探索，也只是摸索出一些极为简单的修炼法门，这根本就是不堪入目。
<br/>　　当人族还在为修炼法门苦恼之际，在人族的一处隐蔽山洞之中，一个人类睁开眼睛，他满脸沧桑，模样为一青年模样，全身气血汩汩。
<br/>　　若是三大老祖见到他，就一定会发现，这人就是当初女娲创造出来的第一个人类。这人只有人族初生的那几年间出现过，后来几年就消失不见了。众人都以为他已经死了。实际上，他并没有死。这些年，他一直再为人族创法。而且，他还已经快要成功了。
<br/>　　此刻，他端坐在洞中，正在悟道。他心无旁骛，双手不断的结印，那是他心中所思所想的体现。
<br/>　　在他的腹部肚脐下三寸之地，一块混沌般的天地出现，这是他开启的人体一处秘境，他称之为——丹田。
<br/>　　丹田，犹如混沌一般，不见日月，灰蒙蒙一片，在人体腹部，是一处生命的源地，蕴含一切生命精华，孕育一切之所在。这里掌握着人的生死命脉。上接人首，下连人尾，乃是人体的黄金分割点，极为重要。
<br/>　　他的心头响起了诵经声，各种心法在他心头浮现。
<br/>　　“混沌生一，一为丹田。是一切的根本所在，是一切汇聚之场所，是最为基础的地方。”
<br/>　　“一生二，二为阴阳。万物都具有阴阳，只有阴阳平衡，人才能健康成长。若人体为阳，那么，灵魂元神则为阴。若武为阳，那么，文则为阴。阴阳不能打破。所以，欲修肉体，还需修元神。”
<br/>　　“二生三，三为天地人三才，对应着人体上中下三大丹田。是故，三大丹田也须开辟。”
<br/>　　“三生万物。人体就为一个小宇宙，万物自在其中，只需朝着人体潜力不断地去开发，便可能实现三生万物。”
<br/>　　……
<br/>　　“人体自身最为重要。”
<br/>　　最后，他得出这样一个结论。
<br/>　　“所以，其他的都可以不要，但人体自身一定不能放弃。是故，人族的修炼，则是修人。”
<br/>　　“而人体之中，肉体与灵魂最为重要，缺一不可。灵魂不是那么容易修炼的，只能以修炼肉体去慢慢滋补灵魂元神，慢慢壮大。所以，修炼第一境界是修炼人体肉身。”
<br/>　　他闭目不语，在不断锻炼自己的肉体，他的肉体之中，不断传来敲打的声音，一点一滴的杂质被锻炼了出来，使他的肉身变得更加纯粹完美。
<br/>　　就这样，一年过去了，他停了下来，皱眉不语。
<br/>　　“肉体锻炼得再强，也终究是有极限的，单纯锤炼肉体，还是不行，所以，还需要在体内积累‘气’。而丹田，则是温养这些‘气’的最好场所。但，丹田也需要开辟。而开辟丹田又只能用‘气’。”
<br/>　　“说到底，还是积累‘气’的问题。那么，这些‘气’又暂时该储存在哪里呢？”他在默默思索。
<br/>　　不久，他眼前一亮，道：“通道！它需要一条连接到丹田的通道。这些‘气’可以暂时储藏在通道之中。最后随着‘气’的增多，一举开辟丹田。”
<br/>　　“而这些‘通道’便是人体中的经脉。人体经脉无数，遍布人体各处，它就好似一张大网，将人体全身各处秘境连接起来。所以，锻体之后，需要开辟人体经脉，积累‘气’，开辟丹田。”
<br/>　　接着，他用了数年的时间，引来天地间的各种‘气’，用它们来开辟自己身上的经脉，最后一举开辟出人体第一个丹田。
<br/>　　“修炼之路，便是不断破开迷雾，去伪求真，所以，这些‘气’，可称之为‘真气’。”
<br/>　　“仅仅开辟经脉，还是不够，因为人体中，不止有经脉，还有五脏六腑，各处穴窍。所以——”
<br/>　　“接下来，需要锻炼人体的五脏六腑以及开辟各处穴窍。”
<br/>　　“五脏对应五行，万物生五气，五行相生相克，演化世间万象，实乃根基之元素。”
<br/>　　接着，他闭关数年，将天地五行之气引入五脏之中，与之对应，演绎出惊世的运转路线，丹田雄浑的真气汹涌。
<br/>　　“五脏有神！”
<br/>　　他轻叹，当他将自己的五脏开发到极致时，能够听见其中不由自主的发出各种诵经声。
<br/>　　“除了人体的元神外，或许这里也有人体的脏神！”
<br/>　　他继续在摸索，探究，研究人体的五脏六腑。后来，他开始开辟人体玄窍。人体之中，穴窍无数，对应着周天星辰，大大小小共有一千两百六十五个穴窍，它们藏在人体各处，经脉中、血肉中、骨骼中、元神中……其中，这些穴窍中还有百窍，一窍通百窍，是故，它实际上共有十二万九千六百个穴窍。共聚一元会之数。
<br/>　　花费数年时间，他一一将这些人体穴窍开辟出来，最终引入真气，淬炼这些穴窍，将它们一一扩大，这样，人体中，能够储藏真气的地方，就又增多了。
<br/>　　“真气终究是气体，气体再强，也是有限的，所以，之后便是要将这些真气液化、固化。所以，下一个境界，便是金丹境。”
<br/>　　又是数年时间过去了，他将自己体内的真气慢慢液化，在丹田与穴窍中，积累成一个个广阔的真气海洋。最后，他将这些真气之海，凝聚成一颗颗金丹。
<br/>　　就这样，他的身体各处穴窍中，出现一颗颗金光灿灿的金丹，使得他全身气势大涨，金光弥漫全身。
<br/>　　当然，他只是将一元会之数的穴窍中，最为关键的三百六十五个祖窍空间中的真气凝聚为金丹，而不是全部。若真是如此，那他的身体可就要被撑爆了。这还是他一直在不断淬炼着肉身的缘故，使得他的肉身强的得一塌糊涂。如若不然，就这三百六十五颗金丹，他的身体也会吃不消的。
<br/>　　此刻，在他的丹田之中，一颗紫金色的金丹在闪闪发光，他全身发出三百六十五道光源，周边无数灵气汇聚而来，形成灵气风暴，灌注到他的体内。他那三百六十五可金丹正与中间丹田中的那颗紫金丹遥相呼应，宛若群星闪耀。
<br/>　　良久，这些异象才停止下来。
<br/>　　“金丹境之后，又会是什么？”
