        苏漫雪再次对上傅夜沉的双眸，不禁不寒而栗。
<br/>
<br/>        她以为这个男人好控制，却没想到这傅夜沉，也有恶魔的一面。
<br/>
<br/>        忽然间，苏漫雪想到了什么，忍俊不禁地咧嘴道：“傅夜沉啊傅夜沉，哪怕我给你下了药，我对你来说，都还是有利用价值的吧！利用我来给童筱颖和傅昼景当幌子，你果然拿我一点办法也没有！哼，就冲着这点！傅夜沉，你也被我吃得死死的啊！”
<br/>
<br/>        “苏漫雪，你连自己都看不清，又岂会看透我？”傅夜沉不禁嗤之以鼻。
<br/>
<br/>        “你什么意思？”苏漫雪顿时有种不祥的预感。
<br/>
<br/>        傅夜沉伸手揪住苏漫雪的衣襟，将她连拽带拖，推出了自己的房间。
<br/>
<br/>        “我傅夜沉要女人，随手一抓就是一大把！我什么意思，你自己回去好好想想！”傅夜沉接着冷冷地说道，“哐”地一声，重重地关上了房门。
<br/>
<br/>        如果不是爷爷和母亲中意这个苏漫雪，又迫于敷衍筱颖，来隐藏自己对筱颖的爱意，他早就把这女人给甩得一干二净了。
<br/>
<br/>        说他渣也好，说他坏也好，总而言之，这都是她苏漫雪自找。
<br/>
<br/>        傅夜沉的话，让苏漫雪愣在房门口一脸懵。
<br/>
<br/>        其实，她如此蛇蝎心肠，他也未把她怎么样。
<br/>
<br/>        他一不爱她，二不碰她，还好吃好住地供着她，真的是为了拿她当幌子来蒙住童筱颖的眼吗？
<br/>
<br/>        苏漫雪凝眉冥思苦想，忽然间想明白了一件事情。
<br/>
<br/>        莫非……
<br/>
<br/>        他傅夜沉是真的打算让她当傅大少奶奶？
<br/>
<br/>        所以，她这学筱颖的声音，筱颖的姿色，还是学到位了！
<br/>
<br/>        苏漫雪再次曲解了傅夜沉的意思，狂妄自大还自以为是地洋洋得意起来。
<br/>
<br/>        楼下，童筱颖叫了出租车，但是迟迟未来。
<br/>
<br/>        刘婶见童筱颖一脸焦急，巴不得立马离开私宅的样子，于是上楼去把她的事情告知了傅夜沉。
<br/>
<br/>        司机小陈已经下班了，家里会开车的人除了苏漫雪就是傅夜沉了。
<br/>
<br/>        傅夜沉从楼上下来，苏漫雪也屁颠屁颠地跟在后面，只要是当着童筱颖的面前，她就立即上前挽着傅夜沉的胳膊。
<br/>
<br/>        童筱颖抬眸无意间在半空中与傅夜沉那深邃的目光对上，脑海里下意识地回想起起初他强迫她的画面，心里“咯”地一下，浑身起了鸡皮疙瘩，立即挪开了目光。
<br/>
<br/>        倘若是以前，他和她不知道彼此的身份，他把她当苏漫雪那般亲密，她还能谅解。
<br/>
<br/>        但是，刚刚那种事情，哪怕他被人下药了，他依旧把她当苏漫雪那般亲热，而她心里的那道坎过不去。
<br/>
<br/>        傅夜沉确实是一个又帅又俊的男人，哪怕近距离的去看他那张脸，都是完美得无可挑剔。
<br/>
<br/>        可是，傅夜沉再怎么帅，也终究是她老公傅昼景的亲哥哥，亦是她的大哥。
<br/>
<br/>        “我送你出去！”傅夜沉走到童筱颖的跟前，刚准备伸手去帮她提行李箱，却只见童筱颖聋拉着脑袋，手里拽着行李箱的拖杆，下意识地后退了一步。
<br/>
<br/>        她对他的抗拒，他能感受得到，心口隐隐作痛，但他对她更多的是无可奈何。
<br/>
<br/>        抬眸的一瞬间，傅夜沉的目光落在了童筱颖那白皙的脖颈上。
<br/>
<br/>        她的长发遮掩处，脖子上有一抹暗紫色的吻痕。
<br/>
<br/>        傅夜沉看得触目惊心，脑海里闪过她哭得梨花带雨，他却强行捂住她的嘴，继续强迫她的那个画面。
<br/>
<br/>        心口此刻恍若被一把匕首深深地扎入，有鲜血在往外冒。
<br/>
<br/>        “不用了，我已经叫了出租车。”童筱颖刚把这话一撂，她的手机便振动了起来，当她接听完电话后，脸色又惨白了几分。
<br/>
<br/>        出租车司机说堵车了，一时半会儿赶不到她那儿，所以撤单了。
<br/>
<br/>        苏漫雪见状，走到童筱颖的跟前，微笑着温柔地说道：“筱颖，我送你出去吧！”
<br/>
<br/>        “麻烦大嫂了！谢谢！”童筱颖一直低着头，不再看傅夜沉一眼，然后向苏漫雪微微欠身。
<br/>
<br/>        苏漫雪松开了傅夜沉的手臂，意味不明地睐了傅夜沉一眼，主动去牵童筱颖的手，微微一笑：“谢什么，咱们是一家人！”
<br/>
<br/>        一家人……
<br/>
<br/>        童筱颖柳眉微蹙。
<br/>
<br/>        “听说你这次是去傅昼景那儿，我觉得是好事。噢，对了，如果你们暂时不计划要孩子的话，记得带套噢！”苏漫雪像个长辈一样，一脸不害臊地说道。
<br/>
<br/>        话语一落，她还悄悄地瞄了傅夜沉一眼。
<br/>
<br/>        果然，傅夜沉的脸色极其难看。
<br/>
<br/>        苏漫雪不禁在心中嗤之以鼻。
<br/>
<br/>        这童筱颖再怎么干净，身子终究是要给傅昼景的。
