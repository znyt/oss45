        傅昼景也不知道自己为什么会担心他现在掌管的地利置业会回到大哥手中，按理来说，他不该有这样的想法才对。
<br/>
<br/>        地利置业，本就是大哥的啊！
<br/>
<br/>        这两年来，他为地利置业付出了很多，看着公司陪着自己一起慢慢地成长，扩展更多的业务，成就自己的业绩。
<br/>
<br/>        一时间，傅昼景心头有点乱，不得不停下手中的工作，走到办公室那边的落地窗前，俯视着窗外的景象发呆。
<br/>
<br/>        如果……
<br/>
<br/>        地利置业完完全全脱离了傅氏集团，该多好？
<br/>
<br/>        不该有的念头，就像一道灵光，闪过傅昼景的脑海。
<br/>
<br/>        而令傅昼景糟心的事情在这一天之内，是接踵而至。
<br/>
<br/>        今天股市开盘，他亏了，血本无归。
<br/>
<br/>        他大学毕业后，领到的第一份工资，就开始用于炒股。
<br/>
<br/>        以前，他都是赚，虽然赚得不大，但终归是赚了。然后，他把赚回来的又投了出去，现在不仅赚回来的没有了，就连自己的本钱投进去的只拿回来百分之五。
<br/>
<br/>        也就是说，他现在自己的“私房钱”仅剩五万，好在工资卡在筱颖手中，他现在每个月只能靠筱颖手里的那张自己的工资卡过日子了。
<br/>
<br/>        下午下班后，傅昼景一身疲惫地回到了自己的公寓里。
<br/>
<br/>        他开门进屋的时候，童筱颖正在厨房里做饭，并未发现他回来了。
<br/>
<br/>        傅昼景抬眸看了童筱颖一眼，连招呼都不打，就自顾自地进了卧室。
<br/>
<br/>        躺在床上，看着天花板上的吸顶灯，傅昼景的心情格外沉闷。
<br/>
<br/>        他和大哥的区别是什么？
<br/>
<br/>        区别就是，大哥不用奋斗，就有取之不尽用之不竭的钱。如今人脉扩及军政商三界，而且一上位就是“集团总裁”。
<br/>
<br/>        而他，为了能赚到钱，什么捷径都会去试一试。努力工作，奋力往上爬，顶多也就爬到一个子公司的“总经理”位置便打止了。
<br/>
<br/>        同是兄弟，区别为什么就这么大？
<br/>
<br/>        只因为，他大哥是“嫡出”，他是“庶出”。
<br/>
<br/>        自古嫡庶有别，更何况，他还是个不光彩的私生子，在傅家从小就不受爷爷的待见，只有大哥把他当亲人对待，但爷爷防着他，就跟防贼一样。
<br/>
<br/>        就连成年礼那年，爷爷给他们两兄弟选未婚妻，爷爷给大哥选那种非富即贵的女子。
<br/>
<br/>        而给他傅昼景选的是姚明娜这种小康家庭又毫无家世背景的女人。
<br/>
<br/>        只不过，那非富即贵的女子担心自己将来会守活寡，不愿意与大哥联姻，正巧大哥也不想要未婚妻，一心只想当兵保家卫国，所以，爷爷允许了，不再逼迫大哥的婚事。
<br/>
<br/>        而他傅昼景不想要未婚妻，爷爷表面上虽然准了，但却在事后特意安排姚明娜跟他念同一所大学，又在他们大学毕业后，将姚明娜安排在他身边工作。
<br/>
<br/>        爷爷的用意是什么，已经最明确不过了。
<br/>
<br/>        在爷爷心里，他就只配姚明娜那种平庸的女人吗？
<br/>
<br/>        或许，爷爷千算万算，都不会想到，他这种人，居然这么快就闪婚了。
<br/>
<br/>        傅昼景的嘴角勾起一抹意味不明的笑意，他这种人……是啊！他这种人，他是一种什么样的人呢？
<br/>
<br/>        他结婚的对象不是姚明娜那样的女人，也不是季思妍那样的女人，而是一个连家世背景都没有，同样是被亲人抛弃的同命相怜的女人。
<br/>
<br/>        或许，物以类聚，人以群分。
<br/>
<br/>        童筱颖做好了晚餐，将饭菜端上了桌，并摆好了筷子才发现傅昼景回来了。
<br/>
<br/>        她去了他的卧室，刚走到门口，见他横躺在床上，连鞋也未脱，于是走过去轻轻地替他把皮鞋脱了。
<br/>
<br/>        就在她伸手拿起一旁的毛毯准备盖在他身上的时候，他突然伸手，一把抓住了她的手腕，将她往床上一拉。
<br/>
<br/>        童筱颖踉跄了一下，膝盖磕到了床边，一个“扑棱”直接趴在了傅昼景的胸脯上，两人的鼻尖也因此碰到了一块。
<br/>
<br/>        两人四目相对，傅昼景的双瞳里映着童筱颖这娇俏的面容。
<br/>
<br/>        她不施粉黛，天生丽质，身上有的只有沐浴乳的香气，不同于外边那些胭脂俗粉。
<br/>
<br/>        在他眼里，她身上有孩子般的稚气，就像一朵花儿一样，需要他温暖地呵护，悉心地照料。
<br/>
<br/>        童筱颖与傅昼景对视了几秒后，双颊瞬间就浮泛着两抹红晕，心跳也跟着加快，在胸口里“咚咚咚”地直打小鼓。
<br/>
<br/>        以前，她和前男友何明旭在一起时，从未如此脸红心跳过。是昼景，给了她不一样的感觉。
<br/>
<br/>        或许，以前她对何明旭只是喜欢，现在对傅昼景才是真正的爱吗？
<br/>
<br/>        童筱颖从未去考虑过这个问题，她自己到底爱不爱傅昼景？
<br/>
<br/>        应该是爱吧！
<br/>
<br/>        因为，他是她的丈夫啊！
<br/>
<br/>        她必须得爱他的丈夫，爱这个家。
<br/>
<br/>        “筱颖，从今以后，你就是我的心灵支柱！”傅昼景抬起了手来，轻轻地按在童筱颖的头顶上宠溺地揉了揉。
<br/>
<br/>        童筱颖甜甜地一笑，将脑袋偏了偏，脸颊贴在了他的胸膛上，一边掩饰着自己的娇羞，一边聆听着他的心跳声。
<br/>
<br/>        她第一次发现，原来，一个人的心跳声，可以像歌声一样美。
<br/>
<br/>        “筱颖，我炒股亏本了……”傅昼景的手从童筱颖的头顶滑到了她的背脊上，他就这样一只手枕着脑袋，一只手轻轻地像抚摸一只宠物般抚摸这童筱颖的背脊。
<br/>
<br/>        童筱颖怔愣了一下，抬起头来，再次对上傅昼景的目光。
<br/>
<br/>        傅昼景一脸无奈：“所以，我近期没法和你办婚礼了。”
<br/>
<br/>        和筱颖领证的那天，他把自己所有的积蓄都给了筱颖，而筱颖则把钱拿给了童爸童妈替他俩还债了。
<br/>
<br/>        其实，那些钱也是他炒股赚来的。
<br/>
<br/>        只不过现在好了，他的身家资本就剩五万，筱颖手里他的那张工资卡上也就上个月发的两万块钱的工资，应该也被他在结婚之前用得差不多了。
<br/>
<br/>        所以，这几万块钱，过日子还是可以的，只是办婚礼可能就……
<br/>
<br/>        傅昼景心里是这么盘算的。
<br/>
<br/>        童筱颖一脸认真地看着傅昼景。
<br/>
<br/>        他以为，她会数落他。
<br/>
<br/>        但是，她只是微微一笑：“没关系。”
