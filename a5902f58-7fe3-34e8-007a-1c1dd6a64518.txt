        &lt;h3&gt;第三卷第一百六十章  财叔的担忧&lt;/h3&gt;
<br/>
<br/>        陈子昂看着张函离去，又哄着俩个兴奋的孩子睡下，她准备将邮箱里的文件处理一下，却发现财叔和春姨还坐在客厅里。
<br/>
<br/>        “春姨，财叔，你们俩怎么还不睡？”陈子昂看着俩个老人。
<br/>
<br/>        “子昂，你过来。”财叔叫陈子昂过去坐下。
<br/>
<br/>        陈子昂坐下后，财叔沉默了一会，叹了口气对陈子昂说：“子昂，我们以为你今天不回家，看着孩子舍不得张函走，就让张函进来了。”
<br/>
<br/>        “进来就进来了，又不是什么大事情。”陈子昂才知道春姨和财叔是因为让张函进了家门而感到不安。
<br/>
<br/>        “其实，在你进落家门不久，我们就知道你和张函之间的瓜葛了，落家想查一个人是很容易的。虽然前几年的信息没有现在发达，但是对落家来说都不是什么问题，这几年你也能知道落家是一个什么样的存在了。
<br/>
<br/>        说白了，落总还是看在张家和雪儿的面子收留了你，但是后期你自己的努力大家也是有目共睹的，雪儿这两年已经是落家总裁，据说明年就要结婚，现在的形势不明了，各种的竞争不断，子昂，你也得注意一些。”
<br/>
<br/>        “财叔，我知道了，雪儿真的要和吴家结婚了吗？”陈子昂忧心的问。
<br/>
<br/>        “是的，日子都已经确定了。子昂，我今天让张函进门也是有原因的，第一：今天天气冷，孩子感冒了没有办法出门，总不能不让张函不见孩子吧？第二：你们终究还是要见面的，以后孩子的事情会牵扯到方方面面。现在张函知道有孩子，不一定代表别人不知道你们有孩子。如果别人知道你们有了孩子，你要想好应对措施。”财叔忧虑的说。
<br/>
<br/>        “你是说张函的太太陆玉吗？”陈子昂还没有想到这个问题。
<br/>
<br/>        “是的，如果雪儿和吴家联姻，吴家会水涨船高，陆家也会跟着水涨船高。”财叔叹了口气。
<br/>
<br/>        陈子昂刚想开口说话，财叔接着开口：“别说你现在和张函没有关系，陆玉不会拿你怎么样，陆玉并不是你表面上见到的那么简单，其心智不是一般的女人能达到的，出手也狠厉，要不然年纪轻轻能到甸城去当副总？子昂，抽个时间好好和张函商量一下吧。”
<br/>
<br/>        陈子昂沉默着，财叔提醒了她，这几年她忙着做生意，现在公司马上要上市，以致于孩子的问题她从来都没有考虑过，那一次心软承认了孩子是张函的，本来就是一个错误，如果这件事情被人知道了，对张函对她都不好，她不是害怕丢人，而是现在到了这个份上，有些事情不能不考虑。
<br/>
<br/>        商场上的明争暗斗她见的多了，这么几年她小心翼翼，别人看着她有多风光，背地里就有多辛酸。如今生意是越做越大，想要考虑的问题也会越来越多。
<br/>
<br/>        财叔说的不错，陆玉她是见过的，女人的妒忌之心是很严重的，张函和陆玉结婚这么多年没有孩子，如果知道她和张函有了孩子会怎么样？
<br/>
<br/>        她不能让孩子有一点点闪失，而且她现在的工作也不能有一点点闪失，她的身后还有很多人，如果雪儿和吴家联姻，陆玉为难她，落家还会维护她吗？
<br/>
<br/>        落家不可能为了一个外人而去得罪陆家，张函也不会因为她而和陆玉决裂，还有张家有可能会要回孩子。
<br/>
<br/>        到那个时候，她陈子昂只怕是孤立无援了。
<br/>
<br/>        她该怎么办呢？她能去找张函商量吗？不，不可能！
<br/>
<br/>        “财叔，我想把孩子送出国。”陈子昂想了一会坚定的对财叔说。
<br/>
<br/>        财叔：“你不找张函商量吗？”
<br/>
<br/>        “张函也就是担着一个父亲的名义，从孩子出生到现在他什么都没有干过，我为什么要和他商量？”陈子昂并不想和张函有太多的纠葛，无论她曾经有多爱，那都是过去，何况现在的张函还是有妇之夫呢。
<br/>
<br/>        财叔：“你想好再说。当然也征求一下孩子的意见。”。
<br/>
<br/>        “团团和嘟嘟去国外语言沟通不是问题，他们现在已经会了多国语言，只是需要有人陪着和选择去哪个地方。财叔，等过年的时候我去找一下落叔叔，问一下落叔叔的意见。”
<br/>
<br/>        “你决定就行，我和你财叔都支持你，孩子我们也一直会带着，直到我们带不动了。”春姨开口了。
<br/>
<br/>        “谢谢春姨和财叔，子昂这一生都不知道该怎么报答您们。”陈子昂热泪盈眶，春姨和财叔就是她的再生父母，没有这俩个老人，她陈子昂也不会有今天。
<br/>
<br/>        “年后吧，这几天也快要过新年了，另外你不是要把耗材推向市场吗？而且你们公司最近要上市，孩子的事情你先不要担心，真要去国外了，就让你财叔来办理，你安心的工作。”春姨安慰着陈子昂。
<br/>
<br/>        “好的，就按照你说的办。张函见娃这么久了，也没有透露什么消息出去，他也是能够明白事理的吧。”陈子昂的语气透着不确定。
<br/>
<br/>        财叔：“张函是没有问题的，我觉得他也不会和你争什么的，但是难保别人有想法，我们也只是猜测，给你提个醒，毕竟孩子不是一个什么物件能藏起来的。”
<br/>
<br/>        “我对我身边的朋友在叮咛一下，让他们都不要说出去了，我的家人我也叮咛过了，知道这件事情的人也不是很多。”陈子昂思考了一下，孩子长这么大，也就是最近张函才发现，以前不是没有问题吗，年后送孩子去国外吧。
<br/>
<br/>        她得去找一下落叔叔，他们在国外有庄园，将孩子送过去，她也放心，只要他和张函不来往，只要她不激起和陆玉的矛盾，应该就没有什么问题。
<br/>
<br/>        财叔：“嗯，你也别太担心了，有事情我们一起承担。早点休息吧。”
<br/>
<br/>        “好的，那春姨财叔，晚安。”陈子昂还有一份文案要过一下，这件事情非常重要，下午才发给她的。
<br/>
<br/>        陈子昂并没有回卧室休息，而是去了书房，天大地大，目前的事情最大，在事情没有来临之前所有的担忧都无济于事，不如安下心来做好手头上的事情。
<br/>
<br/>        陈子昂最大的好处就是，无论面对着怎么样的困难，她都会坚信：车到山前必有路，过好当下才是最重要的，当然，她会有目的有计划的去规划她的人生。
<br/>
<br/>        对于不确定的因素，她会画上斜杆，提醒自己去注意，但是不会被困住。
<br/>
<br/>        财叔和春姨看着陈子昂回书房，都叹了口气。
<br/>
<br/>        他们两个人只有一个儿子，现在在落氏集团任一个部门经理，现在他们实实在在的当陈子昂是他们的女儿了。
<br/>
<br/>        看着陈子昂如此拼命的工作，他们都很心疼，但是也知道，这世界上没有什么事情是一劳永逸的，你不努力就会被淘汰。
<br/>
<br/>        一个没有背景的女人做事情就更是难上加难，前几次那么大的几个企业都轰然崩塌，何况是陈子昂这么小的企业，所以陈子昂现在可以说是如履薄冰，做事情是小心翼翼。
<br/>
<br/>        “我明天去找一下张函吧，子昂不愿意去，我去，我去探探张函的口气。”财叔对春姨说。
<br/>
<br/>        “行，现在你可以去找找他了，叮咛他注意点，别让人发现和我们有来往。毕竟张函是有家室之人。”春姨现在同意了财叔的做法，至于找张函会商量出什么结果，他们也不清楚。
