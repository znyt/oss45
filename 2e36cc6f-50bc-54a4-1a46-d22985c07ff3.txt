　　第354章
<br/>　　钟伟虎看着坐在自己面前拨弄茶杯盖的男人，一种异样的感觉从心头泛起，虽然他也并不明白发生了什么。
<br/>　　“钟大人既然来了，无论如何也是该让钟大人将人带走，可不巧，刚发生了一件小事。”和珅。
<br/>　　“什么事？”
<br/>　　“这个宋公子呐，玩心太重，玩得太开了，失手打伤我们一个弟兄，现在那个兄弟还躺在床上不能下地。”
<br/>　　钟伟虎当然明白，这是蛤蟆寨要开价了，对此，他也并不觉得意外。
<br/>　　“养伤需要多少钱，都由凉州一力承担，绝对不能让弟兄受委屈。这个，能不能让我先见一见公子？”
<br/>　　“好说好说啦。”和珅推着茶杯：“钟大人尝尝这茶，西北偏僻之地，怕茶没有长安的好……”
<br/>　　钟伟虎越是着急，和珅便越是不着急，双方你来我往，打着太极。如果是程大雷，怕早就不耐烦了，可和珅颇有种乐在其中的感觉。
<br/>　　留下和珅应付钟伟虎，程大雷窝在书房阅读浩甲城传来的消息。在江湖传言的加持下，帝国各地的消息都会通过酒楼汇到程大雷这里。
<br/>　　这是少数几件，程大雷没有交给和珅，亲自处理的事情。
<br/>　　一天天看下来，就连程大雷也不免为帝国如今的状况担忧。各地诸侯林立，对皇命阴奉阳违，活不下去的人占山为王，山贼林立。戎族那边也不太平，草原王一死，各部落形成的联盟形同虚设，大家都开始了各自抢地盘的过程。
<br/>　　西颜部向程大雷订购的兵器越来越多，这也变相证明了这一点。
<br/>　　“大当家……”
<br/>　　和珅敲门进来，不知不觉天已经暗下去，程大雷揉揉发酸的眉心问：
<br/>　　“人送走了？”
<br/>　　“送走了。”
<br/>　　“怎么谈的？”
<br/>　　“三天后，他们将赎金送过来，要一百匹骏马，三千长矛以及稻米、绸缎之类的。”
<br/>　　“才这点东西。”程大雷撇撇嘴，看来和珅还没有一个山贼的基本素养：“要少啦。”
<br/>　　“这只是第一笔，等他们把东西送过来，我们随便找个理由，继续把人扣着。”和珅露出奸商的笑容。
<br/>　　“咦，你的意思是人不放？”程大雷眼前一亮。
<br/>　　“当然不能放，这样的肥羊就是我们的护身符，他在一日我们就安全一日。”
<br/>　　这一点程大雷倒是没想到，他问道：“那你打算关多久？”
<br/>　　“能关多久关多久，两年三年，关不下去再说。”
<br/>　　“万一宋伯康打过来呢？”
<br/>　　“这就需要慢慢谈了，该扣押为软禁，时间拖得长了，给那宋伯康找个老婆，给薛问婷找个男人，过段日子把娃一生，人也就是咱们山寨的了。”
<br/>　　“啧啧。”程大雷不得不对和珅报以崇高的敬意，果不愧是满清第一奸臣，瞧人家这手段，如钝刀割肉，杀人于无形呢。
<br/>　　“给宋伯康找个女人好找，随便一个女奴都是可以的，关键是这薛问婷，已经有好几个兄弟去问过我了，我都没松口。要不大当家你把她收了房，日后您就是相府的孙女婿，相府不认你这个女婿，还能不认孩子么。”
<br/>　　程大雷怔了好长时间，坐在那里出神。
<br/>　　“大当家，大当家……”和珅连唤好几声：“怎么办，大当家你给个话。”
<br/>　　“咱们……”程大雷仰起头：“需要玩这么狠么？”
<br/>　　和珅一头雾水，摸摸脑袋：“这样做有什么不好么？”
<br/>　　程大雷无法理解和珅，和珅也无法理解程大雷。二人相隔咫尺，中间却如同隔了一道冰山，难以跨域。
<br/>　　“人还是留着吧，将他们换地方软禁起来。”程大雷。
<br/>　　“那就按大当家说的办。”和珅倒也是没说别的：“我马上就去处理。”
<br/>　　“等等，你正好过来了，我正有些事要与你商量。”
<br/>　　程大雷从书案上拿出一叠准备很久的东西，和珅看到后眼前一亮，道：“大当家，原来你这些日子也没闲着？”
<br/>　　“当然啦，我也日理万机的好不好。”
<br/>　　程大雷要与和珅聊的是山寨的组织结构问题，目前但琴川关住的人就超过一万，再加上蛤蟆岛的人，落玉寨的人，以及云中龙为首的十八家山寨，七七八八算下来，大概在两三万左右，不断有流民迁入琴川，对于这些人也不能放手不管。
<br/>　　如此，就需要一套严谨的制度，把所有人管理起来。程大雷已经给山寨订了军律，却也差不多相当于在一件破褂子上打个补丁。
<br/>　　如今的山寨需要一套完整的制度，一劳永逸，如此哪怕山寨日后有更多人，也可以容纳。
<br/>　　这不是件容易的事情，程大雷与和珅谈了很多，最后又将李行哉、李婉儿、刘悲、关鱼等人唤来。
<br/>　　这些人大概还算山寨里有脑子的，程大雷就很明智的没叫徐神机过来裹乱。
<br/>　　管理两三万人口，再加上以后会有更多人加入，并不是简单定几个职位，不顾是什长，队长、百夫长，或者经理总监之类的，都不能解决实际问题。
<br/>　　这件事商量来商量去，最后足足耗了大半个月。
<br/>　　现在的山寨分成两套班子，军事部和后勤部。军事部由关鱼总揽，下设团、营、队、班，每一级设一参谋，当然也可以叫做军师。后勤部有刘悲负责，和珅辅助，但事实上刘悲只负责人事方面的工作，而其他工匠、商业、教育等诸多问题，都是由和珅负责。
<br/>　　至于徐神机，他目前几乎和程大雷一样，看似什么都管，而事实上，什么都不管。
<br/>　　从上到下，拿出一套军法，当然也可以称作蛤蟆寨的草本宪法，约束所有人。哪怕是程大雷本身，犯法后也照罚不误。
<br/>　　程大雷亲自领着治安队在城里巡逻，对于几个酗酒闹事的、仗势欺人的狠狠教训了一通，保证了这套律法可以执行下去。
<br/>　　好在现在蛤蟆寨人数还少，军法又颁布得及时，还没人做下出格的事情。比如，若高飞豹醉酒后杀了山贼弟兄，你杀不杀都是个问题。
<br/>　　由程大雷主导，和珅、李行哉、刘悲等人参与，程大雷拿出了一套从上到下的制度。自然谈不上完美，甚至谈不上合格，却因为出现得及时，避免了以后可能发生的很多问题。
<br/>
