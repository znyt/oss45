　　风无心思考了一下，冷冷的开口道。
<br/>　　“输了，我就将我的心挖给你，我的心，乃是心中皇者，先天麒麟心，我若是输了，这麒麟心，便是送给你如何？”
<br/>　　“这赌注，足够了吧！”
<br/>　　麒麟心？
<br/>　　麒麟心，号称心中皇者，麒麟，本就是天地神兽，以人身，得麒麟心。
<br/>　　心脏，本就是人的根本，拥有这先天麒麟心，比起先天神体，都丝毫不差。
<br/>　　怪不得风无心居然会如此强大，他居然是有着先天麒麟心。
<br/>　　“这麒麟心虽好，但是我却不需要。”
<br/>　　楚风眠突然摇了摇头，开口道。
<br/>　　先天麒麟心，的确是好东西，但是楚风眠却并不需要。
<br/>　　他拥有战龙血脉，身躯之中的五脏六腑，都已经是向着远古战龙发展，这麒麟心，就算是给楚风眠，楚风眠也毫无意义，融入到身躯之中，只能是让楚风眠身躯中的两股力量相对。
<br/>　　这先天麒麟心，对别人是无价之宝，但是对楚风眠来说，却是一文不值。
<br/>　　“这赌斗，我需要的，只有你的剑术，你若是输了，便是将你的剑术，还是你的剑道领悟，原原本本的全部告诉我如何？”
<br/>　　楚风眠突然画风一转，开口道。
<br/>　　这赌斗，他绝不会败，居然如此，那倒不如正好要一点好处。
<br/>　　先天麒麟心，对楚风眠是没有任何意义，但是这风无心身上的剑术，对楚风眠可是有着大用。
<br/>　　楚风眠要凝练九域剑术，便是要从无数剑术之中，取得其中精华。
<br/>　　越是强大的剑术，对楚风眠的提升，便是更加有用。
<br/>　　这风无心的剑术，虽然还未展露出来，但是楚风眠已经是隐隐约约的感觉到了这风无心剑术的强大。
<br/>　　他的剑术，对楚风眠有着大用。
<br/>　　“你只要我的剑术？”
<br/>　　听到楚风眠的话，风无心有些吃惊的说道。
<br/>　　这个要求，对他来说不是太高了，而是太低了。
<br/>　　对一尊剑修来说，最为重要的，的确便是自己的剑术不错，但是实际上这剑术，就算是被其他剑修得到，也没有任何意义。
<br/>　　因为一般的剑修，就算是穷其一生，也无法将一种剑术，领悟透彻。
<br/>　　像是白萱那般学习万种剑术，万剑合一的，终究是少数。
<br/>　　一道剑术的价值在高，又如何可以跟祖龙至尊剑相比。
<br/>　　“楚风眠这小子是脑子抽了不成，这赌斗，居然只要风无心的剑术？”
<br/>　　“风无心的先天麒麟心，若是得到，比起先天神体都丝毫不差，此人居然都不要，反而是要他的剑术？”
<br/>　　“这小子真是傻了……”
<br/>　　一道道目光看向楚风眠的眼神，都如同看向一个傻子。
<br/>　　这简直是扔了西瓜捡芝麻的行为。
<br/>　　他们都有些无法理解楚风眠的行为。
<br/>　　风无心看了楚风眠一眼，缓缓的开口道。
<br/>　　“你说的，可是真的？你不后悔？”
<br/>　　“自然。”
<br/>　　楚风眠点了点头，大笑道。
<br/>　　“反正我也不会败，这赌约到底是什么，我自然不会后悔。”
<br/>　　楚风眠轻描淡写的开口道，言语中的声音，是说不出来的自信。
<br/>　　“你不是我的对手，我劝你还是现在乖乖认输吧。”
<br/>　　楚风眠的这一句话一出，从擂台上传出去，顿时全场哗然。
<br/>　　乖乖认输？让风无心？
<br/>　　风无心，可是北域年轻一代的第一人。
<br/>　　让他认输？
<br/>　　没有任何人，敢开这种口。
<br/>　　这已经不是狂妄了，是疯狂，简直就是疯狂一样，在这个时候，还敢来挑衅风无心。
<br/>　　“狂妄！”
<br/>　　听到楚风眠的话，风无心的脸色骤变。
<br/>　　“楚风眠，你不可能是我的对手，你的剑术，在我的面前，根本便是不值一提！”
<br/>　　风无心，也被楚风眠的话，如今直接激怒了。
<br/>　　他可是在北域百绝榜上，位列榜首第一位的人物。
<br/>　　楚风眠的展现出来的实力再强又如何？风无心的心中，从来没有想过，他这一战，会有任何的失败。
<br/>　　北域第一人。
<br/>　　从来不会失败。
<br/>　　“败吧！”
<br/>　　风无心手心动了，他的手心之中，猛然一把灵剑，显现出来。
<br/>　　这一把灵剑，极为细长，如同一道剑芒一般，但是其中蕴含的灵力，也是达到天级层次。
<br/>　　“楚风眠，你有着祖龙至尊剑，你就以为我没有天级灵器？你的剑就算是略胜于我，但是你的剑道，却是远远不可能跟我媲美！”
<br/>　　他手中的剑锋，突然动了，一剑斩下，快到极致。
<br/>　　周围的空间，都开始重重扭曲起来，这一道剑锋，直接是躲藏在了无数时空之中，令人一眼望去都察觉不到其中的存在。
<br/>　　好像是这一剑，距离楚风眠，有着万里之远，但是下一刻，刹那之间，这已经便是出现在了楚风眠的面前。
<br/>　　瞬息变化，快到惊人。
<br/>　　风无心的剑术，都已经是足以达到扭曲空间的地步，他的剑术，比起楚风眠这一世所遇到过的所有人，都要精妙。
<br/>　　拥有先天麒麟心，便是在资质方面，达到极致，就算是武道世家，也无法跟这风无心，媲美剑术。
<br/>　　他能够站在北域第一人的位置，所依靠的，正是他的剑术。
<br/>　　“好快的剑！”
<br/>　　“这风无心的随手一剑，都快到了极致，鬼神难防！”
<br/>　　下方观战众人，许多人甚至都无法看清，这风无心的剑锋。
<br/>　　这剑锋快到极致，一般人甚至就连看都看不清，更何谈抵挡。
<br/>　　北域第一人，名不虚传。
<br/>　　“这种剑术，风无心是没有丝毫留手，若是不赌斗，这楚风眠让风无心一时没有动用全力，倒是有可能侥幸击败他，但是现在，认真的风无心，不可能败。”
<br/>　　那夜天君看着擂台上的一幕，自言自语道。
<br/>　　“可惜了，此人败在了风无心的手中，不然的话，落在我的手中，我一定要亲手杀了他！”
<br/>　　不止是夜天君，周围的无数围观强者，都是摇了摇头。
<br/>　　这样一匹强势的黑马，本来还可以再进一步的，进入前十的。
<br/>　　但是如今一看，楚风眠的胜利，却是就要终结在这里了。
<br/>　　在风无心的手中，楚风眠绝不可能，闯过他这一关。
