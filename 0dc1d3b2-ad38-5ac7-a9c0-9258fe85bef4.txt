　　　　落子烟对涂小安就是一个谜一样的女人，很让人捉摸不透的女人。
<br/>　　她就好像是一个矛盾体，你根本搞不清楚她的立场在那里。
<br/>　　或许说这个女人并没有任何的立场，做事全靠心情。
<br/>　　涂小安不单单好奇她的身份，她的实力，更对她的所作所为很是蒙蔽，完全找不到轨迹。
<br/>　　本来他将金蚕蛊蛇作为第三阶段性进化的蛇种，但是看到落子烟跟对方已姐妹相称，关系好的两小无猜，他有点忌惮。
<br/>　　可谁想对方一句，我不管。
<br/>　　涂小安觉得自己还是少跟她接触接触，指不定哪天被她卖了，自己还替她数钱呢。
<br/>　　“你不要用这种异样的眼神看我，我今天出现，是为了救你”落子烟看着涂小安道。
<br/>　　“救我？”涂小安可不需要别人来救。
<br/>　　落子烟道：“你不要将天下妖魔当成了狗屎，纵然你有三阶的战斗力，可梨儿可是达到了伪妖王的地步，是那么好被打败的吗”
<br/>　　“况且，就算你打败了它又能怎样，白家高手如云，只要她将那片封闭的空间接触，白家的人出来，一人一口吐沫，都能淹死你这条皮皮蛇”
<br/>　　涂小安确实没有能真正打败那条金蚕蛊蛇的信心，另外白家的人他也考虑到了，要不是金蚕蛊蛇太自信，将空间封闭，他还不打呢，带着黄金瞳就走了。
<br/>　　一个人跟一个家族打，他吃饱了撑着吗。
<br/>　　另外，涂小安不由得好奇：“你为什么要帮我，你跟她有三百年的交情，又是姐妹相称，我们？我们也就三天左右的交情吧”
<br/>　　这三百年跟三天的交情，落子烟帮自己？
<br/>　　自己的魅力有那么大吗。
<br/>　　莫非她看上了自己？涂小安心中嘀咕了一声。
<br/>　　而落子烟没有回答涂小安这个问题，只是脸颊挂着淡淡的笑，任由涂小安胡思乱想。
<br/>　　半晌后。
<br/>　　“你脚下的猫不错，送我吧，全当这次我帮你的报酬”
<br/>　　落子烟此时将自己的美眸投放在了一只从白家离开，从头到尾都跟在涂小安脚下的猫身上。
<br/>　　黄金瞳听此话，一双阴阳眼骤然射在落子烟的身上，那双眼睛就是黑夜里的明珠，亮的闪耀。
<br/>　　落子烟笑了笑，一点都不怕这双特殊蕴含威势的眼睛，反而冲黄金瞳挤眉弄眼。
<br/>　　黄金瞳仿佛也意识到这个女人的不寻常，没有凶煞的扑过去教训她一顿。
<br/>　　“你想的美”涂小安一个白眼就丢过去。
<br/>　　“老娘是长的美”
<br/>　　“.....”
<br/>　　“是你多管闲事，我就算打不过，我也能安身而退”
<br/>　　涂小安大晚上的去白家折腾，就是为了黄金瞳，这娘们到好，张口就要，弄的大家很熟一样。
<br/>　　“我还有一些事情要办，你自便，我想你也不需要我送你回去”
<br/>　　见落子烟不怀好意，涂小安连忙闪人。
<br/>　　现在虽然是深更半夜了，让这样一位大美女走在无人的街上，算是一件很危险的事情，但涂小安一点不担心她。
<br/>　　那个不长眼的对这个女人起了歪心思，那么呵呵...祝你还能继续活着。
<br/>　　落子烟怔怔的看着涂小安领着一只猫撒腿就跑，顿时无语...
<br/>　　开开玩笑而已，还真怕自己抢他的猫啊。
<br/>　　离开落子烟之后，涂小安打了一个电话，然后站在了一个红绿灯的路口，良久之后，一辆轿车停了下来，他带着猫上车。
<br/>　　等车停下来的时候，只见苏羽站在门口迎接他，看到涂小安下车后，怀中抱着一只老虎皮的猫，惊讶道：“小安兄弟，你真是有本事呀，还真的从白家将这只猫给抢回来了”
<br/>　　看到黄金瞳，苏羽眼中难以自控的流露出贪婪之色。
<br/>　　这只猫哪怕花上数个亿，他都想得到。
<br/>　　可惜现在不是砸多少钱，就可以得到的。
<br/>　　“余子童呢，这猫想见他最后一面”
<br/>　　苏羽一愣，什么叫最后一面，余子童不是不用死了吗。
<br/>　　忽然，他懂了涂小安话中的意思。
<br/>　　这只王者猫，别说他不能惦记了，就连余子童这个亲主人也做到头了。
<br/>　　“嗷...”
<br/>　　只见黄金瞳忽然叫了一声，朝着里面的房子冲了进去，速度非常的快，只留下一个残影。
<br/>　　它感受到自己主人的气息了。
<br/>　　一张病床上，骨瘦如柴的余子童半靠在床头，眼中满是回忆之色。
<br/>　　回忆他跟一只猫的点点滴滴，那只猫刚出生就动如狡兔，将方圆一公里内的老鼠都给抓了一个遍，一个月后，就如一只猛兽可以上山狩猎了。
<br/>　　这只猫的出现，将余子童平凡的生活给彻底的改变，让他光鲜亮丽了起来。
<br/>　　他就如一位父亲，看着自己的孩子十月怀胎的出生，望子成龙的养它长大。
<br/>　　余子童以为自己可以靠着这只猫走上人生巅峰，没想到随着这只猫越来越强，给他带来的不是财富跟荣光，而是杀身之祸。
<br/>　　身体中养着一条可怕的蛇，将他的精元，寿命都吸食了，差一点，差一点他就死了。
<br/>　　捡回一条老命的余子童，知道那只猫不是自己能拥有的了，况且也被白家的人夺走，与他无关了。
<br/>　　但余子童还是充满了舍不得，毕竟，这只猫是他养到大的，岂能没有感情。
<br/>　　也不知道白家的祖奶奶，会怎么对待自己的猫。
<br/>　　嗷...
<br/>　　房内突然出现了一声类似的虎啸声，回忆中的余子童浑身一震，一道影子掠来，扑到了他的床边。
<br/>　　“宝贝，宝贝，你回来了啊”
<br/>　　余子童难以置信，颤抖的将一只皮包骨的手放到了猫的头上。
<br/>　　喵...
<br/>　　黄金瞳看着骨瘦如柴的主人，低下了高傲的头，仿佛在自责，没有保护好自己的主人，让他变成这般样子。
<br/>　　“回来就好，回来就好”
<br/>　　余子童激动的流出了眼泪，对着只猫还是非常的有感情。
<br/>　　黄金瞳此时不在是威风凛凛的样子，异常的乖巧，享受主人的抚摸。
<br/>　　一人一猫温馨的重逢了。
<br/>　　“余大师，你的猫，我给你带回来了，你可还记得自己跟我说的话”
<br/>　　涂小安走进了这件房间，打破了一人一猫的和谐。
<br/>　　余子童一愣，看着那位清秀少年，嘴角不禁有点苦涩，看了看已经被他抱在怀中的猫，唏嘘不已。
<br/>　　“谢谢你还能带着它回来让我见一眼，我已经很知足了”
<br/>　　黄金瞳落到白家的手中，余子童就不敢有非分之想了，还能在看到自己的猫，已经是他的福分了。
<br/>　　这只猫不能留在他的身边了，就算留，他也不敢要了。
<br/>　　要不起，一个一阶的修武者，拥有超等王者猫，麻烦会源源不断。
<br/>　　这点，他真的懂了。
