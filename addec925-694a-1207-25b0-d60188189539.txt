<br/>    燕山的大军出山，整个十万大山之中风声鹤唳，普通的妖王自然是看热闹，没有任何参与的心思，即便是陈明以一个大妖王身份发起这一场战争，去攻打数位大妖王这样必败的战役，他们都没有任何参与的心思，那毕竟是大妖王。
<br/>    
<br/>        根据几个月前燕山的实力来看，银翼有可能已经是大妖王，燕山分兵而出，七位大妖王将目光凝聚在银翼的身上，估摸着面对银翼的大妖王将会遇到一场艰苦的战斗。
<br/>    
<br/>        至于他们自己，根本一点担心的意思都没有，在他们的认知中，燕山也就两位大妖王，众多大妖王也是开始猜测燕山此行这么作死的目的何在。
<br/>    
<br/>        一家打七家，大家都是大妖王，这明摆着是输，作为一个大妖王应该做不出来。
<br/>    
<br/>        金毛吼道：“听说妖族大君逼迫燕山逼迫的有些急，燕山是不是狗急跳墙，没办法了，所以攻打我们表示他对大君的忠心，已经投靠到了大君那边了？”
<br/>    
<br/>        这个推测出来，不少人终于是明白了过来，“狮王说的有理啊！”
<br/>    
<br/>        “也唯有如此，才能解释燕山此举了！”
<br/>    
<br/>        “而且我听说，燕山之主的座驾，已经朝着妖族大君的妖城去了！”
<br/>    
<br/>        “看来真的如同狮王所推测的一般！”
<br/>    
<br/>        妖城之中，妖族大君身披一身雪白的狼皮，看向前方的阴月天狼，问道：“二弟，对于燕山此举，你怎么看？”
<br/>    
<br/>        阴月天狼道：“燕山应该是被我们逼急了，所以才选择投靠我们，而现在，陈明就是在表示他的诚意。”
<br/>    
<br/>        妖族大君四年片刻，然后问道：“既然燕山是来投诚的，那么依你之见，我们要不要派出兵去救援？”
<br/>    
<br/>        阴月天狼道：“只要燕山能够为我们所用，那我们距离主人可就近了一大步，要知道，封印主人的地方，就在燕山的地界。这样重要的地界，最好还是掌控在自己的手里。”
<br/>    
<br/>        妖族大君的眼中闪过一抹阴森的鬼气，“你说的很有道理，我绝不能让主人的封印之地落入其他人的手里。不过既然燕山已经选择投诚了，若是我们不管不顾，名声不太好，还是帮一把吧，既然燕山在进攻六位大妖王，剩下的一位大妖王，你带兵去灭了吧！”
<br/>    
<br/>        阴月天狼拱手道：“遵命！”
<br/>    
<br/>        阴月天狼继续问道：“我灭了燕山没有进攻的大妖王之后，需不需要救援燕山？”
<br/>    
<br/>        妖族大君挥手道：“把陈明救走吧，其他人，无伤大雅。此人精通阵法，说不定可以帮助我等破开大阵，解除主人的封印。”
<br/>    
<br/>        陈明一行人正在优哉游哉赶往妖族大君的妖城，陈明也不急，他想的是先把周围这几位大妖王全灭了，万一有一路兵马打不过，那么自己可以赶过去救援。
<br/>    
<br/>        正在这时，前方忽然有血衣剑卫禀报道：“有一个妖族大君派来的信使，想要见山主！他的姿态很是嚣张，直呼了山主的名字，我等是不是把他做了？”
<br/>    
<br/>        陈明挥手道：“很嚣张吗？带过来看看，我看看有多嚣张。”
<br/>    
<br/>        不过一会儿，一个顶着一个狼头的大妖来到陈明的面前，朝着四周围看了一眼，看向穿着一身青色道袍，手握拂尘的陈明，“你就是陈明？”
<br/>    
<br/>        陈明微微点头，“怎么了，有什么事吗？”
<br/>    
<br/>        大妖朝着四周围扫了一眼，目光落在紫霞仙子的身上，滋溜一声吸了吸流出来的口水，两腿抖着看向陈明，“放心吧，你想要投靠我们大君的意图，我们大君洞察秋毫，已经看出来了，所以派我过来，让你不必着急。”
<br/>    
<br/>        陈明：.......
<br/>    
<br/>        “我啥时候说的要投靠妖族大君了？”
<br/>    
<br/>        那狼头大妖笑道：“怎么，整个十万大山都知道你要投靠我们大君了啊，不会你们自己不知道吧？”
<br/>    
<br/>        陈明还没回答，就听到那个大妖继续道：“我知道，大君接受了你，原谅了你冒犯雾飞花公主，你很激动，现在可能因为太激动不能自已，不知道怎么招待我，没事，我教你，你把这个女人洗干净了丢到我床上，让我好好玩上三天三夜，我就原谅你了！”
<br/>    
<br/>        陈明对这个大妖投去一抹同情的目光，在场这么多人，你惹谁不好，偏偏惹上这个傲娇女王，真是不知道怎么死啊！
<br/>    
<br/>        紫霞仙子的胸口微微起伏，呼吸变重，显然也是气得不轻，她看向陈明，问道：“怎么，你没有下战书？”
<br/>    
<br/>        陈明疑惑道：“下战书到底是谁想出来的？我燕山开战，从来不下战书！”
<br/>    
<br/>        紫霞仙子看向那狼头大妖，然后道：“这个大妖，归我了！”
<br/>    
<br/>        陈明道：“你自己处理吧，玩死了就行。”
<br/>    
<br/>        紫霞仙子展露君威，吓得狼头大妖瞬间就吓趴下了，只见紫霞仙子提着狼头大妖的脖子走到了队伍的最后方，然后陈明就听到了一阵阵杀猪般的惨叫，足足一个时辰才停息。
<br/>    
<br/>        紫霞仙子终于是回到队伍中，道：“我们什么时候去进攻妖族大君？”
<br/>    
<br/>        陈明道：“不急，反正有我在，他也跑不了，而且，现在看起来，他根本没有意识到逃跑这个问题，就让他和其他的大妖王打一会儿吧。”
<br/>    
<br/>        十万大山，一处山谷之中。
<br/>    
<br/>        一位大妖王的麾下一队大妖正在山谷之间巡查，却见有着一片红云飘来，这些大妖抬头，看得到天空之中这么多的血衣剑卫，“该不会是遇上打劫了的吧？”
<br/>    
<br/>        领队的大妖看向逐渐向他们飞来的银翼，问道：“你们是要打劫吗？我们把所有的灵石都交出来！”
<br/>    
<br/>        银翼微笑着看向这一队大妖，认真道：“你说这荒山野岭的，要是遇到其他人，说不定还真是打劫的，你们还好遇到的是我，我可不是来打劫的。”
<br/>    
<br/>        一队大妖们拍着自己的胸口，道：“不是来打劫的就好，这我们就放心了。”
<br/>    
<br/>        银翼笑了笑，道：“是啊，遇到其他人，你们可能连裤子都被抢没了。”
<br/>    
<br/>        领队的大妖问道：“那不知这位妖王大哥，来这儿是干嘛的。”
<br/>    
<br/>        银翼笑道：“所以说嘛，你们遇到的还好是我，我是来杀人的。”
<br/>    
<br/>        银翼挥挥手，高声道：“降者不杀，抵抗者，杀无赦！”
<br/>
