　　林峰看了眼寡妇的皮裤长腿，非常性感，之前被妲己挑逗的一肚子邪火，林峰都没有对妲己做什么出格的事情，因为说实话，林峰他不怎么喜欢萝莉，妲己就属于萝莉那种类型的。
<br/>　　都说萝莉有三好，身娇腰柔易推倒。
<br/>　　林峰喜欢的是长腿美女。这个寡妇就很不错，这个腿这么长，还是极其诱惑的皮裤。
<br/>　　“走吧，我带你去老龙口，正好我也要去一次。”
<br/>　　林峰来到基地门口，看了眼任务，任务几乎是无穷无尽的，每天都一大堆任务。
<br/>　　“猎杀XXX区域的A级怪物，变异蜥蜴。任务奖励一百颗三阶晶石外加军需点。”
<br/>　　“猎杀XXX区域的A级怪物，变异狐狸，任务奖励一百颗三阶晶石外加军需点。”
<br/>　　A级怪物，在军部的说法里就是九阶怪物，这种怪物是绝大部分人都无法应付的，甚至可以说，若是遇到了，基本上就是遇到了死神，一个A级怪物可以把一个一百人全副武装的军队给灭掉，甚至遇到一些皮厚的，还可以顺带把一些坦克给摧毁了。
<br/>　　所以对于曙光基地来说，A级怪物绝对是心头大患。
<br/>　　“啧啧，这军部是个傻子吧，居然发布这么多九阶任务，现阶段，我们曙光基地这几万人里面九阶的不超过五个吧。”
<br/>　　“没错啊，算上那些藏虎卧龙之辈，顶多也就五个，这还是极其乐观的情况，搞不好压根也没那么多藏龙卧虎之辈，说不定就两个，一个军部兵王周华一个战士会会长寒鸦。”
<br/>　　“就是啊，要想灭掉这些A级怪物，一般人怎么可能做到，确切的来说，哪怕是周华他们亲自上也不见得能够干掉一头。”
<br/>　　许多人看到这些A级悬赏任务，都认为军部肯定是疯掉了。
<br/>　　然而下一刻他们看到了更为疯狂的事情，有个家伙，居然把这些任务全部接取了。
<br/>　　“又是他！”
<br/>　　一个人忽然指着林峰到。
<br/>　　“妈的，这家伙又来了。”
<br/>　　“我去，这一次他是急着去送死吗？这可是九阶任务啊。”
<br/>　　“兄弟，这人谁啊，你们难道都认识？”
<br/>　　其中有许多人都是老面孔了，但也有些新人，他们都不知道为什么这群人看到林峰这么大反应。
<br/>　　“说来话长，这个家伙每次接任务都是一大把的接取，就好像去菜市场买菜一样恨不得把一个月的菜都给买齐了，不过实力很强，最起码也是七阶的存在。我们很多人都叫他疯子。”
<br/>　　“我去，七阶？”
<br/>　　一群人都惊愕的看着林峰。
<br/>　　不过这一次，每个人都认为林峰太疯狂了，这简直就不符合常理了，就算林峰是七阶，退一万步来说，就算林峰是八阶或者就是九阶，那又如何，他还能够猎杀这么多A级怪物不成？
<br/>　　林峰却不理会其余人的目光，直接把所有斩杀九阶怪物的任务全部包揽下来了。
<br/>　　这个任务可以得到一百颗三阶晶石，不算少，但也不多，林峰接任务，自然不是冲这些晶石去的，而是冲着九阶怪物本身去的，九阶怪物本身，就是个巨大的财富，无论是他的晶石还是他的尸体。
<br/>　　至于军需点，这个是曙光基地新颁发出来的东西，说白了就是功勋点，做高级别任务可以得到功勋点，这玩意可以去军需官那里兑换很多东西。
<br/>　　比如说弹药、资源、石油、汽车，甚至是军衔等等都可以。所以说任务奖励还是很丰厚的，当然了，能够杀的了九阶怪物的，自然也不会特别看重这些奖励了。
<br/>　　林峰接取了任务后就带着小寡妇来到了老龙口。

<br/>　　林峰路上问。
<br/>　　“我名字叫刘壮壮。”寡妇诚实的回答，说到自己的真实姓名，她还居然有些难以启齿的害羞。
<br/>　　“额。刘壮壮？”
<br/>　　林峰蛋疼，这尼玛是女人的名字，我去，现在村里人也不给自己孩子去这种名字了吧。
<br/>　　“我还是叫你寡妇吧。”
<br/>　　林峰愕然，本来觉得寡妇寡妇的叫，有些感觉不对劲的样子，但不管怎么样也要比刘壮壮好听些，这刘壮壮一听就感觉是个陈勾那样的小胖墩。
<br/>　　带着寡妇来到基地。
<br/>　　李广这小子顿时眼睛就亮了。
<br/>　　甚至眼睛都移不开了。
<br/>　　“峰哥，这女的是。”
<br/>　　李广道。
<br/>　　“我一个朋友。”
<br/>　　“这么漂亮？”
<br/>　　李广嘿嘿一笑。
<br/>　　“美女，我叫李广。”
<br/>　　林峰道。
<br/>　　然而寡妇瞥了他一眼压根没搭理他，这让李广满脸的尴尬。
<br/>　　“她叫寡妇，当然，这是她的代号，以后也是我们的人了。”林峰道。
<br/>　　“我们的人？”
<br/>　　李广嘿嘿一笑，这岂不是代表他有机可乘了吗？
<br/>　　林峰怎么可能看不出来这小子什么鬼心思，那小子口水都要留下来了。
<br/>　　不过林峰没任何意见。
<br/>　　因为林峰喜欢的是苏云，这个寡妇的确好看，身材也好，但是毕竟这个女人之前要杀他林峰，而且对王家很忠心，如今这个样子也是因为被妲己控制精神罢了，所以林峰心里还是对这个女人没多大兴趣，林峰可以肯定一旦让妲己撤掉他的想法，那么这个寡妇肯定要想尽一切办法去弄死他林峰。

<br/>　　所以，这个寡妇，算是个潜在的威胁。
<br/>　　为了把这样一个潜在的敌人变成自己人，那就是让自己人和对方谈恋爱甚至生米煮成熟饭，结婚生子，这样就可以变成自己人了。

<br/>　　“大危险倒是没有，就是遇到了一些不知道从哪里冒出来的丧尸，不多，也就那么二十来只，全部被轻松干掉了。”陈勾摇头道。
<br/>　　“嗯，既然如此，等下我们去一次矿山，看看那个地方的具体情况，若是可以采出煤矿出来，那我们就算是家里有矿的存在了。”
<br/>　　林峰道。
<br/>　　无论是末世前还是末世后，这个矿石都是珍贵的资源，尤其是末世后了。
