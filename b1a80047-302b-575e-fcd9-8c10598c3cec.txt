　　叶少阳在四周转了一圈，本来想看看梧桐他们在不在，结果都没看到，于是转了一圈之后，跟道渊他们一起回去了。
<br/>　　道渊真人继续去打探消息，晚上回来的时候，带给叶少阳一份请帖，是梧桐和张晓寒婚礼的，两天之后，就在桃花山的大殿广场上举行，请各大门派参加……
<br/>　　“后天，在桃花山大殿举行……”叶少阳看着请帖，喃喃道：“故意在桃花山大殿举办婚宴，这是要立威啊。”
<br/>　　毛小方皱眉道：“明天攻打山门，后天举办婚宴，这要是打不下来呢，到时候影响婚宴，这不是让人笑话吗？”
<br/>　　叶少阳笑道：“所以说，他们是在立威，表示他们一定会在一天之内打下桃花山。到时候举办婚宴，又是在桃花山的大殿广场……简直威风到极点了。”
<br/>　　道渊真人道：“正是此理。我听说，法术公会已经派人下山去采购宴席的材料了，这么多吃食和桌椅板凳之类的，从山下运到山上，绝对是个大手笔。”
<br/>　　“他们要的就是这个效果。”叶少阳道。
<br/>　　毛小方看着喜帖上的梧桐的名字，皱眉道：“梧桐姑娘，真的嫁给张晓寒啊。”
<br/>　　“张晓寒不好吗？”
<br/>　　“当然，如此张狂自负的一个人，有多少人内心里真喜欢他，我看梧桐妹子人很不错，应该不会喜欢这种人。”
<br/>　　“是了，她不喜欢。”
<br/>　　“那还要嫁？”
<br/>　　道渊真人道：“老弟，她有得选择吗？”
<br/>　　毛小方瞬间明白了，吸了一口气道：“梧桐妹子很可怜。”
<br/>　　叶少阳笑道：“心疼了？”
<br/>　　“不要乱说！我们毕竟共同患难过，我却是为她感到不值。”
<br/>　　叶少阳拍拍他的肩膀，说道：“我明白，我也一样。”
<br/>　　“不然我们去找她，护送她离开？”
<br/>　　“不必，实话告诉你们，我跟她已经说好了，我让她嫁人。”
<br/>　　道渊真人和毛小方怔住。
<br/>　　叶少阳笑了笑，道：“我有我的计划，真到那时候，你们就知道了。”
<br/>　　第二天，风平浪静，不出所有人的预料，桃花山上一个下山的人都没有。傍晚的时候，道渊真人带回消息，法术公会的人马已经开始集结，就等着第二天攻山了。
<br/>　　好戏要开场了……
<br/>　　当天晚上，在通往桃花山的官道上，一辆两匹马拉的车疾行在官道上。没有车厢，就是一个板车，车上坐着一僧一道，各自都坐在一个小板凳上，身边都是用绳子绑起来的包袱。
<br/>　　是四宝和吴嘉伟。
<br/>　　“宝爷，我说，你确定少阳会在桃花山上吗？”吴嘉伟不放心地问道。
<br/>　　“这么大的盛世，如果少阳知道，以他爱凑热闹的性格，他一定会去的。”四宝啃着一个鸡腿，啃得满嘴都是油。
<br/>　　吴嘉伟在边上看着，忍不住皱眉，“我说宝爷，你这……还是收敛点吧，你是和尚，被人看到这么光明正大的吃肉，实在不雅。”
<br/>　　“怕什么，这些人又不认识我。”四宝无所谓。
<br/>　　这理由……吴嘉伟也是醉了。
<br/>　　叶少阳在这个世界待了一个月，他们也待了一个月，之前，以四宝的想法，是去装神弄鬼，结交权贵，然后通过权贵的力量去寻找叶少阳的下落，可惜在不久之前，他们投靠的那个军阀，跟别的军阀开战然后打输了，两人本来是住在这军阀的府上，差一点还被当成军阀的手下给抓起来，还好两人跑得快，行李，赚来的银子全都没了。两人用仅有的钱雇了这辆板车，逃出了北京城。
<br/>　　也是巧合，两人出城之后，本来没什么去处，正好遇到几个法师，聊天的时候，听说了桃花山的这件事，然后想着叶少阳可能会在，这才急匆匆地赶了过去。
<br/>　　路上没钱吃饭了，经过一个村庄的时候，四宝去偷了一只鸡，在野外烤熟，自己吃了一大半……
<br/>　　一路狼狈，跟逃难似的。
<br/>　　两人做梦都想回去自己的时代。
<br/>　　第二天，一大早，叶少阳给凤兮夫妻俩身上贴上了隐气符，又一人分了一枚铸母大钱放在身上，用铜钱的灵力盖住他们身上的气息，而且今天人多，也不会有人特别注意到他们。做好这一切，叶少阳才带他们出门。
<br/>　　至于美华和包子，早就被叶少阳收到了阴阳镜中，带着一起出门。
<br/>　　桃花山下，已是人潮涌动。
<br/>　　叶少阳终于看到了法术公会的人，不分道佛，全都穿着一样的蓝色长衫，连布鞋的样式也都是一样了，一共二十来人，在山门外一字排开，叶少阳一眼扫去，全都是年轻人，男女都有，道渊真人在一旁告诉他，这些也都是法术公会十几年前从法术界选的，跟之前死了的那个曹雨兴一样，都是天赋略差一些的，不过放在各中小门派之中，也绝对都是佼佼者，就算在大门派，也是内门弟子的水平。
<br/>　　这一行二十来人，神情肃穆，又穿着一样的衣服，看上去倒是有一种威风凛凛的感觉，在这些人身边，还有几个弟子，叶少阳在中间看到了梧桐，还有张晓寒、卢晓清、陈晓宇等人。这几人姿态比较轻松，身上的衣服，是在前面那些人穿的蓝衫的基础上，加上了一些白色的纹饰，一看便知，比前面那些人的身份要高出一档。
<br/>　　这些，是法术公会在人间扶持的真正的核心弟子。
<br/>　　在法术公会的这些人后面，各大门派的弟子鱼贯而立，一个个都不敢出声，人虽然多，但是场面却是十分安静。
<br/>　　叶少阳对着梧桐吹了一声口哨，这声音在安静的人群中显得是那样刺耳，几乎所有人都转头看过来。
<br/>　　叶少阳冲梧桐笑了笑。
<br/>　　梧桐看到是他，怔了一下，随即脸色有点不自然起来。
<br/>　　张晓寒冷冷地看着他。
<br/>　　“叶少阳，是你！”陈晓宇尖着嗓子叫道，“叶少阳，你过来！”
<br/>　　“有事？”叶少阳挑了挑眉毛。
<br/>　　“你我赌定，在龙华会上一争高下，今日龙华会取消了，你我就在这当着众人的面打一场如何，输的人，给对方磕头赔罪！”
