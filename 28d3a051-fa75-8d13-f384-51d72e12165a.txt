　　第455章
<br/>　　程大雷心思越来越重，忽然从马上跳下来，手重重捶在林少羽肩膀上。
<br/>　　“还没死呢，用不着说这些丧气话。本大家既然在此，就不会看着你死，握好兵器，随我冲杀出去，回蛤蟆寨。我倒要看看，谁敢拦我兄弟。”
<br/>　　林少羽剧烈咳嗽，程大雷又一次扯动他的伤口。他好不容易止住咳嗽，抬眼望过去，道：“出去，还出得去嘛？”
<br/>　　程大雷顺着他的目光瞧过去，只见兵马如虎，刀兵如林，披甲之士，足有十余万。
<br/>　　“大当家，这满天下都是要杀我的人呐。”林少羽叹道：“以大当家的本事，逃生应该不难，就不要为少羽耽误性命。能与大当家兄弟一场，少羽已经此生不愧。”
<br/>　　听着耳边的话，程大雷心若刀绞一般，忆起初见之时，林少羽虽然看着有些傻，但也称得上天真烂漫，如此天真一生，却也算不错。怪就怪听了自己一番胡言乱语的话，后家里又遭到横祸，一步一步，走到今日绝境。
<br/>　　追根溯源，自己难脱干系。
<br/>　　“能与你兄弟一场，也是我平生幸事。”程大雷扶稳林少羽身子：“今日你就看着，本大家带你杀敌，斩将，突围。”
<br/>　　程大雷翻身上了战马，将大斧举起，首领光环开到最大。
<br/>　　“起义军的儿郎听令，从现在起，你们由本大家指挥，随我杀出去。”
<br/>　　在首领光环的作用下，诸人对程大雷的好感度蹭蹭往上涨，在加上刚才林少羽一番话，此刻看程大雷的模样，感觉他浑身上下，如同罩着光晕一般。
<br/>　　“圣主！”
<br/>　　“圣主！”
<br/>　　三千残兵一阵阵大吼，杀威震天。
<br/>　　程大雷接过了指挥权，冲林冲挥挥手，将林冲叫到面前。
<br/>　　“大当家……”
<br/>　　程大雷看到他便一肚子火，如果林少羽身边跟着一个明白人，林少羽也落不到这步田地。关键是林冲这个人，脑袋一根筋，什么事都拎不清楚。
<br/>　　程大雷不是很愿意搭理他，将斧头背在身后，从林冲手中要出义字旗。
<br/>　　大旗猎猎而舞，程大雷在马上大喝：“所有人随着大旗冲锋，旗到，人到，有一人向后，后队杀前队。”
<br/>　　“大当家，这桩事让我来吧。”
<br/>　　林冲知道这扛纛的差事不好干。扛着一杆大旗，是军中最显眼的目标，冷箭兵器全往上面招呼。程大雷担起这扛纛的事，确实是将生死置之度外。
<br/>　　“废什么话！”程大雷喝道：“跟紧我！”
<br/>　　“我们与大当家护旗！”
<br/>　　林少羽和林冲翻身上马，各自握紧了兵器。
<br/>　　“好，随我杀出去！”
<br/>　　尉迟离一惊，这程大雷明显是个比林少羽更楞的存在呐，他这是要主动冲锋。
<br/>　　程大雷目光转了一圈，想着从何处突围。他这个时候看到，李行哉冲他挤了挤眼睛。
<br/>　　好呀！程大雷心中大骂，不想着救我，竟然还冲我耀武扬威。
<br/>　　李行哉心中暗暗祈祷，希望程大雷能够领悟自己的意思。从自己这里走，自己力量不大，也就是尽力看不能给你腾出一条生路。
<br/>　　程大雷看着这一幕心中好恨，好呀，还向我炫耀，好好好，我偏偏不给你机会杀我。
<br/>　　程大雷大旗一指，直指三军之主，口中爆喝一声，拍马向前。
<br/>　　“与我杀！”
<br/>　　他冲的帝国军的主阵，坐阵的大将军尉迟离。随着程大雷一声大喝，三千残兵跟着冲了过去。
<br/>　　尉迟离万万没想到，程大雷选择突围，竟向自己的大撵冲锋。一时间防备有些仓促，忙指挥人手防御。
<br/>　　李行哉气得跺脚，这程大雷还是没领会自己的用意呐。这个人，有时看着挺鸡贼，有时却蠢得令人发指。难道我能看着你去死么，不说你我之间的情分，我妹妹还在你蛤蟆寨，你死了，谁去照顾她。
<br/>　　可此刻生闷气也是无用，李行哉心中一动，忙道：“快撤，快撤，回撤杀虎口，只要堵住杀虎口，程大雷就是插翅也难逃。”
<br/>　　宋游渠一听李行哉此话有理，忙带着人回撤到杀虎口前。此地远有兵把守，许进不许出。
<br/>　　“殿下，你们怎么撤回了？”把守军官不解道。
<br/>　　“快快进去，用一个起义军脑袋能换不少银子呢。”李行哉大喊：“这里由我把守。”
<br/>　　此人一听，立刻露出感激的神色：“殿下，你为人……太厚道了。”
<br/>　　“快去吧，再不去就被人杀光了。”李行哉挥手。
<br/>　　程大雷并不是不知尉迟离周围的布兵更多，但其实都差不了多少。当尉迟离也指挥着大军冲过来时，程大雷大旗突然一扫，令尉迟离的大军扑了个空。
<br/>　　尉迟离心中好恨，这程大雷当真是狡诈卑鄙得很呐。
<br/>　　“放箭，放箭！”尉迟离下令。
<br/>　　程大雷使一招调虎离山之计，捡了一个兵力薄弱处，带人冲了过去。其实，程大雷不是很喜欢带这些起义军打仗。他们勇则勇矣，但只知道没头没脑的送死，领会命令自然差些。程大雷用起来也不得心应手。
<br/>　　程大雷还是喜欢蛤蟆寨自己调教出来的兵，那是自己一手培养的，动静之间得心应手，程大雷指到哪儿，他们就杀到哪儿。
<br/>　　聊胜于无吧，程大雷只能如此安慰自己。手中的义字旗抖开，将射来的羽箭扑落，冲进了敌人的大军中。
<br/>　　他用大旗横扫，打出一片空挡。身旁有林少羽和林冲护旗，倒是也无人能拦得住程大雷。
<br/>　　三千残兵也随着他冲杀过来，双方陷入血战。程大雷进一步，他们便进一步，程大雷不退，他们便不退。
<br/>　　如果坐下的坐骑再给力些，程大雷当有把握突出重围，少损失一些人手。不过他随意抢来的这匹战马，很显然是比不过那黑牛。
<br/>　　一路杀杀死死，程大雷不知刺穿了多少人的心脏，也不知留下了多少具尸体。
<br/>　　从大河到杀虎口，程大雷用尸体铺开一条路。
<br/>　　终于冲到杀虎口前，程大雷抬头就看见了李行哉，他骑在马上，耀武扬威，神情十分可憎。
<br/>　　“好呀，竟敢断我后路！”
<br/>(本章完)
