　　　　周围的擂台比试都已经开始，台下弟子也在催促，青云门长老也不再拖延，正色道：“比试规则想必你们都知道，互相不得下杀手，否则门规处置，比试开始！”
<br/>　　声音刚落，两人也没有行礼，谁也没有在意，慕清霄双手负背，面色淡然，手中也不见法宝，对孙弘扬视若无睹。
<br/>　　看他的样子，孙弘扬深吸了口气，压下眼中的杀意，虽然很想杀了对手，但为了这种狂妄自大，没有前途的垃圾，触犯了青云门门规，绝对是划不来的。
<br/>　　冷笑一声，除了下手杀人，他心中可有许多中办法，让慕清霄痛不欲生，道：“慕师弟，刀尖无眼，你可要小心点。”
<br/>　　“喝！”
<br/>　　伴随着一声大喝声，孙弘扬直接祭出一柄青色仙剑，剑身上有着绿色的荧光，散发着极其凌厉的气息，同时玉清境三重巅峰的气势爆发而出，在空中荡起一阵涟漪。
<br/>　　在孙弘扬的气息爆发而出后，站在左侧的长老脸色微变，如此年龄就有玉清境三重巅峰的修为，确实是难得的青年才俊，值得培养，只是心性不佳，难以托付重任。
<br/>　　台下弟子也是一阵惊呼，玉清境三重巅峰的修为，在年轻一代弟子中，已经能算中上层次，颇为不俗，心中不免有些期待起来。
<br/>　　站在擂台右侧，慕清霄依旧面无表情，没有因此而露出丝毫的动容之色，负手而立，望着气势如虹的孙弘扬，星辰一样的眼眸中尽是不屑之色。
<br/>　　透过云气，看到慕清霄眼中毫不掩饰的不屑，孙弘扬心中暴怒，区区大竹峰的弟子居然敢如此狂妄！
<br/>　　随着他的一声怒吼，气势攀升到巅峰，微微响起剑鸣声，旋即携带着破风声与凌厉的威势，愤怒的冲向慕清霄。
<br/>　　绿色的剑芒乍然而现，让孙弘扬的仙剑威力更甚，他昨日从自己弟弟口中得知，慕清霄修为不俗，也许能够与他抗衡。
<br/>　　他弟弟修为是玉清境二重，虽然两个境界距离已经很大，但他也不敢轻敌，上来便施展全力，虽不能将只斩杀，却能够直接废了他的修为。
<br/>　　仙剑夹杂着绿色的剑气，眨眼间便抵达慕清霄身前，近身的孙弘扬嘴角勾起一丝狰狞，果然与弟弟口中所说，狂妄的都不知道躲闪。
<br/>　　“铮！”
<br/>　　正当孙弘扬认为得手的时候，慕清霄连手指都没有动一下，周围浓郁的灵气就化作一张无形的壁障，轻而易举的挡住了后者的仙剑。
<br/>　　“好快的凝气速度，好强的控制力！”
<br/>　　左侧的青云门长老骤然色变，暗自惊叹，先前到是看走眼了，这看似普通的小子，道行竟如此深厚，今年大试中真是藏龙卧虎啊！
<br/>　　台下弟子也是目瞪口呆，心跳的贼快，他们原本都认为，在这强大的一击下，慕清霄不死也得重伤，可现在结果却超乎他们的想象，令他们不敢相信自己的眼睛。
<br/>　　“卧槽，龙首峰的孙师兄是玉清境三重巅峰吧，刚刚一剑我绝对接不下来，慕师兄好强啊！”
<br/>　　“刚刚真是刺激，我连心跳都快停止了，慕师兄的凝气速度真快，对太极玄清道的领悟绝对比我高！”
<br/>　　“真是深藏不露，都说大竹峰首座田师叔教导弟子无方，看慕师兄的修为，也不见得如此。”
<br/>　　听闻四周弟子的惊叹，孙弘扬脸色胀红，心弦一颤，望着面无表情的慕清霄，越发的愤怒，双手握住剑柄，全身灵气化作剑气，铆足了劲力向慕清霄刺去。
<br/>　　望着孙弘扬脸色胀红拼命的模样，慕清霄嘴角勾起一丝不屑的笑容。
<br/>　　刚上来不用试探自己的修为，反而直接施展出全力，杀自己谅他也不敢，但刚才的一击，绝对能够废了任何一位修为比他低的弟子。
<br/>　　既然如此，正好给了自己机会，慕清霄脸上的轻笑散去，面无表情，周围云气轻微一颤，宛如被炸开的水面，云气急速向四周溅射开来。
<br/>　　霎时间，一股不可抗力的劲道向慕清霄半米范围内弹射而出！
<br/>　　“蓬！”
<br/>　　“噗...!”
<br/>　　眨眼间，孙弘扬的手中仙剑掉落于擂台的同时，身躯也如同断了线的风筝，不受控制的向后倒飞出去，在空中喷出几口鲜血，脸色萎靡不振。
<br/>　　“砰砰...！”
<br/>　　孙弘扬的身躯以迅雷不及掩耳之势倒飞而出，最终重重的砸在玉石地砖上，身下的玉石地砖在恐怖的力量下，顿时炸裂开来。
<br/>　　此时，躺在玉石地砖上，孙弘扬气色萎靡不振，神情恍惚，周围一片寂静，只剩下微风拂动声，他只觉得自己的身体剧痛无比，连动一动手指的力量都没有。
<br/>　　孙弘扬眼中仍旧不敢置信，自己居然就这么输了，输给了大竹峰的弟子，口中再次喷出一口鲜血，意识逐渐消退，耳边传来龙首峰弟子的惊呼声......
<br/>　　青云门长老回过神来，瞬间来到孙弘扬身前，抓住他的手腕，瞳孔顿时一缩，倒是没有生命危险，但体内的经脉如同缠在一起的蜘蛛丝，乱的一塌糊涂。
<br/>　　显然，这是被强大无比的劲力所震乱的，真是恐怖的控制力，青云门长老忍不住看了一眼台上镇定自若的慕清霄，眼中闪过一丝惊骇！
<br/>　　良久，他才回到擂台上，开口宣布道：“只是重伤昏迷，没有生命危险，此次比试，由大竹峰慕清霄胜！”
<br/>　　嘴上虽如此说，但他清楚却清楚孙弘扬的伤势有多么的严重，先前一番查探，后者体内经脉乱七八糟，往后别说修炼，连吸纳灵气都会变得比普通弟子困难百倍。
<br/>　　虽然世间有神药能够恢复，但神药是何物，天地间最珍贵的至宝，也不曾听说青云门内有，就算有又如何，掌门又怎会将宝贵的神药浪费在他的身上。
<br/>　　青云门长老暗自叹息一声，希望孙弘扬醒来后，能够挺住打击吧，最后暗自赞叹了慕清霄一声，这小子脸上虽挂着人畜无害的笑容，可下起手来却比谁都狠辣啊！
