　　戴安娜看到这一幕，顿时脸色大变，
<br/>　　“叶先生，您不该动手的！”
<br/>　　叶尘却一脸悠然，淡淡道：
<br/>　　“不就是一个小喽啰吗？这你都搞不定？”
<br/>　　戴安娜顿时气得差点吐血，
<br/>　　如果不是因为自己的小命，还被攥在叶尘的手里，估计早就把他骂的狗血喷头了。
<br/>　　“这人可不是小喽啰！他是罗刹城四大将之首北山家的嫡子，北山铁男！”
<br/>　　“因为钟情于的十七公主，所以甘愿作为公主的亲卫……”
<br/>　　还没等戴安娜说完，叶尘直接摆了摆手，道：
<br/>　　“管他是谁，他若是不识趣，杀了便是！”
<br/>　　叶尘方才已经动用神念，将整个罗刹城扫描了一遍，应该不存在能够威胁到他的高手。
<br/>　　戴安娜听到叶尘这话，差点忍不住就想要破口大骂，不过最终还是忍住了。
<br/>　　就在她准备继续劝说叶尘之时，
<br/>　　那边人仰马翻的北山铁男，已经从地上爬了起来，气冲冲的朝叶尘走了过来，
<br/>　　“小子！你好大的胆子！你知道我是什么人吗？跪下道歉，可饶你不死！”
<br/>　　北山铁男虽然惊愕于叶尘的实力，不过当他看到，叶尘穿的不过就是普通侍卫的盔甲，顿时大为轻视，直接怒声训斥道。
<br/>　　叶尘顿时两眼微微一眯，
<br/>　　“你是什么人不重要，再敢出言不逊，死！”
<br/>　　轰！
<br/>　　叶尘这话一出口，无论是过往的行人，还是门口的守卫，全都愣住了。
<br/>　　继而，
<br/>　　又全都露出一副看傻瓜的表情。
<br/>　　北山家族，
<br/>　　在整个罗刹城，那都是最为顶尖的大家族。
<br/>　　而这北山铁男，
<br/>　　作为北山家的嫡子，自然是无人不知，无人不晓。
<br/>　　放眼整个罗刹城，在年轻一代之中，除了罗刹王族一脉，此人绝对是最顶级的纨绔大少！
<br/>　　而且，此人深得罗刹城主的喜爱，即便是罗刹王族一脉里面，也只有那几位得宠的公主、王子，才能压过此人一头。
<br/>　　而如今，一名普通侍卫，竟然敢如此和北山铁男说话，简直就是不知死活啊！
<br/>　　戴安娜听到这话，顿时脑袋一懵，险些晕厥过去，连忙拉了拉叶尘的胳膊，几乎是低声哀求道：
<br/>　　“叶先生，不可……”
<br/>　　虽然她和叶尘才认识不久，可是她能够感觉到，眼前的这个人族男子，绝对是一个杀伐果断之人。
<br/>　　至于那北山铁男，先是一愣，显然没有想到，叶尘竟然敢口出如此狂言，继而哈哈大笑，然后笑容又猛然一收，脸上露出一抹狰狞之色，
<br/>　　“好好！好小子！本公子长这么大，你是第一敢和我这么说话之人！我倒要看看，你如何让我死！”
<br/>　　说完这话，北山铁男就要朝叶尘扑过来。
<br/>　　叶尘两眼低垂，不由得摇了摇头，正打算抬手将此人轰杀。
<br/>　　然而就在这时，
<br/>　　忽然一道清脆动听的声音响起，
<br/>　　“北山铁男！你给我住手！”
<br/>　　叶尘不由得抬头一望，只见之前已经奔入城中的一头灵兽，又掉头奔了出来。
<br/>　　而在那头灵兽之上，正坐着一名身穿金色盔甲的魔族少女。
<br/>　　这少女的脸庞，看上去还有些稚嫩，不过姿色绝顶，还在戴安娜之上，而且身材修长，目测已经超过一米八，只是有些单薄，不及戴安娜身材火爆。
<br/>　　魔族身材普通高大，所以估摸着她的年纪，也就十四五岁而已。
<br/>　　“是十七公主来了！”
<br/>　　“拜见公主殿下！”
<br/>　　“十七公主不愧是我罗刹城第一美人，果然是倾国倾城啊！”
<br/>　　……
<br/>　　伴随着众人的惊叹声，十七公主已经驾驭着灵兽，来到跟前。
<br/>　　戴安娜当即上前，恭恭敬敬行了一礼，然后将大致情况，向这位十七公主汇报了一遍，言语之间，自然更加偏向于叶尘。
<br/>　　毕竟不管怎么说，她现在还受制于叶尘，只能尽量为他开脱。
<br/>　　但是一旁的北山铁男，顿时便忍不住了，
<br/>　　“公主殿下！这小侍卫对我无礼，竟然敢将我从灵兽上拉下，而且还口出狂言，我非杀他不可！”
<br/>　　那十七公主瞥了一眼北山铁男，美眸之中闪过一抹厌恶之色，继而目光又转移到叶尘的身上，顿时两眼一亮，露出一抹喜色，转而望向一旁的戴安娜，笑道：
<br/>　　“戴安娜侍卫长，你的这个手下，模样倒是生的不错嘛！”
<br/>　　叶尘顿时一阵无语，
<br/>　　罗刹族的男子，几乎没有长的好看的，为此他之前已经改头换面，刻意让自己变丑不少，没想到竟然还是被这位公主看上了。
<br/>　　那北山铁男听到这话，顿时醋意冲天，恨不得要把叶尘生吞活剥。
<br/>　　“我要杀了你！”
<br/>　　叶尘一脸淡然，仿佛没有听到一般，
<br/>　　北山铁男在他眼中，不过就是一只微不足道的小蝼蚁罢了，如果想要取他的性命，随时都可以办到。
<br/>　　然而，
<br/>　　没等北山铁男动手，那十七公主却已经再次开口了，
<br/>　　“住手！北山铁男！你是不是连本公主的命令都不听了？”
<br/>　　“公主……”
<br/>　　北山铁男顿时大急，正想要开口解释。
<br/>　　不料，
<br/>　　十七公主直接摆了摆，将他的话打断，转而指着叶尘道：
<br/>　　“本公主宣布，从现在开始，你就是本公主的人了！”
<br/>　　哗！
<br/>　　周围众人听到这话，顿时便躁动了起来，望向叶尘的眼神之中，各种羡慕嫉妒恨。
<br/>　　要知道，十七公主不但身份高贵，又是罗刹城第一美女，不知道有多人男人，拼了命想要讨得她的欢心。
<br/>　　就连北山铁男，如此尊贵的身份，都心甘情愿成为的随从。
<br/>　　说完这话，十七公主又望向一旁的戴安娜，笑道：
<br/>　　“戴安娜侍卫长，你应该不会舍不得吧？”
<br/>　　戴安娜顿时脸色大变，
<br/>　　“公主殿下，这……这个……”
<br/>　　原本，对于十七公主的命令，她自然不敢不从，可问题是，眼前这个人族男子，又岂是她能左右的？
<br/>　　十七公主听到戴安娜迟疑，顿时俏脸一板，身上升起一股寒意，
<br/>　　“怎么？莫非戴安娜侍卫长不愿意？”
<br/>　　戴安娜顿时大为难为，就在不知道该如何回答之时，叶尘忽然开口了，
<br/>　　“是我不愿意！”
<br/>　　哗！
<br/>　　叶尘此话一出口，所有人再次哗然。
<br/>　　他竟然，
<br/>　　敢拒绝十七公主？
<br/>　　然而，
<br/>　　这还不算！
<br/>　　叶尘接下来的话，顿时让所有人，全都震惊的说不出话来。
<br/>　　“如果你给我当婢女的话，或许我还可以考虑一下！”
