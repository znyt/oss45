　　即便是在摇光峰的山顶上，所有修为稍差的弟子，都被这声不存在的钟声震的瘫软在地。摇光仙子这样的强者也是感到了体内一阵悸动，罡气翻涌起来，一个个骇得说不出话来。
<br/>　　“阿弥陀佛，不愧是东皇钟，大象无形，********……”一个和尚双手合十，喃喃说道。
<br/>　　在山顶上的这些强者，都产生了如此可怕的感觉，更不要说是山谷中：
<br/>　　原本汹涌澎湃的血浪，已经被震碎了，远程操控的女魃，口中喷出了一股血箭，后卿强行按捺着钟声对头脑的刺激，飞身上前，一只手抱着芮冷玉，挡在了女魃面前，说道：“走！”
<br/>　　“现在……这么走了，不是功亏一篑！”女魃还不死心。
<br/>　　“你自己看看！”
<br/>　　两人放眼望去，山谷之中，所有的僵尸都在钟声的震荡中石化了，场面静寂无声，然后，在以东皇钟为中心的半径开始，那些僵尸身上的血肉逐渐融化，从身上脱落下去，只剩下一副白骨……
<br/>　　无数僵尸，一圈一圈地烂掉，然后倒下，这等恐怖的场面，让在场的这些见惯了生死、一向宠辱不惊的诸神也感到了胆战心惊。
<br/>　　东皇钟，居然有这样的威力。
<br/>　　道风和李浩然，因为没有登上东皇钟避难，也受到了钟声的震动，不过因为离东皇钟比较远，再加上两人修为极深，没有受到重伤，不过神识也是受到了重击，从空中坠落，一时间提不起罡气。
<br/>　　“走，转攻各大山门！”
<br/>　　后卿和女魃缓了一下，勉强提气，大喝一声，朝着尸族大军进来的那两座山的中间飞奔离去，芮冷玉也被带走了。
<br/>　　不过叶少阳却是管不了这些了，在念动咒语的瞬间，他所有的法力都被掏空了，人虚脱地倒在东皇钟上，在意识最后清醒的一刻，他感到自己的神识被东皇钟吸收了进去，然后沉沉地昏了过去。
<br/>　　“老大，老大你怎么了！”小白等人从之前那惊心动魄的一幕中回过神来，发现了叶少阳的异常，立刻紧张起来。
<br/>　　“他没事，不要惊慌！”广宗天师在叶少阳身边坐下，转头对阴间四神说道：“你们去追击尸族吧，战斗还没结束。”
<br/>　　阴间四神这次来空界的任务有两个，一个是抓捕转世鬼童，一个是抵抗尸族入侵，现在尸族大军还在，芮冷玉也在后卿的手上，他们自然不敢松懈，各自看了看叶少阳，然后抽身离去。
<br/>　　“你们几个也去吧。多杀点僵尸，也算是将功补过了，对少阳也有好处。”广宗天师吩咐小白等人。
<br/>　　大伙一想也是，小白问道：“那老大怎么办？”
<br/>　　“少阳没事，我来看着他就行。”
<br/>　　四宝说道：“我是人间法师，阴司暂时管我不着，暂时不需要对他们示好，你们去吧，这里有我。”
<br/>　　小青等人于是离去。
<br/>　　四宝留下，吴嘉伟也留下了。凤兮要走，林三生说道：“凤兮你留下，我找你有事。”
<br/>　　凤兮愣了一下，也留下了。
<br/>　　四宝上前扶起叶少阳，问广宗天师：“少阳真没事？”
<br/>　　“当然，你不相信我？”
<br/>　　“不是。不过……我说老爷子，你不是一直关在山河社稷图中，我怎么感觉你神机妙算的，什么都知道？”
<br/>　　广宗天师没理会他，对林三生道：“可以了。”
<br/>　　林三生还犹豫着，道：“师父，真的要……”
<br/>　　“只有这么做，才能保护少阳。”
<br/>　　“什么鬼东西？”四宝问，“你们师徒打什么哑谜？”
<br/>　　林三生没做声，从衣服了摸出了一个小东西，是一块褐色的木牌，上面刻着一些金色的符文。
<br/>　　林三生伸手把符文抹去，然后，木牌粉碎，一道人影落在东皇钟上，是一个黑须冉冉的道士。
<br/>　　摇光仙子等人在山顶，看见这个道士，本来在吐纳的他们，看见这一幕，腾地一下都站了起来，盯着那个老道士，互相看去，都摇了摇头，表示不认识。
<br/>　　“这种样式的道袍，至少是隋唐以前……是了，这还在汉以前，应该是战国或秦朝的。”黎山老母喃喃说道。
<br/>　　战国或秦朝的……众人面面相觑，他们不怀疑黎山老母在这方面的判断，但是……就算是在空界，年代最早的修行者，也是晋朝时候人，因为三国时期，道教才真正成立，在那之后，道法不断完善，才给后人提供了飞升的可能，在那汉之前，只有术士和方士，在那种道法不完善的情况下，基本上没有飞升的可能，能够堪破生死的，都是旷世奇才，寥寥无几。
<br/>　　李浩然虽然是老子所骑的青牛，算是春秋时期的人，不过他是以轮回入道，转世了不知道多少代，只能从名为李浩然的这一代来算，不过就是近几十年的人，辈分还是很低的。
<br/>　　秦朝以前的方士……就算是这些空界的大佬们，也是第一次见到，不禁纷纷猜测这个人是谁。
<br/>　　不光是他们，四宝等人也是第一次见到这个老道士，顿时也都惊呆了，看看老道士，又看看林三生，四宝说道：“军师你搞什么鬼，这是谁？”
<br/>　　林三生喃喃吐出四个字：“徐福祖师。”
<br/>　　徐福！这就是徐福！
<br/>　　四宝和吴嘉伟都怔住了，一脸懵逼。
<br/>　　凤兮缓过神来之后，激动得难以自已，失声道：“您就是徐福祖师！”
<br/>　　她的声音，被山上一干竖起耳朵听动静的人听到。黎山老母几乎打了个趔趄，傻傻地盯着徐福。摇光仙子捂住了嘴巴。
<br/>　　“天呐，这个传说中的人物……居然尚在人间！”
<br/>　　众人悚然震惊。
<br/>　　阴间四神正在追击尸族大军，白无常腰间突然放出了一束白光，他自己没发现，马面却发现了，说道：“老八你腰里别着什么宝贝，怎么自己就亮了？”
<br/>　　宝贝？黑无常低头看了一眼自己腰间，也是疑惑起来，伸手一摸，将一块碎裂的镜面摸了出来，亮光正是这块碎镜子发出的。
<br/>　　“这是……呀！”黑无常猛然想了起来，“这是老徐给我的孽镜碎片！他预测徐福会出现，让我留意，一旦徐福出现，立刻汇报！”
