
    鱼馆外天色大黑，天边挂着一轮明月，澜沧江边江水滔滔。<br/>　　<br/>　　    墨家巨子单手扶着凉绣，一张脸上挂着满不在乎的神情，对面是刚刚从屋里追出来的王惊蛰和孔德荣，还有几名孔府子弟，小草十分聪明的没有出屋，她也怕双方交手的时候，那老头要是突然把目标对准自己的话，恐怕会让王惊蛰分了心。<br/>　　<br/>　　    这个对峙，要是放在唐梁山上墨家巨子的主场，他俩绝对一点胜算都没有，山中机关遍地重重令人防不胜防，墨家巨子在出手那就是如虎添翼了，但在这澜沧江边的野外，要啥啥没有的话，所以差不多可以算是势均力敌了。<br/>　　<br/>　　    墨家巨子搀着凉绣往后退了几步，在她耳边安慰了几句，然后让她靠着一棵树坐了下来，凉绣仍旧眼神呆滞，毫无反应，随即再次转身的老人一瞬间气势突然爆棚起来，似乎从老态龙钟的状态，华丽转身到了生龙活虎的层次，这老头无疑是看王惊蛰和孔德荣仍旧追着不放，是打算要认真了。<br/>　　<br/>　　    孔德荣压低声音说道：“你家小草说的没错，咱俩没必要跟他硬扛，缠斗就行了，他的年岁在那摆着呢，再加上可能是要续命了，咱直接给他耗到油尽灯枯就行了……”<br/>　　<br/>　　    孔德荣一句话还没说完，让他俩侧目的意外就发生了，将凉绣放在树下的墨家巨子，居然率先就冲了过来，速度之快，几乎是脚尖点了两下地面就已经冲到了王惊蛰和孔德荣的身前，同时再次扬手，连续抖了几下，他袖口里就有寒芒闪过，王惊蛰和孔德荣的面门上，又再次被几把尖刀给瞄上了。<br/>　　<br/>　　    小草在屋内有点皱眉，这墨家巨子出现的时候身无长物，走路的时候还颤颤巍巍的，但抖手之间连续飞了好几把刀，这刀应该是藏在他唐装的袖口里的，几把刀也得有好几斤重了，可他刚出现的时候，谁也看不出他举手投足间袖子里是藏了武器的，动作显得特自然和随意，这说明啥，说明这老头可能一辈子都是随身携带着这些玩意的。<br/>　　<br/>　　    小草略微琢磨了下，忽然出声提醒了一句：“他可能浑身上下都是机关术数，你们得小心了……”<br/>　　<br/>　　    王惊蛰和孔德荣刚躲过一刀，豁然一惊，墨家巨子也扭着脑袋深深的望了眼小草，他这一看小草就知道自己是猜对了，这老头绝逼像一个机器猫似的，挥一挥手说不准就会出现什么暗器了，墨家机关术独步天下，他也许将自己也给经营成了一个机关大阵。<br/>　　<br/>　　    茅小草的提醒也不过就是一瞬间的事，墨家巨子看了她一眼后，谁也没看见他手上忽然多了个木头匣子，四四方方巴掌大小，这老人将木匣看似随意的就扔在了身前的地面上。<br/>　　<br/>　　    “啪”木匣落地发出一声闷响，但按常理来讲，这木头匣子怎么说也得翻滚两下才是，但落地之后却很平稳，明显是匣子本身有点重量了，王惊蛰和孔德荣脸色“唰”的一变，他俩几乎同时都反应极快的朝着两旁扑了过去。<br/>　　<br/>　　    虽然不知道接下来会发生什么，但谁也不可能站着不动，就干巴巴的等着。<br/>　　<br/>　　    于此同时，木头匣子落地之后就稍微的停顿了大概一两秒钟，上方的盖子突然就被弹开了，从中射出十几枚钢珠，速度极快的朝着前方以扇形的方式射了过去，此时王惊蛰和孔德荣扑倒之后，身子还没落地，他俩就感觉到下半身有的地方一凉，钢珠在弹射中，就穿透了皮肉。<br/>　　<br/>　　    “噗通”<br/>　　<br/>　　    “噗通”<br/>　　<br/>　　    两人倒地，翻滚了几下后躲远了，王惊蛰伸手抹了下小腿，手上沾了一片血迹，腿上已经血肉模糊了，有两枚钢珠打了进来。<br/>　　<br/>　　    在冷兵器时代，这种机关杀人术曾经大放异彩过，墨子曾经流传出一书叫《墨子·备城门》，就曾经被应用到攻城和守城中，其中很多最为尖端的技艺到现在都已经失传了，这些技术手段哪怕就是放到现在，都够呛能够仿制出来了，还有诸葛亮曾经创造的木牛流马，也是机关术的一种，和后期曾经出现的木弩，投石机等等。<br/>　　<br/>　　    只不过可惜了，自从墨家悄然隐退之后，墨家的机关术也逐渐的失传了，不在世间展现了，真要是墨家机关术延续到现在，一直都在各朝各代中传承下来的话，可能历史也会随之改变了。<br/>　　<br/>　　    再说王惊蛰和孔德荣，两人几乎都被钢珠给打中了，但这伤不算致命，他俩心有余悸的从地上爬了起来，谨慎的望着墨家巨子。<br/>　　<br/>　　    巨子淡淡的说道：“你们若要是想拼个你死我活，我这老家伙拼着不要这条老命也能把你们给拉过来垫背，那眼尖的小姑娘说的不错，我这一身上下都是机关术，到底有多少种我自己也忘记了，你们现在看到的不过是边边角角而已，真要是等我把这一身机关都给用了一遍的话，在我黔驴技穷之前，也肯定能从你们身上讨到足够的利息，怎么样，还往下继续？”<br/>　　<br/>　　    王惊蛰皱了皱眉，孔德荣阴着脸说道：“你别说的那么危言耸听，都到这个时候了，我们哪里还有收手的道理？你也不过就是一个人罢了，我就算你浑身上下都是机关，但总归是放一个就少一个，你还能再生出来不成？不好意思了，江湖规矩什么的在我这里也不存在，我就不信在车轮战下，你还能坚挺多久……你们也上，给我困死了他！”<br/>　　<br/>　　    孔德荣说完，墨家巨子脸色大变，随即孔德荣带来的孔府弟子，大概有十几个人全都谨慎的围拢了过来。<br/>　　<br/>　　    他说的话没错，我就算你浑身上下全是装备弹药，你不也得有弹尽粮绝的时候么？<br/>　　<br/>　　    “无耻！”墨家巨子咬牙怒吼了一声。想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
