　　白冰雪四人离去。
<br/>　　萧逸则在宫殿内随手触发了一道禁制。
<br/>　　一个掩藏于禁制内的房间，凭空而现。
<br/>　　同一时间，一道黑影，疾速袭来。
<br/>　　这是一头七级妖兽，名唤暮影苍狼，大概在七级初阶实力。
<br/>　　擅速度，以无声无息偷袭而闻名。
<br/>　　对于天元境以下武者，或者刚刚突破天元一重的武者来说，简直是致命的危机。
<br/>　　而对萧逸而言，这只是寻常妖兽。
<br/>　　随手一道剑气打出，将这头暮影苍狼分尸。
<br/>　　随后打量了几眼这房间，房内，有不少天材地宝和武器。
<br/>　　天材地宝，清一色在六品阶级。
<br/>　　武器，则是一把上品灵器。
<br/>　　还有一些杂七杂八的勉强算珍贵之物。
<br/>　　这些东西，对萧逸并无大用，只是聊胜于无罢了。
<br/>　　大手一挥，将这些东西全部收于乾坤戒，并将暮影苍狼的内丹与精血吸摄。
<br/>　　随后离开了房间。
<br/>　　整个洞府遗迹，占地方圆百里，宫殿数百座。
<br/>　　真正的宝贝，都在主殿中。
<br/>　　若想找到主殿，难度并不小。
<br/>　　而本身每座宫殿都很大。
<br/>　　宫殿内，隐藏着各种禁制。
<br/>　　禁制中，有危机，也有宝贝。
<br/>　　十数分钟后，萧逸已将这座宫殿逛了个遍。
<br/>　　里头所有禁制也已触发，并未得到什么重宝。
<br/>　　倒是得了不少五品与六品的天材地宝，还有两门地阶中级武技。
<br/>　　以及两把上品灵器。
<br/>　　很显然，这座宫殿不是主殿。
<br/>　　离开这座宫殿。
<br/>　　外头，已经有大量武者越过那片雪地广场。
<br/>　　密密麻麻的身影，涌向各个宫殿。
<br/>　　萧逸眉头微皱，身影一闪，瞬间前往下一个宫殿。
<br/>　　十数分钟后，萧逸略带失望之色，摇了摇头。
<br/>　　这座宫殿，仍旧只是普通的宫殿，并无太大收获。
<br/>　　数个时辰后，萧逸已寻了数十个宫殿。
<br/>　　五品、六品的天材地宝，以及地阶中级以下的武技功法，得了一大堆。
<br/>　　这些宫殿内的禁制，瞒不过萧逸的感知。
<br/>　　萧逸破掉禁制，击杀里头的妖兽，并不难。
<br/>　　这让得他在各宫殿的搜索速度极快。
<br/>　　而其他武者，速度则慢了许多，甚至还会有伤亡的情况出现。
<br/>　　很可能，一个宫殿，其他武者得划伤近一个时辰的时间，才能全部搜索完。
<br/>　　这数百座宫殿，若想全部搜索个遍，恐怕得数天时间。
<br/>　　又是数个时辰。
<br/>　　萧逸，大概来到整个洞府遗迹的中间部分宫殿。
<br/>　　刚来到其中一座宫殿，却忽然从其内传入一阵哀嚎之声。
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　很显然，这座宫殿已经有别的武者率先进入了。
<br/>　　不过，丝毫情况并不太妙。
<br/>　　萧逸闪身进入，顿时看到十数个武者陷入了苦战，且个个带伤。
<br/>　　这十数人，在这一届冰缘大会的参加武者中，应该算是较强的一批了。
<br/>　　有好几人都是地元九重，其余清一色地元八重修为。
<br/>　　而他们的年纪，大概在25、6左右。
<br/>　　估计在参加冰缘大会前，也起码有着地元四、五重修为。
<br/>　　这般年纪，有如此修为，已经可以列入绝世天才的行列了。
<br/>　　绝世天才，个个都有越级战斗之力，有一定的底牌。
<br/>　　故，他们现在虽是地元八、九重修为，但战力，绝对踏入天元境。
<br/>　　只可惜，他们的对手，是一只双头冰蛇，这才陷入了困境。
<br/>　　双头冰蛇，七级初阶妖兽。
<br/>　　有两头，一头控冰，一头控雪；双头齐攻，冰雪齐至，绝对有着媲美天元境中期实力。
<br/>　　此时，这十数人，个个带伤。
<br/>　　恐怕要不了多久，就会完全落败，甚至殒命。
<br/>　　果然，下一秒，双头冰蛇一道寒冰打出。
<br/>　　十数人瞬间被冰封，化作冰雕。
<br/>　　随后，双头冰蛇头颅一动，一道风雪，肆虐而来。
<br/>　　一旦被这些风雪打中，冰封下的十数人，将会随同寒冰，化作冰屑，就此殒命。
<br/>　　恰在此时。
<br/>　　数道剑气凌空而来。
<br/>　　嗖嗖两声，风雪轻松被挡下。
<br/>　　“破。”萧逸低喝一声。
<br/>　　剑气，极其精准地打在冰雕上，将冰雕破开。
<br/>　　十数人，破冰而出，个个面露余悸。
<br/>　　“退出去吧。”萧逸淡淡地说了一声。
<br/>　　十数人点点头，拱了拱手，道，“谢谢。”
<br/>　　随后，悉数退出这座宫殿。
<br/>　　前方，双头冰蛇狰狞地吐着信子，冰冷的目光，盯住了萧逸。
<br/>　　似乎在怨恨萧逸放跑了它的‘猎物’。
<br/>　　下一秒，双头冰蛇两个狰狞的头颅同时一动，冰雪双双攻来。
<br/>　　“找死。”萧逸低喝一声，手中数道剑气挥出。
<br/>　　冰雪被破，随后剑气威力不减，打在双头冰蛇身上。
<br/>　　嘶的一声，如同切开一张薄弱的白纸般。
<br/>　　双头冰蛇，收尾分尸。
<br/>　　两个蛇头，无力地躺在地面，蛇身，却还在翻滚个不停。
<br/>　　萧逸没有理会，取了内丹和精血，随后看向宫殿内的宝物。
<br/>　　顺带一提，双头冰蛇，拥有两颗内丹。
<br/>　　“嗯？”萧逸看了眼宫殿内的宝物，脸色一喜。
<br/>　　这里头的东西，品阶显然比其它宫殿高上不少。
<br/>　　天材地宝无数，皆是七品。
<br/>　　还有两把极品灵器，以及一本地阶高级武技。
<br/>　　大手一挥，萧逸将其收入乾坤戒。
<br/>　　十数分钟后，萧逸破掉了这座宫殿的所有禁制，取了所有宝物。
<br/>　　随后，离开这座宫殿。
<br/>　　刚打算前往下一座宫殿，远处，却猛地传来一阵激烈的打斗。
<br/>　　看其威势，绝对是天元境层次的战斗。
<br/>　　最重要的是，那个方向的上空，正有一股青色火焰，燃烧不停。
<br/>　　“空青火。”萧逸眉头一皱，随即身影一闪，连忙往那个方向赶去。
<br/>　　不过半晌，当萧逸来到时。
<br/>　　看到叶铭和钟无忧二人，正和一群武者起着争执。
<br/>　　那群武者，清一色是修炼寒冰武道之人，很明显是极寒之地本土势力的武者。
<br/>　　而这群人，为首的，正是风花月。
<br/>　　“萧逸。”风花月看到萧逸，脸色突变。
<br/>　　萧逸没有理会他，而是走到叶铭身旁，问道，“怎么回事？”
<br/>　　叶铭满脸喜意，道，“萧逸师弟，我找到主殿了。”
<br/>　　“哦？”萧逸微微有些惊讶。
<br/>　　他可相信钟无忧和叶铭二人搜索宫殿的速度会比他快。
<br/>　　一旁钟无忧一边与风花月等人对峙着，一边撇撇嘴，道，“叶铭这家伙，也不知说他运气好，还是倒霉好。”
<br/>　　“随便摸了个禁制，把主殿给摸出来了。”
<br/>　　“呵呵。”萧逸笑笑，看了眼主殿。
<br/>　　主殿前，有一巨大牌匾，牌匾上，写着四个字…
<br/>　　“暴雪山庄？”萧逸脸色微变，惊喜中，带着忌惮。
