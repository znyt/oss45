        “”明司寒随后报出了自己的手机号码。
<br/>
<br/>        连星儿给朱小溪发完信息后，下意识地偏头看向明司寒，不解地反问：“你又想做什么？”
<br/>
<br/>        “我的手机号码，记住了吗？”明司寒微微一笑。
<br/>
<br/>        连星儿眨巴着眼睛，随即一脸嫌弃：“谁要你的手机号码啊？”
<br/>
<br/>        “你刚刚不是主动跟我求的吗？”明司寒咧嘴道。
<br/>
<br/>        连星儿顿时不可思议地瞪着这个男生。
<br/>
<br/>        他看起来与她年龄相仿，但是，此刻的脾气性子怎么跟那天在网咖里撞见的完全不同啊？
<br/>
<br/>        那天这男生的脾气又臭又高冷，今天怎么对她这么主动？
<br/>
<br/>        “看到是你之后，我就没兴趣了。反正，会打游戏的大神又不止你一个。更何况，我还有一个超级厉害的师父了，他可比你厉害多了！”连星儿不禁洋洋得意地说。
<br/>
<br/>        明司寒不假思索地说道：“我就是你的……”
<br/>
<br/>        他的话只说到一半，就没下文了。
<br/>
<br/>        因为，连星儿此刻已经懒得再理会他，而是从手提包里掏出耳麦戴到耳朵上，两眼一闭，开始自顾自地听歌。
<br/>
<br/>        明司寒顿时难以置信地看着连星儿，索性也懒得再理会她。
<br/>
<br/>        到了云城后，两人在同站下了车，一前一后地出了站，就跟陌生人一样又先后上了各自拦下的出租车。
<br/>
<br/>        连星儿去了群主约定的酒店准点集合，和大家一起住进了这家实惠的酒店。
<br/>
<br/>        虽然，连星儿更想住五星级的酒店，但这个时候，她也只能“随波逐流”，跟着大众走。
<br/>
<br/>        群主给他们订的都是双人间，所以和她连星儿住在同一间房的还有另外一个女孩子。
<br/>
<br/>        这个女孩，戴着眼镜，扎着马尾，穿着白色蝴蝶袖和淡蓝色的牛仔裤。
<br/>
<br/>        女孩一进门，就十分热情地自我介绍道：“你好，我叫顾千凝，是洛羽的死忠粉。”
<br/>
<br/>        “你好，我叫连星儿，也是洛羽的死忠粉。”连星儿也友好地应了声。
<br/>
<br/>        顾千凝接着二话不说，就把自己的背包放了下来，拉开拉链后，拿出了一本相册递给了连星儿。
<br/>
<br/>        “来来来，我给你分享一些我收藏的洛羽的‘干货’！网上都没我这么全噢！”
<br/>
<br/>        连星儿一听这话，立马接过相册，一屁股坐在床上，脱了鞋盘起腿来。
<br/>
<br/>        从连星儿翻开的第一页起，顾千凝便坐在连星儿的身边，一一解说起了每张相片的来源。
<br/>
<br/>        连星儿没一会儿就跟顾千凝混熟了，两人有说有笑，就像认识了很久一样。
<br/>
<br/>        当连星儿翻看到最后一张相片时，目光的焦点，顿时落在了相片上洛羽的脖子上。
<br/>
<br/>        她本能地用指甲去抠了抠相片，顾千凝见状，忍俊不禁道：“这不是什么脏东西啦！这是洛羽脖子上的痔！”
<br/>
<br/>        “痔？！”连星儿惊怔。
<br/>
<br/>        洛羽的脖子前正中央有颗痣，她还是第一次知道，并且，这颗痣，和小时候那个救了她的小哥哥……
<br/>
<br/>        想到这里，连星儿暗自窃喜起来。
<br/>
<br/>        难道，洛羽就是她一直在找的小哥哥？
<br/>
<br/>        连星儿笑眯眯地抬起手来，捂着自己的胸口，隔着布料，她触到了那颗挂在自己脖子上的小纽扣。
<br/>
<br/>        看来，这颗小纽扣是有灵性了。
<br/>
<br/>        居然能带着她找到它的主人！
<br/>
<br/>        第二天，群主领队，带着他们去了云城电影城，洛羽所在的《联盟》剧组。
<br/>
<br/>        他们在外排队等了一会儿，然后一一核对了身份后，才有条不紊地入场。
<br/>
<br/>        探班须知，他们全都记住了。
<br/>
<br/>        所以，当大家看到洛羽正在和女主角对戏的时候，都很安静地远远观望。
<br/>
<br/>        连星儿站在人群里，心里想的是，待会要怎么和洛羽单独接触，跟他求证小时候的事情。
<br/>
<br/>        几场对戏下来后，终于到了洛羽的休息时间。
<br/>
<br/>        群主跟洛羽的经纪人打好了招呼，让大家一起过去跟洛羽合影。
<br/>
<br/>        合完影后，连星儿和顾千凝拿着洛羽的照片，一起排队等着洛羽的亲笔签名。
<br/>
<br/>        在快轮到她连星儿的时候，连星儿连忙将脖子上的那颗被红绳串着的小纽扣取了下来握在了手心里。
<br/>
<br/>        前面的女孩们一一离去后，她便离洛羽越来越近。
<br/>
<br/>        不知不觉中，连星儿一阵脸红心跳，紧张地做着深呼吸。
<br/>
<br/>        终于轮到她时，连星儿双手抓着照片递给洛羽后，又将那颗白色的小纽扣伸到了洛羽的眼皮子底下。
<br/>
<br/>        “小时候，你救过我，你还记得吗？”连星儿满心期待地说。
<br/>
<br/>        洛羽拿着笔一直是低头签名的姿势，听到连星儿这么说，才好奇地抬起头来看向连星儿。
