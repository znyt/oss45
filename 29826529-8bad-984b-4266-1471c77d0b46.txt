　　“若是白萱师姐喜欢，这块王品灵石，便是送与白萱师姐了。”
<br/>　　楚风眠随意笑了笑，将那块王品灵石递了过去。
<br/>　　反正他在灵泉之中，吞噬了无数灵力，这样的灵石，足足有着两千多块。
<br/>　　这样一块，他还不放在心上。
<br/>　　“不用，这有机缘，也是属于你的，能够成就如今的实力，谁没有多少机缘，我还不至于要你的东西。”
<br/>　　白萱看到楚风眠的举动，随口笑了笑道。
<br/>　　“先走吧，估计人已经到了。”
<br/>　　说着，白萱便是带领楚风眠，来到了这星天峰最大的一处茶楼之中。
<br/>　　“白萱姑娘来了。”
<br/>　　一进茶楼，便是看到两名年轻男子，坐在茶楼之中。
<br/>　　这两名年轻男子，一人乃是一身澎湃水气，显然是一尊修行水系功法的武者，而另外一人，则是一身无双剑意，显然是一尊剑修。
<br/>　　这二人的境界，都是神力境九重的境界，实力都不比白萱要弱，甚至是那尊剑修，比起白萱还要强大几分。
<br/>　　论起实力，已经比起生死境一重的武者还要强大许多了。
<br/>　　这种实力，显然都是出身自不小的势力。
<br/>　　“这位是司空轩，是来自与北域一方庞大武道世家，司空家的天才，实力极强，在我之上。”
<br/>　　白萱见到这二人，也都悄然之间对着楚风眠传音解释道。
<br/>　　这司空轩，便是那一尊剑修，一身剑意，显然达到了极高的层次。
<br/>　　“司空家的人。”
<br/>　　楚风眠眼神中，也露出几分凝重，司空家，在北域也是赫赫有名的一方势力。
<br/>　　虽然说不如北邙学院那般出名，极为低调，但是其中的武者若是不弱。
<br/>　　在武道家族中，司空家都位列前排，只是家族子弟极少，不能跟北邙学院正面抗衡。
<br/>　　但是相传司空家中，也有着生死境高阶，甚至是超越了生死境的强者，所以北邙学院一般也不会去主动招惹司空家。
<br/>　　“而这一位是麟木，是我们北邙学院的内门弟子，在地榜，位列第六位。”
<br/>　　白萱再一次开口悄然传音道。
<br/>　　“这关于颜古龙帝洞府的事，其实就是我跟他们一起发现的，不过跟这二人，我并不熟悉，所以你要小心一点。”
<br/>　　白萱不由的提醒道。
<br/>　　她跟楚风眠，倒是无冤无仇，更何况楚风眠性格分明，白萱对楚风眠一直不错，楚风眠肯定不会恩将仇报的。
<br/>　　但是这二人，却不同，楚风眠这样的人，终究只是少数，真正遇到大机缘，只怕是这二人翻脸的可能性也极高。
<br/>　　所以白萱才会在这里注意提醒这。
<br/>　　武者之间，真正的朋友极少，许多只是在互相利用，为了利益联合，一旦是遇到利益，便是会再次崩塌离析。
<br/>　　“恩。”
<br/>　　楚风眠点了点头。
<br/>　　他虽然不会做出什么恩将仇报的事情来，但是这点防备，楚风眠还是清楚的。
<br/>　　就算是白萱，他也不能完全信任，更何况是这二人。
<br/>　　“白萱，他就是你叫来的帮手？”
<br/>　　那名北邙学院的弟子，麟木，打量了楚风眠一眼，不由的冷哼一声道。
<br/>　　“如果我没记错的话，此人就是这一届的新人王吧，一个小小的外门弟子，神力境四重的境界，这样的人，在虚空海闯荡都费劲，带他去颜古龙帝的洞府，不是找死？”
<br/>　　麟木眉头挑了挑道。
<br/>　　“颜古龙帝的洞府，危险无比，比起虚空海都要危险的多，其中的危险白萱你也见识过的，带他去，不过就是一个累赘罢了。”
<br/>　　白萱听着麟木的话，平静的开口道。
<br/>　　“颜古龙帝的洞府，其中的种种危险，我自然知道，这位楚风眠，可是一个不错的帮手，他的实力，少说也是在地榜前十。”
<br/>　　“地榜前十？”
<br/>　　麟木听着白萱的话，眼神中微微有些吃惊道。
<br/>　　“他才不过神力境四重的境界，能够成为新人王，他的实力的确应该不错，但是地榜前十可不是那么容易的，你不是在开玩笑？”
<br/>　　地榜前十，都是内门的风云人物，许多人都是在修门修炼数十年，甚至是上百年才能爬到的位置。
<br/>　　像是白萱，麟木这等，少数也都是十数年的苦修，才踏入了地榜前十的位置。
<br/>　　如今楚风眠一个这一届的新人，居然白萱说是地磅前十，令麟木如何相信。
<br/>　　“麟木兄的话没错，不是我们信不过白萱姑娘，一个神力境四重的小子，就想要跟我们平起平坐，的确是有些笑话的。”
<br/>　　那司空轩，也是起身，冷冷的开口道。
<br/>　　“实力可不是吹出来的，地榜前十，你至少要展现出自己的实力才行。”
<br/>　　“展现实力？司空兄可要小心了！”
<br/>　　听到司空轩的话，楚风眠冷哼一声，猛然之间，凝聚远古战龙精血，化为战龙之力！
<br/>　　“轰隆！”
<br/>　　猛然之间，战龙之爪便是凝聚，不过这战龙之爪，被楚风眠压缩成了手臂大小，并没有真正的那般庞大。
<br/>　　如同是化为了他的右手，顿时猛然之间，对着司空轩打了过去。
<br/>　　“剑！”
<br/>　　面对这战龙之爪的袭来，司空轩冷喝一声，猛然拔剑，一剑批了过去。
<br/>　　“砰！”
<br/>　　两股力量相撞，一时间居然是不分伯仲，随后战龙之爪的力量，才缓缓的压过了司空轩的剑锋。
<br/>　　在压过司空轩的剑锋一刻，这战龙之爪的力量，被楚风眠一收，便是全然消散了。
<br/>　　“这一剑，只有我三成力量，能够压倒我三成力量，你的实力，至少是地榜前二十的地步，至于地榜前十，也不是没有可能。”
<br/>　　司空轩冷冷的开口道。
<br/>　　“这等实力，勉强足够跟我们平起平坐了。”
<br/>　　“既然白萱姑娘选择的人，倒是不用怀疑。”
<br/>　　说完司空轩便是坐了回去，继续的喝着桌上的茶水。
<br/>　　司空轩的话，自负到了极点，话中的意思，便是他全力之下，想要击败楚风眠，便是轻而易举。
<br/>　　只是看在白萱的面子上，才勉强同意了。
<br/>　　三成实力？勉强足够？
<br/>　　听到这话，楚风眠心中冷笑连连。
<br/>　　他的刚刚的力量，表现出的，甚至不足他真正实力的十分之一。
