陆子羡的神色变化莫测，夏夕绾迅速问，“你怎么了？”
<br/>
<br/>        “绾绾，我听说过你妈咪的名字。”陆子羡据实相告。
<br/>
<br/>        “什么？真的吗？”夏夕绾喜出望外。
<br/>
<br/>        “是真的，不过我也是听说，你给我一点时间，有些事情我需要查明白，到时我再告诉你。”
<br/>
<br/>        陆子羡现在没有办法告诉她，怕她一时之间接受不了，他不想吓到她，而且这件事他需要去查，按道理说林水瑶跟陆家是有交集的，那就不难知道林水瑶在帝都的行踪轨迹。
<br/>
<br/>        夏夕绾是相信陆子羡的，她用力的点头，“恩，好。”
<br/>
<br/>        “绾绾，天色很晚了，我送你回家吧，你现在住在哪里？”
<br/>
<br/>        “我住在…西苑…”
<br/>
<br/>        听到“西苑”这个名字，陆子羡迅速一僵，他当然知道西苑是陆寒霆名下的房产，从海城回来后，他都住在西苑。
<br/>
<br/>        “绾绾，你和陆寒霆又复合了？”陆子羡问。
<br/>
<br/>        夏夕绾摇了摇小脑袋，“并没有，我和他也是不久前才相遇的，而且我们相处的不太愉快，我只是暂时在西苑住两天，奶奶开始的时候病的很厉害，我还在为奶奶调理身体，现在我是夏家的小女儿，我都是住在夏家的。”
<br/>
<br/>        “知道了，那上车吧，我送你回西苑。”陆子羡绅士的拉开了副驾驶的车门。
<br/>
<br/>        夏夕绾看着他俊美的五官，犹豫了一下然后开口道，“陆子羡，你对我的心意，我知道，也很感动，虽然我和陆寒霆已经成为过去式了，但是我现在并不想谈感情，我不能给你希望，更不能心安理得的享受着你对我的关爱，这对于你不公平，所以…”
<br/>
<br/>        她的心思永远这么通透，不愿意给别人机会，陆子羡在想，他终究去迟了，没有比陆寒霆更早的找到她。
<br/>
<br/>        “绾绾，我都知道，但是你不能连做朋友的机会都不给我，对你好是我自己的事，喜欢你，也是我自己的事，不要对我这么残忍，更不要总是将我推得这么远。”
<br/>
<br/>        夏夕绾的心慢慢的软了下来，那好吧，她低头进了迈巴赫的副驾驶座，让他送她回家。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾回到了西苑，因为时间已经不早了，十点多钟了，老夫人和吴妈都已经睡下了。
<br/>
<br/>        她尽量轻手轻脚的，不发生什么声音，可是很快她就停在了书房的门口。
<br/>
<br/>        书房门并没有关上，而是留了一条门缝，她透过门缝看到里面灯火通明的，陆寒霆坐在办公椅上，一手拿着钢笔在“刷刷”的批阅文件，另一只手的修长两指里夹了一根香烟，正在蹙眉抽着。
<br/>
<br/>        青烟缭绕模糊了他的俊颜，不过隐约可以看到他蹙成了一道“川”字的眉心，他偶尔在烟灰缸里叩烟，那烟灰在猩红的火苗上掉落下来，覆了一层森然的戾气。
<br/>
<br/>        他的烟灰缸里已经满满一下的烟蒂了，他抽了好多烟。
<br/>
<br/>        他怎么了？
<br/>
<br/>        心情不好吗？
<br/>
<br/>        夏夕绾已经感觉到他身上沉闷压抑的气场，想到两个人又在车里不欢而散，她打算不理他，直接回自己的房间的。
<br/>
<br/>        可是，她还是转身下了楼，给他亲手煮了一杯咖啡。
<br/>
<br/>        再次来到书房门前，她抬手“叩叩”的敲响了书房门。
