　　九面猫冲他一笑：“从你试图用法术唤醒我的记忆开始，我就已经清醒了，只是一直在装糊涂。”
<br/>　　叶少阳有点无语，回想起来，不得不佩服，她装得真的还是挺像的。
<br/>　　“第二，我们想浑水摸鱼，夺取轩辕剑。”
<br/>　　叶少阳心中冷笑。
<br/>　　“第三……”九面猫望着他，说道，“我们是来保护你的。”
<br/>　　叶少阳怔了一下道：“你快得了吧，你们巴不得我死！”
<br/>　　“那是以前，从现在开始，我们要绝对保护你的安全。”
<br/>　　她的表情看上去不像是开玩笑，叶少阳沉默了一会，道：“为什么。”
<br/>　　“因为——”
<br/>　　“不许说！”钟馗突然吼了一声，用罕见的严厉的口吻冷冷说道：“你若说出这件事，我今日拼着同归于尽，也要先灭了你！”
<br/>　　九面猫冲着叶少阳无奈一笑，钟馗认真起来，她是不敢硬刚的。
<br/>　　本来叶少阳以为九面猫是乱说的，但是钟馗居然表现得如此激动，这反而证明了九面猫没有说谎。
<br/>　　叶少阳看看九面猫，又看看钟馗，“你们到底有什么事瞒着我？”
<br/>　　钟馗微微皱起眉头，说道：“少阳，有些事不是不告诉你，只是知道了，反而没有好处。”
<br/>　　“有没有好处，也是我自己说了算吧，你不告诉我，又不让别人说，这……”
<br/>　　钟馗吹胡子瞪眼，斥道：“随便你怎么说，反正而今不能告诉你，我不说，别人也不能说！否则别怪我动手！”
<br/>　　叶少阳倒是没想到他反应这么大，一时间有点懵，不知道怎么办。大伙也都傻愣愣地站着发呆。
<br/>　　“叶天师，你就不要问了，钟大士这么说，反正我是不敢说了，总之你知道我没骗你就行了，咱们还是要团结一致的。”
<br/>　　叶少阳还在犹豫，林三生走过来，趴在他耳边低声说道：“今天是问不出了，等将来出去之后有的是机会，我们没时间耽误了。”
<br/>　　叶少阳抬头望着九面猫，道：“如果我不跟你们合作呢？”
<br/>　　“那就只好拼死一战，纵然我们人少，不是对手，也得伤你们几个人，或者我们先逃走，等你们进去之后再跟进去……你们放心让我们跟在后面吗？”
<br/>　　叶少阳无语，转头看着林三生。
<br/>　　林三生对九面猫说道：“可以合作，但有个要求，你们要先进去。”
<br/>　　九面猫道：“这个好说。”
<br/>　　林三生于是对他们招了招手，九面猫带着十来个手下走过来，双方面对面，都十分警惕着对方，一旦有什么风吹草动，立刻就是一场你死我活地战斗。
<br/>　　“你瞅啥？”小马面前站着的是个农夫打扮的男子，一直盯着他看，小马大为不爽，抡起板砖说道：“不许瞅我，听着没有，再瞅我特么削你！”
<br/>　　那“农夫”强压怒火，将脸转到一边去。
<br/>　　叶少阳将核桃交到九面猫手上，另一只手却抓着她的手腕不放，边上吴嘉伟和碧清也都围上来，万一九面猫想要拿着东西逃跑，或者玩什么花样，立刻就会遭到他们集火攻击。
<br/>　　九面猫看着那核桃，深吸一口气，握紧之后，运用修为，对核桃进行压迫，里面的九颗珠子立刻上下活动起来，放出了五颜六色的光。
<br/>　　九面猫将神识引到上面，立刻从里面产生了一股强大的吸力，将她整个人瞬间吸了进去。核桃凭空掉落，被九面猫一个手下接住，一样照做，人影也被吸了进去。就这样一个个地都进入了九星阁。
<br/>　　叶少阳突然想到是不是要拖延一下时间，让他们在里头先顶着，消耗一下，小马突然一把抢过去，道：“我先来！”
<br/>　　几秒钟后，小马进去了。
<br/>　　叶少阳一行人只能紧随其后，钟馗是最后一个，然后核桃落在了曲波手中，腾永清、水波等人警惕地守在边上，保护着核桃。因为实力不够，他们没有被带进去。
<br/>　　几个人互相望着，一个个神色都格外紧张，谁也不知道，叶少阳他们在九星阁中会遭遇什么，更不知道他们等来的会是一个什么样的结局。
<br/>　　什么都做不了的他们，也只有乞求神明来安慰自己了。
<br/>　　汪洋大海，上面漂浮着一座岛屿，差不多有两个足球场那么大，四周浊浪滔天，不时有海浪拍打在岛屿的四周，却似乎被无形的结界给挡住了。
<br/>　　天地间混沌未开，岛屿之外的海上，到处弥漫着雾气，让人看不到远处。天也是黑的，有气息滚动，宛如旋流一般，整个空间里风云涌动，充斥着诡异而肃杀的气氛，身处在这座岛屿上，立刻就感觉到一阵莫名的紧张。
<br/>　　叶少阳他们就降落在岛屿的一头，落下之后，朝前方看去，九面猫一干人就站在他们稍微靠前的位置，没敢轻举妄动。
<br/>　　所有人默不作声，一起朝岛屿另一头望去，天昏地暗，但前面有一排九盏灯照明，灯光的颜色各不相同，那迷迷蒙蒙的感觉，让叶少阳想起了澡堂里面的水银灯。
<br/>　　这九盏灯挂在半空，下面连着一道道的长条形状的黑影，应该是架设在什么东西上。大伙仔细看去，顿时倒吸冷气：是九座人形雕塑，脸庞一致朝着他们这边，“灯光”就悬在它们的头顶上方。
<br/>　　九尊雕像，全都是星月奴，一模一样，都是身穿霞衣、头戴法冠，脸上带着佛像专有的那种悲天悯人之色，同时又透着一股至高无上的威严。
<br/>　　让人一眼看上去，就会有一种想要臣服的压力。叶少阳这一干人全都不是善男信女，自然不会想要臣服，但这些雕像还是给他们带来了一些内心的压力。
<br/>　　在这些雕像的身后，是一团迷雾，什么都看不见。
<br/>　　几个人再仔细看这雕塑和上面的灯光，这些发现那不是灯光，而是一缕缕的气息凝聚在一起、所形成的光球，悬浮在雕像的上端。
<br/>　　细心的叶少阳观察了一下这些雕像站立的位置，发现正好暗合九宫之位，九宫在道教法术之中，是一种根据最基础的理数组成的最为朴素的阵法，不主攻击也不主防御，是一种聚气养息的法门，只要运用得当，有着无穷的理数变化。
