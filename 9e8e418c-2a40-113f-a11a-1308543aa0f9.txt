        “补偿？什么补偿？”连星儿一脸懵。
<br/>
<br/>        妇人眼睛一红，突然掉起了眼泪，哭诉道：“连同学，你可不可以撤诉？可不可以念在你和我家小溪同学一场的份上，宽容小溪，不要让小溪去坐牢啊？虽然只有三个月，但是她这一进去，往后的人生就被毁了一大半了。”一旁的中年男子连忙帮腔道：“是啊！我们就这么一个女儿，平日里被我们宠坏了，没把她教好。她所做的那些事情，我们代她向你道歉。”
<br/>
<br/>        中年男子这话一落，他便从沙发上起身，直接在连星儿的腿边跪了下去。
<br/>
<br/>        连星儿吓得连忙起身将中年男子从地上拉了起来：“叔叔，您别这样啊！”
<br/>
<br/>        “我也给你下跪，好不好？求求你，别让我女儿去坐牢。”
<br/>
<br/>        连星儿刚拉起这个中年男子，转眼间，那个妇人说着哀求的话，也起身在她面前跪下了。
<br/>
<br/>        “阿姨，你也别这样啊！”连星儿苦着脸，左右不是地说。
<br/>
<br/>        她放开了中年男子，不得不又去将那妇人给拉起来。
<br/>
<br/>        “连同学，我知道你心地善良。阿姨和叔叔，都求你了好不好？别让我女儿去坐牢。”妇人一把鼻涕一把泪，看着连星儿苦苦哀求道。
<br/>
<br/>        连星儿皱着眉头，无奈地回答道：“这事，我帮不了你们。你们还是回去劝劝朱小溪，要她在里面好好表现，争取早点出来吧！”
<br/>
<br/>        被连星儿这么一拒绝，妇人的情绪顿时激动了起来：“我以为你会是个善良的孩子，没想到，你是个记仇的贱丫头！”
<br/>
<br/>        “随你怎么想！”连星儿斜睨了妇人一眼，打算转身离开。
<br/>
<br/>        不料，她的手臂却被这妇人给紧紧地抓住。
<br/>
<br/>        “我不许你走！你不答应放过我女儿，我就不让你走！”妇人的情绪越是激动，她手下的力度便越大。
<br/>
<br/>        连星儿见这妇人的红指甲快要掐破自己手臂上的皮肤，疼痛感传来时，她下意识地推了妇人一把。
<br/>
<br/>        明明她没有用很大的力气，但是那个妇人却顺着她甩手推出去的方向往后仰。
<br/>
<br/>        一旁的中年男子更是反应迟钝，眼睁睁地看着妇人的后脑勺，直接撞在了玻璃方桌的尖角上。
<br/>
<br/>        顿时，一滴、两滴、三滴……鲜红的血液从妇人的后脑勺处顺着玻璃方桌的尖角掉落在了木质地板上。
<br/>
<br/>        “老婆！啊——老婆！”中年男子不禁瞪大了眼睛，惊慌失措地大喊了起来。
<br/>
<br/>        连星儿整个人傻眼了，愣了一下，才慌忙掏出手机拨打急救电话。
<br/>
<br/>        随后，朱小溪的母亲被送往了医院。
<br/>
<br/>        连星儿陪同着中年男子一起上了救护车，途中，这中年男子还拿食指指着连星儿的鼻子，对着她大大咧咧地谩骂：“你给我等着！我一定要告你！我一定要告你！让你也去坐牢！”
<br/>
<br/>        “……”
<br/>
<br/>        连星儿眉头紧锁，默不吭声。
<br/>
<br/>        就在中年男子辱骂她父母的时候，她终于忍不住怒斥了一声：“你给本大小姐闭嘴！”
<br/>
<br/>        中年男子顿时被连星儿的气势吓得一句话也骂不上来了。
<br/>
<br/>        得知自己母亲住院的朱小溪，在两个警察的陪同下来到了医院。
<br/>
<br/>        手术室门外，朱小溪一看到连星儿，就像发了疯似的向连星儿扑了过去，幸亏两个警察及时将她拉住。
<br/>
<br/>        “连星儿，你这贱女人！竟然敢推我妈妈！我一定不会放过你的！死也不会放过你！”朱小溪的眼睛里布满了血丝，瞪着连星儿咬牙切齿地怒吼。
<br/>
<br/>        连星儿懒得理会朱小溪，只是担忧手术室里的朱阿姨。
<br/>
<br/>        中年男子站在一旁，背靠着墙壁，独自默默地抹着眼泪，嘴里还不停地念叨：“家门不幸、家门不幸！”
<br/>
<br/>        随后没过多久，傅夜沉便带着自己的助理赶了过来。
<br/>
<br/>        连星儿看到傅夜沉时，并不意外，虽然她没给他打电话，但是她身上的一点风吹草动，她爹地倒是一清二楚。
<br/>
<br/>        都是当父亲的人，中年男子在看到傅夜沉后，不禁自惭形秽了。
<br/>
<br/>        特别是他身后跟着的四个黑衣保镖，气势足以将旁人给怔得不敢轻易靠近。
<br/>
<br/>        “恒信，送少小姐回学校。”傅夜沉对身旁的助理郑恒信冷冷地命令道。
<br/>
<br/>        郑恒信颔首应道：“是！”随后，他走到连星儿的跟前，毕恭毕敬地说道，“少小姐，我送你回学校。”
<br/>
<br/>        连星儿一脸委屈地瘪着嘴，聋拉着脑袋跟着郑恒信先行离开了。
<br/>
<br/>        傅夜沉直接走到中年男子面前，向他礼貌性地伸出了手来：“你好，我是星儿的父亲，有什么事情，你直接跟我提即可。”
