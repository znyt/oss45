　　孙涛走上前，跟齐凡配合着把笨重的卷帘门拉起，其他人迅速从卷帘门下钻过；罗昊和莱昂两个人钻过卷帘门后接替齐凡和孙涛把卷帘门托住，让两个人钻过卷帘门，轻轻放下卷帘门。
<br/>　　穿过卷帘门，战术中队快速沿着走廊快速奔跑，从走廊尽头的转角转过，朝着地下停车场跑去。
<br/>　　战术中队刚从走廊尽头的转角转过，一队荷枪实弹的机动部队队员便是从卷帘门前小跑而过，在经过卷帘门的时候，有机动部队队员扭头看了眼卷帘门，然后继续小跑着往前，他们并没有发现卷帘门的异常。
<br/>　　这个博物馆的地下停车场原先是巴坦伦停放他豪车的地方，而现在停车场中的那些豪车已经不见了踪影，整个停车场中空荡荡的看不到一辆汽车。
<br/>　　罗昊、齐凡等人手中端着从“黑箭”队员手中抢来的M4A1战术步枪，眼神警惕的打量着停车场四周，脚下踩着战术步伐向前移动着。
<br/>　　停车场的入口也已经是被卷帘门封闭，停车场入口处的卷帘门不是之前那种可以看到锁孔的地锁，而是通过电脑控制的埋地锁，所以想要用子弹把锁打烂基本是不可能做到的。
<br/>　　罗昊拿出自己从“黑箭”队员身上顺来的枪榴弹塞进挂在M4A1战术步枪下的M203榴弹发射器中，对着紧闭的卷帘门轰了过去，把卷帘门炸开了一个洞。
<br/>　　“走！”
<br/>　　罗昊收起M4A1战术步枪，朝着停车场门口跑去，拉开被炸得翻卷过来，钻过卷帘门，在底下停车场外建立起防线。
<br/>　　榴弹的爆炸声吸引了博物馆内机动部队队员和force-recon队员的注意。
<br/>　　罗昊一个向前扑滚，对着冲到博物馆玻璃窗前的机动部队队员和force-recon队员扫出一个长点射，然后迅速跑到一辆汽车的引擎背后，退出手中M4A1中打空的弹匣，换上一个备用弹匣，推弹上膛。
<br/>　　哒哒哒哒哒……
<br/>　　铿铿铿铿铿……
<br/>　　密集的子弹扫射而来，步枪子弹毫不费力的就打穿了被罗昊当做掩体的轿车，车身被子弹穿出一个个透光的弹孔来，也幸的罗昊藏在轿车引擎背后，如果是藏在车门背后，现在他就已经子弹打成了马蜂窝。
<br/>　　一排子弹打在汽车引擎盖上，将汽车引擎打得起火燃烧起来。
<br/>　　罗昊扭头看了眼从引擎盖上蹿起的火焰，嘴里骂了一句，快速从汽车背后离开，转移到下一个掩体处。
<br/>　　轰——！
<br/>　　引擎起火的汽车很快就发生了爆炸，火球从引擎盖下升腾而起，把整个引擎盖都炸飞起来，火焰迅速包裹住了轿车，滚滚的黑烟直窜天空。
<br/>　　借着轿车燃烧黑烟的掩护，罗昊爬上一辆架设着车载机枪的吉普车，拉开枪栓，转过枪口对着博物馆窗口前的机动部队队员和force-recon队员扫射。
<br/>　　哒哒哒哒哒……
<br/>　　车载机枪枪口吐出一道火舌，7.62mm口径的步枪子弹汇聚成一柄火焰之剑斩向博物馆中的那些机动部队队员和force-recon队员。
<br/>　　滚烫的弹壳夹杂着弹链上塑料支架的碎片向外弹射，有些直接飞出了吉普车，有些则是滚落进吉普车车厢内。
<br/>　　博物馆内的机动部队队员和force-recon队员被罗昊用车载机枪压得不敢抬头，一名机动部队队员刚露头想要偷鸡，一颗子弹飞来削掉了他的天灵盖，血浆就跟喷泉一样直冲被粉刷的雪白的天花板。
<br/>　　一名FoRCe-ReCoN队员抱着一具AT4反坦克火箭筒矮着身体走到窗户边，在其他人的掩护下，起身瞄准了罗昊所在的吉普车。
<br/>　　“你妹！！”
<br/>　　罗昊对着那个扛着火箭筒的force-recon队员咒骂了一句，迅速跳下了吉普车。
<br/>　　咻——！
<br/>　　轰——！
<br/>　　罗昊刚从吉普车上跳离，火箭弹便是拖着一根火焰尾巴呼啸而来，击中了吉普车，剧烈的爆炸声中将吉普车炸成了一团火球。
<br/>　　崩飞而来的玻璃碎片在罗昊脸上和挡在脸前的手臂上划开几道口子，机密的血珠立刻从那几道口子中涌了出来。罗昊没有去管脸上和手臂上这些伤口，挣扎着从地上站起来，一边用点射对着博物馆中冲出来的机动部队队员射击着，一边朝着旁边另外一辆防弹突击车移动过去，打算以那辆防弹突击车作为自己的掩体。
<br/>　　吱——！
<br/>　　在汽车轮胎跟地面尖锐的摩擦声中，玄龟和天蛇驾驶着防弹突击车冲向博物馆的后门。
<br/>　　两辆突击车上，张子航和刘子锦各控制一挺车载机枪对着冲出博物馆的机动部队队员进行扫射。
<br/>　　子弹扫过，那些机动部队队员在子弹的洗礼下抖动着，身上爆出一团团血雾。
<br/>　　“上车！”
<br/>　　布谷跳下吉普车，用M200狙击步枪精确狙杀着博物馆中的重火力点。
<br/>　　“手雷！”
<br/>　　罗昊大声提醒了一句，甩手扔出一颗手雷，压制住还在不断往外冲的机动部队队员，快速朝着布谷的方向跑去。
<br/>　　齐凡、莱昂等人全都钻进防弹突击车后，天蛇和玄龟立刻驾驶着防弹突击车退到街道上。
<br/>　　防弹突击车上，罗昊、齐凡等人又是扔出几颗手雷把停在博物馆外的汽车给炸毁，阻止了博物馆中机动部队队员和force-recon队员驱车追击。
<br/>　　齐凡拿起放在防弹突击车中的步话机对梅庆伟说道：“宿鸟，我们已经从博物馆中突围成功，按计划在A1汇合。”
<br/>　　“收到，A1汇合！”
<br/>　　梅庆伟驾驶着“支奴干”直升机从城市上空飞过，朝着汇合点飞去。
<br/>　　两辆防弹突击车在街道上狂奔，埋伏在街道两边楼房中的机动部队这个时候也是纷纷露头，用自动步枪对着街道上跟发了疯一样的防弹突击车进行射击。
<br/>　　绝大部分子弹全都是落空，少许子弹打在防弹突击车的车身上，迸溅起几点黄色的火星，在防弹玻璃上留下几个白色的印痕。
<br/>　　两辆防弹吉普车从街道转角处转过，转入一条宽敞的大路，天空中，梅庆伟驾驶着“支奴干”运输机降低高度，同时运输机尾部的舱门也是被打开，打开的舱门拖在马路上，在地面上留下一串火星。
