　　第389章
<br/>　　李行哉迎风而立，鬓发轻扬，程大雷第一发现，原来他竟生得如此英俊。
<br/>　　像一位皇子了。
<br/>　　“长安的形势很复杂，即便你去了，也未必能解决什么。”程大雷。
<br/>　　“这不是关键，关键是我必须出现在长安，直面逆贼。”李行哉顿了顿：“我知道林少羽是你兄弟，但他不是我兄弟，战场之上无情分，或许，我会杀死他，也或许，他会杀死我。无论如何，这件事和你无关。”
<br/>　　程大雷点点头，示意自己明白。什么样的身份，该做怎样的事，听起来又是明帝那一套。果然是明帝的子嗣，他体内依旧流着李家的血。他们的祖先曾经赤手空拳打下整个天下，李家的后人也从来不是懦夫。
<br/>　　“一路保重。”程大雷双手抱拳。
<br/>　　李行哉抱拳还礼：“照顾好婉儿，她的命已经死在草原上，之后的命是你救的，她的余生安稳，自然要你负责。”
<br/>　　“这个自然。”
<br/>　　“走喽。”
<br/>　　李行哉手重重拍在福德勒的肩膀上，福德勒正在收拾棋盘，抬起头脸上表情一怔。
<br/>　　两人出了琴川关，程大雷没有送，就连李婉儿也尚不知道李行哉离开的消息。
<br/>　　李行哉和福德勒骑马出关，任所有人看来，他们都像是去野外郊游打猎一般。可二人却是一去不回头，自从今日离开，怕再也不会回到这块土地。
<br/>　　即便回来，也绝不是以朋友的身份。
<br/>　　钟伟虎带人早在琴川关外等待，见到李行哉出现，一众人同时俯下身，以头碰地。
<br/>　　“属下见过皇子殿下。”
<br/>　　“好啦，走罢。”
<br/>　　“殿下，要不要换过衣服，属下备好了轿子，宋大人此刻正在凉州城等待。”钟伟虎。
<br/>　　李行哉在自己身上扫了一眼，不禁也哑然失笑。此刻自己身上还穿着蛤蟆寨的制服，背后纹着一只丑陋的蛤蟆。
<br/>　　这是蛤蟆寨的标志，却不该是帝国皇子的标志。
<br/>　　“衣服在哪里？”
<br/>　　“都已在轿子中。”
<br/>　　从轿子上下来两个宫装少女，看年纪至多十六岁上下，并肩站在一起，款款向李行哉道了个万福。
<br/>　　李行哉哈哈一笑，从马上跃下，揽着二人肩膀步入轿子。
<br/>　　不多时，他已由人伺候着更衣完毕，再从轿中出来，已经完全变了个模样。
<br/>　　身着锦绣黑袍，上纹龙云，吞云吐雾，下身是左右分开的八宝立水裙，站在荒凉的野地，却凭空有种富贵之气。
<br/>　　不仅富，而且贵，富贵逼人的贵。
<br/>　　……
<br/>　　送走李行哉后，程大雷还不知怎么向李婉儿解释，此刻，琴川关又迎来了新的客人。
<br/>　　一行三人，打马行在琴川关的土地上。
<br/>　　道路两旁散落着民居田地。如今已过了秋收，田里的事物都闲下来。偶尔还有人在田野中打猎，抓个兔子、狍子什么的改善伙食。
<br/>　　看到这三人，大家都露出好奇的眼神，忍不住张望。碰到他们眼睛看过来，也不避让，直勾勾的看过去。
<br/>　　琴川关的日子总还算不错的，每人都能吃饱饭。因为背后的老大是程大雷，所以这些人也有些普通农夫没有的底气。
<br/>　　别处吃不饱饭，这里可以吃饱饭，别人担心战火，却没人敢打程大雷，这便是底气的来源。
<br/>　　为首一人，身穿儒生长袍，看到这一幕微微点头。
<br/>　　“琴川关果然是个好地方，不愧有塞北江南之称。”
<br/>　　“若能拿下此地，我们的粮草就无忧了。”
<br/>　　“那琴川之主程大雷，怕是不太好说话。”
<br/>　　“走，去会会他。”马上人大笑。
<br/>　　结果，还未进入琴川关城门，他们就被拦下了。
<br/>　　“你们是什么人，想要入关，去那边登记，写明来历，所携带物资，才可入关。”
<br/>　　守门小卒大声道，他看着面前三人，心情不是很痛快。琴川关来来往往的人不少，本地居民有通行证，不用查验的太过仔细，但前来做生意的商队，都要盘查清楚后，才能放入关内。
<br/>　　因为和珅治下甚严，谁如果敢吃拿卡要，直接就是四十大棒。说也奇怪，以和珅和大人的性情，反倒最痛恨别人贪墨。
<br/>　　而以前来往的商队也都是特别客气的。虽然说，琴川关是个讲规矩的地方，但琴川关的主人程大雷却未必是个讲规矩的人。
<br/>　　但今天，这三人到了城门口，竟然还不下马，一副趾高气扬的态度，实在令人看了生气。
<br/>　　马上人微微一笑：“去告诉程大雷，莫将难来了，让他出城迎接。”
<br/>　　“莫将难是谁，不认识。不管什么人，入城都得登记。”守门小卒道。
<br/>　　“你在西北，连莫将难都不认识？”
<br/>　　“我管你姓什么，要不下马，要不滚蛋，再敢废话，我直接抓了你丢进大牢。”
<br/>　　“或者，我需要你认识认识了。”
<br/>　　莫将难手摁刀柄，已经打开了绷簧。
<br/>　　他用的是刀，而不是剑，看虎口的茧子，应该是个用刀的老手。
<br/>　　“手持利械，意图闯关，来人，与我擒了。”城门队长大叫。
<br/>　　“大哥，我们是来谈事的，不可伤人。”莫将难的一名随从道。
<br/>　　“明白。”
<br/>　　莫将难将刀还鞘，以刀鞘砸人。城门十人组的卫兵小队，竟然拦不住他一个。只听砰砰声响，眨眼之间已砸倒一大片人。
<br/>　　“什么琴川关，我看也不过如此么。”莫将难在马上冷笑。
<br/>　　城门队长脸胀得通红，忽然抓起长矛，从地上跃起向莫将难扎来。
<br/>　　莫将难盯着对方动作，已暗动杀机，一刀斩出，砸向对方脑袋。刀虽然仍在鞘中，但如此砸中头颅，对方不死也得重伤。
<br/>　　是该给他们一些颜色看看了。
<br/>　　突然，一柄长枪中中间刺过来。拨开城门队长的长矛，挡住莫将难的刀鞘。莫将难只感觉一股巨力顺着刀身传来，震得他手腕有些发麻。
<br/>　　回过神来，看到面前离着一黑脸大汉。虎背熊腰，木纳一张脸像是岩石。
<br/>　　“蛤蟆寨，秦蛮。”秦蛮以枪撑地：“阁下为何伤我山寨弟兄？”
<br/>　　莫将难收起脸上狂意：“我乃万马堂莫将难，特地来见程大雷，让他出来见我。”
<br/>　　“万马堂莫将难？”秦蛮摇摇头：“没听过。”
<br/>
