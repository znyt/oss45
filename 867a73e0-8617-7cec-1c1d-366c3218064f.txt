        &lt;h3&gt;第三卷第一百六十三章  朋友情&lt;/h3&gt;
<br/>
<br/>        陈子昂和寒伯安他们合作的软件开发已经进行了一半，因为公司的人手调配不开，之前有合作的软件公司，他们商议了决定和别的公司一起合作。
<br/>
<br/>        今天是第二次坐下来商定合作的意向，事关寒伯安公司后期发展的走向，寒伯安、郝景文、江俞轩他们都到了。
<br/>
<br/>        他们都是公司的大股东，尤其是寒伯安对这次合作看得非常重视，陈子昂亲自将规划做好，在公司经过了一次又一次的论证。按照寒伯安的意愿进行一次又一次的修改，预留了几家地图软件商的接口。
<br/>
<br/>        设置了租车服务，叫车服务、接机、送机。并按照寒伯安的要求设置了各种套餐。
<br/>
<br/>        陈子昂经常出差，面对着出行难有着深刻的体会，她是从一个小山村出来的人，对各种消费群体也有着深刻的认识，定位清晰，综合大家的意见，加上之前有过物流app的借鉴，一套完整的体系图出现在大家的面前，陈子昂让赵绾绾来讲解。
<br/>
<br/>        在赵绾绾的讲解下，大家都纷纷点头，有了这套方案，大家一目了然需要干什么，这样有了目标行动起来就事半功倍。
<br/>
<br/>        而和合作方谈论的也非常顺利，他们让出了开发软件的著名权，合作方直接安排人来他们公司进行软件开发，负责人依旧是赵绾绾。
<br/>
<br/>        寒伯安他们也没有异议。
<br/>
<br/>        陈子昂看着大家都很满意，常舒了一口气，天知道她花费了多少时间来做这件事情，每天都在考虑怎么做，每次出行都会想到这件事情，在大家的努力下终于确定下来了。
<br/>
<br/>        ....
<br/>
<br/>        寒伯安：“陈子昂，你是怎么做到的？”
<br/>
<br/>        “我经常出差，知道我自己的需求，而且你的意见给了我很大的启发，我出门的时候都会想着这个问题，再加上综合大家的意见，时间长了，就在上一次的方案上反复进行修改和论证，这样就成了。还有，这一次是找人合作，如果思路不清晰就会形成枝丫太多，最后费时费力，还拿不到自己想要的东西。”陈子昂笑着解释。
<br/>
<br/>        “你这脑洞也很强大嘛，和我们安男有得一拼了。”寒伯安提起老婆心里都是蜜。
<br/>
<br/>        “大哥，你能不能不要走哪儿都要提起你的老婆，那个优秀的人你都要和你老婆比一下。”郝景文笑着说。
<br/>
<br/>        “不能，优秀的人不怕比较，是吧，陈子昂。”寒伯安继续秀。
<br/>
<br/>        “文文，你也加油啊，要不然吃狗粮这种事情会经常发生。”江俞轩怎么能不知道郝景文那点小心思。
<br/>
<br/>        “这能是说加油就加油的事情吗？”郝景文有些无奈，最近约某人都约不到了。
<br/>
<br/>        陈子昂看了一眼郝景文，想到了陈思宇。
<br/>
<br/>        “俞轩，中午在公司吃还是出外面吃？”陈子昂看看时间，快到饭点了。
<br/>
<br/>        “去外面吃吧，我们四个人好久都没见了。”公司吃饭人太多，再加上大哥很少和那么多人在一起吃饭。
<br/>
<br/>        “不是经常开网络会议吗？”郝景文很想在公司吃，在公司吃是不是还可以见到陈思宇？见到陈思宇他要好好问问她，为什么不接他电话，不回他信息？
<br/>
<br/>        “那我叫上陈思宇吧，我今天有点事情想和大家说。”陈子昂确实是有事情想和大家说，那一次孩子认爸爸的时候就他们几个人，张倩楠和安男不用叮咛，有寒伯安和江俞轩在。
<br/>
<br/>        但是陈思宇她还是得叮咛一下的。
<br/>
<br/>        郝景文听说叫陈思宇就看了一眼陈子昂，莫非她知道他心里的想法？难道我有这么明显吗？
<br/>
<br/>        陈子昂打电话叫来了陈思宇，五个人开着两辆车出去吃饭。
<br/>
<br/>        江俞轩和寒伯安不约而同的上了陈子昂开的车，而陈思宇看了一眼郝景文，将抬起的脚又放下，走到了郝景文的车上。
<br/>
<br/>        “你故意叫上陈思宇？”江俞轩问陈子昂。
<br/>
<br/>        “嗯。”陈子昂并没有解释什么。
<br/>
<br/>        “上一次叫上陈思宇是文文要求的吧?”寒伯安也笑着问陈子昂。
<br/>
<br/>        “你们觉得他们俩个有可能吗？”陈子昂反问寒伯安和江俞轩，她也想替陈思宇问问这俩人的意见，毕竟他们在一起的时间很长，更了解郝景文。
<br/>
<br/>        江俞轩：“没想到你还有闲心当媒婆，哈哈哈哈。”
<br/>
<br/>        “没有，只是我今天真的有事情，上次孩子认爸爸的时候，陈思宇也在。”陈子昂郁闷的说。
<br/>
<br/>        “怎么了子昂？”寒伯安的语气透露着关切。
<br/>
<br/>        “张函去我家了。”陈子昂叹了口气。
<br/>
<br/>        “嗨，他不会是想和你复合吧？想复合也得把问题解决了啊。”寒伯安感觉不可思议。
<br/>
<br/>        “不是想要和我复合，只是我觉得孩子的事情还是不能让太多的人知道，我怕会有麻烦。”陈子昂无奈的说，要不是财叔提醒她，她也不会去想那么多。
<br/>
<br/>        “你二嫂一直和张函的母亲有来往，你知不知道？”江俞轩问陈子昂，他都忘记提醒陈子昂了。
<br/>
<br/>        “她和陆家张家好像都有来往，至于关系怎么样，我也不太清楚。”陈子昂叹了口气，冯睿和二哥离婚了，二哥到老了还离婚。
<br/>
<br/>        “你二嫂将我们的事情告诉了夏颜。夏颜找到我，洛城的项目我们没有参与。当然，冯睿说孩子是我的。”江俞轩这句话虽然说的没有逻辑，但是陈子昂明白了，上一次他们在一起吃饭，江俞轩说他和她都有了孩子，没有想到冯睿真的相信了并且告诉了夏颜，夏颜用这件事情来威胁江俞轩。
<br/>
<br/>        “那她也有可能会告诉张家和陆家的。”陈子昂深深的叹了口气，这些人就和狗皮膏药一样，让她甩不掉。
<br/>
<br/>        江俞轩：“也好，这样他们就不会注意到张函是孩子他爹了。”
<br/>
<br/>        “轩轩，你以为你这锅背得自在？如果真的传出来你和陈子昂有孩子的说法，怕是对你们都不利。”寒伯安听了半天，才明白陈子昂担忧的问题，在这件事情上江俞轩是绝对不能背锅的，何况中间还夹着张倩楠。
<br/>
<br/>        “我决定将孩子先送到国外去。”陈子昂现在也不把寒伯安当旁人，有什么事情不回避寒伯安。
<br/>
<br/>        财叔提醒了她，有些事情还是应该找朋友商量一下，似乎在这个世界上，除了江俞轩和寒伯安他们几个，她竟然没有地方可以商量的。
<br/>
<br/>        寒伯安：“你准备送哪里去？”
<br/>
<br/>        “我落叔叔在欧洲有庄园，我想将孩子放那里，我能放心一些。”陈子昂知道落家要和吴家联姻了，她将孩子放在落家，其实也是变相的寻求保护。
<br/>
<br/>        寒伯安：“那我给我弟弟说一声吧，他们最近就在欧洲庄园。”
<br/>
<br/>        “啊，那太好了，我等耗材正式推向市场，我就将孩子送过去，这几天就不打扰他们了。”陈子昂是知道落雪的行踪，这一两年落雪慢慢的浮出水面，她们也偶尔有联系，只是不多，落雪似乎是有意在回避她。
<br/>
<br/>        “陈子昂，公司最大的股东是落雪？”寒伯安问陈子昂。
<br/>
<br/>        “是的！当初开这个公司是她出的资金，我什么都没有。”陈子昂心怀感激，阴差阳错中她遇到落雪，改变了她的一生，没有落雪她现在会是什么样子的呢？
<br/>
<br/>        “我很好奇，她出了多少资金？”寒伯安好奇的问陈子昂。
<br/>
<br/>        “两万，她攒的私房钱。”陈子昂微笑着说，当初那个女孩的话啊，给了她希望，也给了她的动力。
<br/>
<br/>        “买噶，陈子昂，你是疯子吧？”寒伯安吃惊的问。
<br/>
<br/>        “大哥，你不知道吧，公司里的人都叫她是疯子。”江俞轩补充了一句。
<br/>
<br/>        “你公司早都可以上市了，迟迟不上市是因为等落雪？”寒伯安没有理会江俞轩而问陈子昂。
<br/>
<br/>        “什么都瞒不过大哥。”陈子昂确实在一直等落雪。
<br/>
<br/>        “别等了，浪费大好时光，落雪不会管你上不上市，她顾不上。”寒家和落家是至交，他弟弟就是在落家。
<br/>
<br/>        陈子昂：“我已经让凤盛夏在做了，前期工作都已经做好了，随时都可以的。”
<br/>
<br/>        “不要纠结在一个点上了，虽然别人帮助过你，但是没有你自己的努力，也是不行的。加油吧，陈疯子！”寒伯安此时刷新了对陈子昂认识。
<br/>
<br/>        这个女人确实很疯狂，他们起步是有家族垫底的，而陈子昂.........，寒伯安摇了摇头。
