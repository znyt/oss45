　　佟千雪现在根本闻不得油烟味，所以中午饭自然是慕容飞做的。
<br/>　　考虑到佟千雪的情况，慕容飞特意做了一些清淡的菜，还从冰箱里拿了水果给佟千雪榨了一杯果汁。
<br/>　　可即便如此，佟千雪还是没有表现出什么胃口，喝了几口果汁，在慕容飞的要求拿起筷子去夹菜，可是菜刚入口，便是一声干呕，然后起身跑进了浴室，对着马桶吐得稀里哗啦。
<br/>　　听到从浴室中传来的阵阵干呕声，慕容飞也是完全没有了完全没了胃口，放下碗筷起身走进浴室，看着坐在浴室地上，抱着马桶吐得快要虚脱的佟千雪，把她从地上扶了起来，递给她一杯水，问道：“没事吧？”
<br/>　　佟千雪轻轻摇头，接过水杯漱了漱口，说道：“这段时间都是这样，晚上的时候还能好点，中午的时候基本闻到食物的味道就想要吐。很多人都说女人孕吐越厉害，越是容易生男孩，可是我妈当年生我和我姐的时候就吐得很厉害，而我姐怀小飞的时候，就没怎么吐，别人前三个月几乎都有孕吐反应，她第二个月开始就很少会吐，那时候都以为她怀的是个女孩，祁伟坚的父母当时是一脸的不高兴，谁知道最后生下来的是个男孩。”
<br/>　　“我就纳闷了。”佟千雪放下水杯，双手抱着胸，看着慕容飞说道：“这生男生女跟我们女人又没关系，生不出男孩凭什么赖在我们女人身上，真是不公平！”
<br/>　　“你别看我，我从来没这么想过。”慕容飞对着佟千雪连连摆手。
<br/>　　“这没这么想过？”
<br/>　　佟千雪一脸不相信的盯着慕容飞。
<br/>　　“真的没有！对于我来说，生男生女都一样，我还倒是更喜欢女儿多一点。狐狸、狂龙他们都生了女儿，还特别漂亮，羡慕死人了。”
<br/>　　“他们都是女儿？”佟千雪问道。
<br/>　　“狐狸第一个是女儿，现在这个还在肚子里，不知道。”慕容飞回答道：“狂龙那家伙就比较气人了，现在不孕不育的人那么多，很多人要一个孩子都很难，这家伙倒好，一下子生了俩，还是龙凤胎，儿子女儿都有了。”
<br/>　　千里之外，京城高档别墅小区，正在吃饭的罗昊狠狠打了一个喷嚏，碗里的汤都洒了出来，惹来姐弟俩一阵鄙视。
<br/>　　佟千雪美眸盯着慕容飞看了好一会儿，随后说道：“最好你说得是真的，要不然，哼哼！”
<br/>　　佟千雪冷哼了两声，伸手做了一个切刀的动作，威胁道：“我切了你！”
<br/>　　慕容飞往后退了一步，看着佟千雪说道：“你刚才自己说的，不能让我憋坏了，现在又要切了我，切了我，你下半辈子怎么办？”
<br/>　　“玩具！”
<br/>　　佟千雪酷酷的丢下一句回答，转身走出了浴室。
<br/>　　看着佟千雪的背影，慕容飞心中暗暗嘀咕，女人啊，真是一种善变的生物。
<br/>　　杭山市夏天的雨季除了潮湿，就是闷热，就算是待在家里什么都不做，身体中的水分都会不受控制的从毛孔中往外冒，就算是有空调吹着，也是觉得闷热。
<br/>　　晚上的时候，黄虹和佟母一起来了，还是慕容飞做得晚饭，留下两个人一起吃了饭。晚饭的时候，佟千雪胃口好了一些，吃得也比中午多，除了小半碗的米饭，还喝了一些汤。
<br/>　　黄虹和佟母来就是来看看佟千雪的情况，自从佟千雪怀孕之后，她们每天都会要来两次。
<br/>　　晚饭后，慕容飞又切了水果让众人吃了一点，说明了部队那边的结婚申请已经拿到了，随时都能去民政局把结婚证给办了。
<br/>　　本来按照慕容飞的打算，是要等天气好了以后再去，但是黄虹不同意，执意要让慕容飞明天就去把事情给办妥了，慕容飞和佟千雪拗不过两个大人的意思，只能点头答应。
<br/>　　黄虹和佟母也没有在佟千雪这里逗留很长时间，就起身告辞；离开前，黄虹还是不忘用眼神对慕容飞警告了一番。
<br/>　　慕容飞收拾厨房的时候，佟千雪已经洗了澡靠在床上在看书，看到慕容飞走进卧室，合上书对慕容飞说道：“坏家伙，需不需要我帮你把白天没做完的事情给做完了？”
<br/>　　本来慕容飞已经刻意把这事儿给忘了，可现在被佟千雪再次提起，心里那团火又是冒了出来，盯着只穿了睡裙的佟千雪，沉声说道：“你这样可就过分了啊！”
<br/>　　“我是看你憋得那么难受，好心帮你，不需要就算了。”
<br/>　　佟千雪来，不再去理慕容飞。
<br/>　　突然，佟千雪觉得身边的床垫一沉，扭头就看到了慕容飞的脸，还没来得及反应就被慕容飞抓住一通狼吻。
<br/>　　最后，慕容飞还是如愿以偿的让佟千雪做了以前只能想想，但却从没有让佟千雪付诸于实践的事情。
<br/>　　当慕容飞舒爽的发出一声低吼后，佟千雪捂着嘴跳下了床，跑进了浴室，对着马桶又是一阵呕吐。这次直接是把她的胃给吐空了，慕容飞走进浴室的时候，佟千雪已经吐不出东西了，这看的慕容飞非常后悔，后悔自己不应该让佟千雪做那事儿，恨不得抽自己两下。
<br/>　　“你舒服了，肚子里那个小的就得挨饿了。”
<br/>　　佟千雪白了慕容飞一眼，说道：“等他出来之后，有你好看的！”
<br/>　　慕容飞心疼的抱着佟千雪，忙不迭的向她道歉。佟千雪双手也是紧紧的环绕着慕容飞，贪婪的嗅着他身上的味道，这种味道能给她一种特殊的安全感。
<br/>　　佟千雪松开慕容飞，在他身上拍了一下，说道：“赶紧去洗澡，身上都是汗味，臭死了。”
<br/>　　“我再给你去做点吃的？”慕容飞对佟千雪问道。
<br/>　　“刚吐完，哪儿还有胃口。”佟千雪想了想说道：“我现在特别想吃泡面。”
<br/>　　“不行！”慕容飞想也不想就拒绝了佟千雪的要求。
<br/>　　“那我不吃了。”
<br/>　　佟千雪也跟着耍起小脾气，转身走出了浴室，回到床上不再理慕容飞。慕容飞想了想，最后还是走进了厨房，捣鼓起来。
