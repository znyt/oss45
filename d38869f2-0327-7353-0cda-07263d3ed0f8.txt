　　“恭喜天帝了。”
<br/>　　见其他人走后，接引与准提连忙出声道贺，只要好好的沟通了昊天与瑶池，日后做事也就方便多了，他们也可以光明正大的到东方传教，至于其他圣人对昊天与瑶池不屑一顾，那是他们的事，自己只要做好自己的事就好了。
<br/>　　“多谢两位圣人。”
<br/>　　昊天与瑶池经历过如此之多，自然也看得清形势，不是什么自大之辈，虽然他们被道祖封为天帝，执掌天下，但也不敢得罪任何一位圣人。要知道，这是世界终究是弱肉强食，实力至上，圣人之下，终究为蝼蚁啊。
<br/>　　两位西方圣人一番道谢之后，也是走了，至于玄天，更是在三清走后，便已经走了，天庭之事，与我人族何干。
<br/>　　偌大的紫霄宫，此刻只剩下了昊天与瑶池两人。昊天顿时脸色阴沉，看着消失的几位圣人，心情极度不爽，暗骂一声可恶。他已经想到今后被圣人架空，当做傀儡的命运了。
<br/>　　瑶池见昊天脸色不好看，顿时问道：“怎么了？昊天。”
<br/>　　“没事，我们走吧，先去看看天庭。”昊天回过神来，摆了摆手，说道。
<br/>　　“嗯。我也想去看看今后我们居住的的天界是什么样的。”
<br/>　　随即，二人便出了紫霄宫，向着天界而去。
<br/>　　来到天庭，两人顿时惊呆了，这里没有鸟语花香，没有灵气逼人，也没有仙音袅袅，仙鹤飞舞的仙家福地的模样。一眼望去，到处都是残垣断壁，一片破败的景象，哪还有一点皇家的威严气派啊。
<br/>　　两人见天庭这幅模样，顿时心中升起一股寒意，嘴里不停的嘟囔着：“这就是天庭吗？怎么与想象中的天庭不一样啊？”一副不可置信的样子。
<br/>　　随即，他们便想到了，巫、妖、人三族大战，天庭为妖族重地，被大战波及，毁了也是在所难免的，如今出现这幅破败的景象，也不是不能理解。只是，他们如今要怎么办啊，他们可是如今的天庭之主啊，难道就让他们住在这么破败的地方？到时候天庭的威严何存，天庭的颜面何在啊？
<br/>　　顿时，昊天与瑶池心中哇凉哇凉的，整个人变得不好了。
<br/>　　良久，他们才想起，要成为天帝，还要如同之前的巫、妖二族一般，祭天行礼，祷告苍穹，不然得不到天地的认可。
<br/>　　于是，他们两人当即便摆出祭坛，向着紫霄宫方向叩拜九次，祷告祈天，道：
<br/>　　“承蒙道祖不弃，吾昊天、瑶池，境登临至尊天帝之位，今后必以天下苍生为重，望苍天庇佑，天庭气运永昌。”
<br/>　　轰隆隆！
<br/>　　只见三拜九叩之后，天庭一阵轰响，昊天皇袍加身，尊称玉皇。全称“太上开天执符御历含真体道昊天至尊玉皇大帝”，或者“昊天金阙无上至尊自然妙有弥罗至真玉皇上帝”、“高天上圣大慈仁者玉皇大天尊玄穹高上帝”。
<br/>　　瑶池凤袍加身，瞬间变得高贵起来，人称“九灵太妙龟山金母”，“太灵九光龟台金母”“瑶池金母”。
<br/>　　二人见此，还来不及大喜，便见天庭之上的残垣断壁、倒塌的宫殿庙宇，竟然一个个的慢慢消失于虚空，不一会儿的功夫便全部消失不见，整个天庭变成一片平地，不见任何事物。
<br/>　　在两人疑惑的目光之下，只见天空之中飞来一物，由远及近，慢慢放大，竟是一座规模宏大的宫銮，占地千万里之广，散发出万道金光，有瑞气千条笼罩其中。
<br/>　　不得不说，这才是一个真正的仙境。
<br/>　　放眼看去，但见宫銮入口处，有一支巨大的通天神柱直插云霄，不知有多高，天柱内侧立起了一座巨大的牌楼，上面全部是由琉璃造就的，宝光闪烁，“南天门”三个鎏金大字赫然在目。
<br/>　　透过南天门，再往里面看去，一片片缭绕若烟雾般的仙云，上下翻滚，隐隐有阵阵仙音轻轻飘出，轻灵动人，婉转悦耳。数只灵禽白鹤，自由自在的飞舞，激起一阵阵仙风，仙云在上面如流水般飘来飘去。
<br/>　　再往里面看去，有一座大殿，殿前几根大柱，柱上雕刻着金灵曜日的赤龙，又有几座仙桥，桥上盘旋着彩羽凌空的凤凰。
<br/>　　大殿乃是仙官朝拜玉帝的凌霄宝殿，分为上下两层，上层有九龙宝座、九凤宝座，乃是天帝与帝后之主位，下层有数根紫金宝柱，乃是众仙官站立之所。
<br/>　　凌霄宝殿外又有无数仙宫。如太阳宫、仙乐宫、极乐宫、无极宫、五明宫……又有数重宝殿，乃朝会殿、凌虚殿、宝光殿、天王殿……又有一座座花园、城池、闹事，关卡分布在宫殿四处。
<br/>　　天界再次恢复三十三重天的模样，九为至尊，玉帝居住在九重天。其他地方，或者仙官住所，或者荒凉之地，亦或者是花园、城池等。
<br/>　　众周天星辰也有宫殿升起，有二十八星宿守护西极勾陈商星，三十六天罡、七十二地煞守护北极紫薇帝星，周天星象的星力直接照入天庭之中，随着这日月精华、星辰精华的照入，使得天庭灵气十足，丝毫不弱于洪荒破碎前。
<br/>　　昊天与瑶池见了此景，顿时大喜，感觉只有住进这样的奢华豪丽的宫殿中，才对得起天地至尊这个称呼。
<br/>　　这时，又有数百仙人来到昊天与瑶池面前，跪拜行礼，却是鸿钧见昊天与瑶池没有什么班底，便遣了几个小仙，前来供他差遣。这些小仙的修为不高，从天仙到金仙，各有不同，他们只听天帝号令，。
<br/>　　不过，这些人的职务也大多被定了下来，都是先乱七八糟的小官，什么管姻缘的，管福禄的，管杂役的，都是一些鸡毛蒜皮的小角色。
<br/>　　昊天也明白自己当下是缺少人才，有了他们终究是比没有要强。他们虽然不强，但也了胜于无。至于以后的事，以后再解决，自己都成了天帝了，还怕没有人前来投靠吗？
<br/>　　两人商议，分配好各人的仙籍，天禄等事，从此他们听命于天。
<br/>　　商量好后，他们便准备上表祭天，正式举行昊天的登基仪式。两人也很是有礼，准备发帖邀请洪荒各位大神通者与几位天地圣人前来观礼，见证自己的登基仪式。
<br/>　　经过一番商量，两人将正式登基一事定在百年之后。
