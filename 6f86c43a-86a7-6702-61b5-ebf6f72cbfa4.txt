　　云熙听到詹姆斯对自己的称呼，先是一愣，随后报以一笑，没有多说什么，接过詹姆斯手中的纸条，转身便离开了房间。
<br/>　　对于詹姆斯能知道自己在“炼狱”时的身份，云熙并不奇怪，毕竟美国人的情报系统是这个世界上最全面、最系统的情报系统之一。
<br/>　　云熙回到自己的房间，用她带来的笔记本电脑通过加密的方式把詹姆斯给自己那张纸条上的坐标发送给了罗昊，詹姆斯之所以会突然改变Alpha和Bravo小队的撤离坐标，显然也是因为担心在这个联合行动司令部中有人会走漏消息，导致Alpha和Bravo小队的撤退受阻。
<br/>　　中东，某不知名小镇，CIA情报安全屋。
<br/>　　在这里Alpha和Bravo小队得到了补给和休整，并且等到了前来接应他们撤离的飞机。
<br/>　　Alpha和Bravo小队回到“尼米兹”级核动力航母的时候，已经是第二天的下午，詹姆斯亲自到甲板上迎接了他们。
<br/>　　“勇士们，你们干得漂亮，非常漂亮！”詹姆斯鼓着掌，大声对着从直升机上走下来的Alpha和Bravo小队队员说道：“勇士们，我为你们准备了美食和热水，等待着你们的凯旋。去吧小伙子们，接下来一段时间你们可以尽情的放松一下。”
<br/>　　詹姆斯走到罗昊、李然、李毅峰、慕容飞面前，那双如鹰一般犀利的眼睛从四个人身上扫过，随后带着善意的微笑说道：“干得漂亮小伙子们，你们用你们行动向我证明了，中国军人是好样的。”
<br/>　　詹姆斯在罗昊肩膀上拍了拍，指了指站在甲板上的云熙，说道：“去吧少校，你的妻子很为你的安全担心。”
<br/>　　“将军，他是你的了。”
<br/>　　罗昊把Parker推到詹姆斯面前，然后朝着云熙走去。至于詹姆斯会怎么对待Parker，会以什么样的方式，从他的嘴里挖出想要的东西，这就是詹姆斯的事情了，跟罗昊不再有关系，
<br/>　　罗昊走到云熙面前，张开手一把将云熙抱在怀里。云熙用力推开把罗昊推开，举起拳头在罗昊胸口捶了两下，说道：“身上臭死了，还不快点去洗洗，换身干净的衣服，穿着这身黏糊糊的衣服感觉很舒服吗？”
<br/>　　洗完澡，罗昊藏着湿漉漉的头发刚走进房间，就被云熙以命令的口吻要求坐在床上，然后让他脱掉身上的衣服，拿出红花油在他胸前两个被流弹击中后留下的淤青上轻揉着。
<br/>　　“每次出去，身上都要弄点伤回来。”云熙在手中倒了一些红花油，擦在罗昊胸口的淤青上，说道：“不让我担心，你就浑身不舒坦是吧？”
<br/>　　“这子弹又不长眼睛，要是我的速度能跟得上子弹，保准不会让它打中我。”罗昊回答道。
<br/>　　“还嘴贫！”
<br/>　　云熙用手指在一块淤青上用力戳了一下，疼得罗昊龇牙咧嘴，苦着脸看着她说道：“我现在是伤员，你不能这么对我。”
<br/>　　“嚯，现在知道自己是伤员了。”云熙白了他一眼，说道：“看到你们被围困在军事监狱里的时候，我真想自己也在Bravo小队那架直升机上，然后冲到你的身边去。”
<br/>　　云熙抬起头，神情认真的看着罗昊。两个人的眼神在空气中发生激烈的碰撞，溅出几点无形的火花，罗昊抬起云熙的下巴，一低头亲吻上她的嘴唇，云熙原本拿在手中的红花油玻璃瓶也掉落在了地上，后面的事情就变得顺理成章……
<br/>　　一段不足为外人所道，水乳交融般的交缠后，罗昊低吼着在云熙身体中达到高潮，抱着身下脸色酡红的女人，喘着粗气。良久，两人才分开，用纸巾清理了一下身体后，云熙软软的躺在罗昊怀中，说道：“罗小昊，我们再要一个孩子吧？”
<br/>　　“不是说等到我从作战部队退下来之后，再考虑这件事情吗？”罗昊问道。
<br/>　　“等到你从作战部队退下来的时候，还有三年的时间，那时候你35岁了，我都37了。”云熙又往罗昊身上靠了靠，说道：“而且你每次出去，我都提心吊胆的，生怕你出去以后就回不来了。如果我现在怀孕，等到孩子生下来的时候，就是一年后，到时候距离你从作战部队退役还有只剩下两年的时间，那个时候，小战也去了红旗学校，我就可以安心带孩子了。”
<br/>　　“你早说，刚才就不用那东西了。”罗昊抱着云熙，眼睛看向那个被抱在纸巾中仍在地上的安全套，咬着云熙的耳朵说道：“要不我们再来一次，说不定这次就能有了呢。”
<br/>　　“不来了，不来了。”云熙慌乱的挣脱开罗昊的双手，说道：“我全身都快散架了，我不来了。而且这里还在船上，这么长时间不出去，别人会有想法的。”
<br/>　　“他们又不蠢，肯定知道我们两个这么长时间待在房间里干什么。”
<br/>　　罗昊再次抱住云熙，云熙不愿意，罗昊也没有勉强她。可能是因为担心了一晚上没有休息，再加上被自己折腾了一番确实累了，很快云熙就睡着了过去，传来轻微的呼吸声。在云熙的额角浅吻了一下，罗昊也跟着睡着了过去。
<br/>　　等到两个人醒来，已经是黄昏了。穿好衣服，走出房间，找了点吃的填饱肚子后，两个人手牵着手，走在航母甲板上。
<br/>　　一架架F/A-18E“超级大黄蜂”安静的停在甲板上，让得罗昊不禁感叹什么时候能在自己国家的航母上看到眼前这样的画面。强大的制空权和制海权，才是一个超级军事大国的标志。
<br/>　　这时，一架SH-53E“超种马”运输直升机螺旋桨开始旋转起来，准备起飞。
<br/>　　麦克米兰身上挎着HK416A5战术步枪，带着其他几个海豹队员，还有几名海军陆战队员朝着准备起飞的“超种马”运输直升机跑去。
<br/>　　麦克米兰和那几名海豹队员从罗昊和云熙面前经过时，皆是对两个人竖起大拇指。
<br/>　　“Bravo1，你们有行动吗？”罗昊叫住麦克米兰，对他问道。
<br/>　　“是的。”麦克米兰回答道：“‘八爪鱼’那家伙招了，我们现在要去他供出的几个地方看看。”
