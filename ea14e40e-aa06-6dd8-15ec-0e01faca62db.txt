<br/>　　“之前是我有眼无珠，冒犯了您，恳求您饶了我这一次，恢复我的人身！以后我唯您的命令是从！！”

<br/>　　叶尘略微沉吟了片刻，方才缓缓道：
<br/>　　“想要我饶了你也行，不过你必须要帮我办一件事情！”
<br/>　　叶家家主顿时大喜，
<br/>　　“是是是！无论什么事情，只要您吩咐，我一定照办！”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“先别答应这么快，这件事可没有那么容易！”
<br/>　　说着，叶尘随手一招，便将化身成小猫咪的叶家家主擒在了手中，然后身体一晃，朝之前小兰逃跑的的方向追去。
<br/>　　片刻之后，叶尘找到了婢女小兰，这小丫头并没有走远，蹲在一条隐蔽的小溪边，正哭的稀里哗啦，
<br/>　　“少爷，对不起！不过你放心，你要死了，小兰也不会独活……”
<br/>　　叶尘听到这话，顿时心头一暖，
<br/>　　虽然他并不是真的叶枫，可是经过三个月的相处，和这小婢女也建立不浅的感情。
<br/>　　“傻丫头！你家少爷，怎么可能会这么容易就死了呢？”
<br/>　　叶尘微笑着说着，同时缓缓走了过去。
<br/>　　小兰听到叶尘的声音，娇躯微微一颤，继而猛然转过身，
<br/>　　“少爷！！”
<br/>　　兴奋不已的小兰，当即犹如小鸟入林般，扑进了叶尘的怀中。
<br/>　　“少爷，你真的没死？真是太好了！！”
<br/>　　叶尘不由得伸手揉了揉小兰的脑袋，然后在她的脑门上轻轻一弹，
<br/>　　“少爷我不但没有死，还给你带来了一个礼物！”
<br/>　　说着，叶尘将叶家家主变成的那只折耳猫拎了起来，递到小兰跟前，道：
<br/>　　“努，这就是少爷送你的礼物，喜欢吗？”
<br/>　　“哇，好可爱的小猫啊！”
<br/>　　小兰顿时两眼放光，像这种年纪的小女孩，显然都很喜欢可爱型的宠物。
<br/>　　不过叶家家主听到这话，顿时一下子就受不了了，不由得对着叶尘怒目而视，咆哮了起来，
<br/>　　“你这是什么意思？居然将本家主，作为一个宠物，送给一个低贱的婢女？你还不如杀了我算了！！”
<br/>　　小兰原本还想伸手去摸这小猫的脑袋，听到这道咆哮之声，顿时吓得小脸发白，不由得连连倒退，
<br/>　　“少爷，它、它会说话？”
<br/>　　“哼！”
<br/>　　叶尘轻哼一声，冷冷道：
<br/>　　“你觉得你有选择的权力吗？从现在开始，小兰就是你的主人，我要你给她当一百年的灵宠，保护她的安全！”
<br/>　　参悟到了造化法则之后，叶尘已经感觉到，自己随时都会送离这个世界，所以在离开之前，叶尘必须要为小兰这小丫头，找一个保镖，来保证她的安全。
<br/>　　而眼前被变成猫的叶家家主，显然是一个不错的选择。
<br/>　　叶尘说到这里，微微一顿，冷冷瞥了一眼地上的叶家家主，缓缓道：
<br/>　　“当然，你也可以拒绝，不过我保证，会让你在无比痛苦中死去！！”
<br/>　　叶家家主感受到叶尘，那凌冽的目光，同时又想到叶尘之前的种种逆天手段，整个猫身不由得狠狠一颤，眼眸中透出一抹恐惧，最终缓缓低下了头颅，
<br/>　　“好吧！我答应你的条件，不过你也必须答应，一百年之后，恢复我的人身！”
<br/>　　叶尘早就料到，叶家家主肯定会答应，不由得嘴角微微一扬，
<br/>　　“成交！”
<br/>　　嗖！
<br/>　　叶尘屈指一弹，一道白光打入叶家家主的猫体之中，等到百年之后，他的人身就会自动恢复。
<br/>　　叶家家主低着头颅，猫眼之中却露出一抹凌厉之色，心中暗想，
<br/>　　‘我且先答应你，区区一个婢女，也想命令本家主，简直痴心妄想！’
<br/>　　叶家家主心里正想着呢，叶尘忽然再次抬手一挥，
<br/>　　呼！
<br/>　　一道道神秘的符文，在白光的包裹之下，钻入到小兰的脑海之中，叶家家主顿时隐隐感觉到，自己的命运，似乎已经被一股无形的规则之力，和眼前这个小婢女，紧密的联系在了一起，而且自己明显是处于一个附属的地位，甚至小兰只需要一道念头，就可以让他身死道消！
<br/>　　‘这……这真的是……神灵才能具备的手段啊！’
<br/>　　叶家家主不由得暗暗感慨了一声，心中又惊又恐，彻底没有了任何反抗之念，乖乖匍匐在了地上，变得温顺无比。

<br/>　　于是，小兰当即蹲下身，将小猫咪从地上抱起，拍了拍它的脑袋，笑道：
<br/>　　“以后我就是你的主人了，你可要乖乖的哟~”
<br/>　　叶家家主顿时郁闷的想死，可是却又根本无可奈何。
<br/>　　轰隆隆！

<br/>　　叶尘的头顶上空，忽然出现了一道巨大的漩涡！

<br/>　　“少爷，你这是……”
<br/>　　叶尘再次揉了揉小兰的脑袋，笑道：
<br/>　　“少爷有一些事情，不得不离开一段时间，等到少爷走了之后，你要好好修炼，如果有机会的话，我一定会回来看你，知道吗？”

<br/>　　叶尘又一番好说歹说，才终于将这小丫头安抚下来。
<br/>　　而此时，
<br/>　　轰隆隆！
<br/>　　那道漩涡突然落在了叶尘身上，将他整个人吞噬其中，彻底消失不见……
