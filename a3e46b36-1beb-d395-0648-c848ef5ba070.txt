　　“耶，刘君果然是最好了，那么，我可就不客气了哦。”脸上带着灿烂的笑容，什造的心情似乎很不错的样子。
<br/>　　直到这时，他才想起什造和金木的体质近似，都会有一种吸引喰种的特殊气味。
<br/>　　一辆载满乘客的电车缓缓停靠在站牌前，上面写着一些日文，大体是多少路的意思，什造似乎看到了什么，表情一愣，伸手拉了拉刘哔的衣角，一起挤上了车。
<br/>　　说起来，刘哔现在身上没有日元，所幸什造替他投了硬币。
<br/>　　车上有些拥挤，挤满了形形色色的乘客，人和人都挤在一起，如同沙丁鱼罐头一般，连转身的余地都没有。
<br/>　　身处这种环境下，看着四周站在他身边的乘客，刘哔眼睛微微有些泛红，他有些忍受不了空气中挥发出来的肉香味。
<br/>　　那些裸露在外面的脖子，让他莫名想到了鸭脖，如果撒点辣椒粉烤一烤，一定十分好吃。
<br/>　　怕被什造发现什么，刘哔停止了呼吸，随后将头低了下来，他眼中的淡红色，很容易引起误会。
<br/>　　电车缓缓启动，不知道要去往那里，他对这个城市的路线很陌生，不过去哪里对他来说都没多大区别。
<br/>　　反正，能抓喰种就好了吧，什造拉着他上来，估计也是发现了一些什么吧。
<br/>　　一路有些颠簸，电车前进的方向越来越偏僻，车上的乘客也逐渐开始稀少了下来。

<br/>　　被气流搞得耳朵有些瘙痒，刘哔不自在的动了动身子，点头示意自己收到。
<br/>　　该死，这家伙的身体，怎么会有一股清香味，好像喷了一点香水，反正让他有种打喷嚏的冲动。
<br/>　　要不是什造突然趴到自己后背上，他还闻不到这股味道。
<br/>　　真是让人讨厌的气味。
<br/>　　刘哔强忍着鼻间的瘙痒，默默向旁边挪了一点，不经意一般的拉开了和什造的距离。
<br/>　　刚才他怕什造发现什么异样，暂时恢复了呼吸功能，结果却被刺激到了。
<br/>　　又不是娘们，喷什么香水啊。
<br/>　　心中吐槽了一句，刘哔手中握着吊环，面无表情的看着窗外，此时电车的路线已经到了郊区。
<br/>　　大约半个小时之后，电车的速度开始减慢了下来，刘哔观察了一下四周的环境，看到了一个有些破败的公交站点。
<br/>　　“终点站到了，你们两位，还不下车吗？”司机停好车，从驾驶位上站起，肥胖的身体令人有些恶心。
<br/>　　在他敞开一颗纽扣的胸前，挂着一个蓝色的牌子，上面有他的照片，还有一行日文，大概是他的名字。
<br/>　　宫本赤翔，好像是这样叫吧，这名字取得挺有味道。
<br/>　　“大叔，我们好像坐过站了，我们再投个币，你们能不能让我再坐回去啊？”什造坐在前排的塑料硬座上，揉了揉眼睛，似乎是一副刚睡醒的样子。
<br/>　　司机脸色缓和了下来，但出于面相原因，就算是笑，也会让人莫名的感觉有些狰狞。
<br/>　　“这样啊，那你们下车等一会吧，我打扫一下卫生，如果没有人上车，我就发车带你们回去。”和他可怖面相不同的是，司机的声音出人意料的温柔。
<br/>　　什造没有拒绝，拉着刘哔下了车，站在公交车站牌前等候着。
<br/>　　“什么啊，这家伙，一点都不像是喰种，该不会是认错了吧。”看着正在车内打扫卫生的中年大叔，什造嘀咕了一句，有些懊恼的踢着路边石子。
<br/>　　想起先前司机用手指捏灭烟头的动作，什造心中一阵嘀咕，她有些不确定自己心中的想法。
<br/>　　很多喰种都会有这种习惯，目的是为了磨掉手指上的指纹，这样一来警方也很难追查到它们。
<br/>　　这是筱原特等告诉她的，对于那位大人的话，她深信不疑。
<br/>　　“额，车内的油不多了呢，我得想个办法，加一下油，你们能帮我一个忙吗？”中年司机从车上下来，费劲的挪动了一下身体。
<br/>　　什造回过神来，脸上露出一个可爱的笑容：“好啊，怎么帮大叔你呢？”
<br/>　　向两人站立的方向靠近了一点，中年司机抹了一把额头上留下来的汗珠，从车上观察了这么久，他觉得自己应该能吃下这两个猎物。
<br/>　　真是令人兴奋啊，这俩人虽然瘦点，但也足够吃好长时间。
<br/>　　“哦，其实就是个小忙啦，附近没有加油站，我想，用两位的血液来代替汽油，应该是可以的吧。”脸上露出一个诡异的笑容，中年司机张开了嘴巴，留下了一串令人恶心的津液。
<br/>　　拍了拍额头，什造松了口气，果然，筱原大叔告诉她的没有错吧，这个家伙，果真是一个喰种呢。
<br/>　　“诶，是喰种呢，那这样的话，我应该可以杀了你吧。”什造从腰间掏出两柄黑色匕首，在手中慢慢转动着，语气有些兴奋：“那我可动手了哦，喰种的话，老师说过是可以杀的呢。”
<br/>　　看到对面两人反常的表现，中年司机心中突然有些不好的预感，但嗅到空气中血肉的气味，他已经按耐不住腹中的饥饿感了。

<br/>　　有些像是触手，但顶端却是无比锋利，赫子顺着手臂蔓延，将中年司机整个人衬托的如同怪物一般。
<br/>　　“吃了我……你居然想吃了我。”什造的状态突然亢奋起来，苍白的脸上，露出一抹病态的潮红，口中用只有两人能听到的声音低喃着：“真是不可饶恕呢，我要杀了你，我一定要杀了你，这样，我就可以有自己的库克因了。”
<br/>　　饶有兴致的看着眼前的场景，刘哔没有动手，他能感觉到，那个喰种的实力并不算强，最多也就和他先前碰到的行僵不相上下。
<br/>　　不过，什造现在的状态，让他有些摸不到头脑。
<br/>　　一说道杀喰种，那满脸兴奋的样子，倒更像是喰种的吧。
