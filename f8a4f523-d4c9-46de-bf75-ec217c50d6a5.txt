　　一步，两步，三步……
<br/>　　就这样，元始在疯狂的修炼之中，不断地往上攀爬，越是往上，九天罡风层中蕴含的毁灭之力也就越强，但同时，对元始的真身淬炼也就越好。
<br/>　　只是，随着元始的不断上升，每次他的身体都会被轰击受损，都需要相当长的时间来修复，等修复好了，才能继续向上。
<br/>　　就这样，修复，受损，修复，再受损，好似永无休止。
<br/>　　但同样的，元始的肉身强度再次增加，他的无上玉身，已经有开始的数十万丈缩小到如今的九万丈。经过罡风的淬炼，玉身法相虽然缩小了，但也更凝练了，无论强度，还是威力，也都更强了。
<br/>　　随着元始再一步踏出，他的周围变了，变得不再是无尽罡风，而是变成了无尽烈焰，好似一个无尽火狱！
<br/>　　无尽火狱！
<br/>　　目之所及，到处都是烈火。
<br/>　　太阳真火、太阴真火，三昧真火，紫薇天火，九天玄火，风雷怒火、焚心烈火……
<br/>　　光是十大真火，元始看到的就有六七种，还有其他的无尽火种，以及他不知道的更为可怕的闪烁着五彩、七彩、九彩之色的火焰。此处，就是火焰的天堂，也是火焰的地狱。
<br/>　　元始刚踏入此处，就有无尽的火焰向他袭来，也幸好在之前的罡风层中，元始的真身有过突破，不然还真让这些火给烧坏了不成。
<br/>　　“罡火层吗？九天罡风层之后是罡火层。火吗？我修炼《玄黄不灭》第二层五行开道宫不就是要五行之力吗，此处有如此之多的火之力，正好为我所用。”
<br/>　　元始看着这无尽火狱，顿时想到了他修炼功法需要的五行之力，此处正好可以解决。
<br/>　　想到就做，元始运转《玄黄不灭》第二重，顿时，无尽的火焰向着元始涌来，什么太阳真火，太阴真火，九天玄火，三昧真火……他都来之不拒。
<br/>　　所有火焰之力，都被元始引入心之神藏之中，随着火焰之力的不断灌入，心之神藏愈发的火红，状如未开之莲花，生出一道道精气，然而，却始终未蜕变成神邸。
<br/>　　天仙之后，元始胸中就已五气圆满，但现在随着火之力的不断注入，元始感觉人体之中的天日冉冉升起，再缓缓降落，往复循环，先天之精炽烈燃烧。
<br/>　　后天之精气滋养，不断涌来，融入此神藏，变得通红晶莹，像是完全成为天日。
<br/>　　元始体内轰隆隆作响，他看到大海无量，看到日升日落，看到生气，看到蓬勃，无限生机！
<br/>　　同时，心之神藏之中，一座火红色的道门若隐若现，而且，随着无尽火焰之力的注入，那道门变得越来越清晰，直至凝聚到实质！
<br/>　　“轰！”
<br/>　　如日中天！
<br/>　　心之神藏升腾，道门大开，无尽火力涌动，定在人体世界虚空之中，像是当空骄阳，达到了鼎盛，无尽气息弥漫而出。
<br/>　　元始身体之外，火焰更加沸腾了，无尽火焰朝他涌来，整个人像是一个巨大的火人。同时，无尽火焰化作旋涡，融入元始身体之中。
<br/>　　心道之门大开，神藏升腾，化为一个身穿火红帝袍，身绕无尽火焰的先天神邸，这神邸与元始有七八分面似。
<br/>　　在神邸脚下，无尽火焰沸腾翻滚，他就像是主宰这无尽火狱的帝君，在接受万火的朝拜。
<br/>　　心之神藏道宫，达到顶峰，步入圆满！
<br/>　　在他体内，塞满迷雾，他步入下一个神藏道宫，像是登天路，成功跨过之后，等若仙凡之别。
<br/>　　天日当空，始终悬浮在他头顶之上，为他照亮了前路！
<br/>　　很快，元始就感受到了无比锐利的气息，他寻到了肺之神藏。在五形之中，当属金。
<br/>　　但，此地并没有金之力，他亦无法晋升。故而，元始放弃了探索。
<br/>　　他，睁开双眼，看到眼前无尽的火狱，但，现在，这些火对他已经没有任何威胁了，他就像是火焰帝君，所过之处，万火臣服。
<br/>　　元始一步一步，再次上路，继续攀向不周顶峰。
<br/>　　火之力齐聚一层，除了狂暴炙热的火焰外，能在其中留下的东西，自然不会是什么不好的东西，其中一些神材，足够元始炼一些宝物了。其中得到最好的宝物，是无尽火灵气经过无数年汇聚而成的火灵珠，其中蕴含无尽火之力，是个不可多得的宝物。
<br/>　　不久，元始再次停下，因为他再次到了下一层。
<br/>　　在这一层天之中，元始感受到的是无尽的庚金之气，那仿佛要将天地一切化为灰烬的庚金之气。
<br/>　　天空充满无尽金戈之声，无数的庚金之气，形成风暴，像是要撕碎一切。天空充斥满无尽锐利，像是要斩断天地。
<br/>　　“罡金层？果然不出我所料，罡火层之后是罡金层。不过，这也正是我所需的。”
<br/>　　语罢，元始便不再多说，而是找了一个庚金之气密集之地，盘腿坐下，继续运转《玄黄不灭》第二层功法，来吸收这无尽的庚金之气。
<br/>　　随着无数庚金之气的涌入，元始的身体被这锐利之气斩为碎片，但他还是坚持住了。
<br/>　　终于，他来到了肺之神藏。
<br/>　　肺之神藏，接连灵气之神道，覆盖脏腑，位置最高，号称华盖，称为五脏之长。天地精气至清，吞纳天地神源，以其为主，实在是五大神藏重中之重。
<br/>　　他感受到了蓬勃的生机。
<br/>　　此神藏，输送精气神，布散到全身，外达于皮毛，以温养、滋养五脏六腑、四肢百骸、肌膝皮毛，作用之大，超乎想象。
<br/>　　元始盘做不动，不断炼化此层天地之中的庚金之气，源源不断的庚金之气全部纳入体内，汇入华盖神藏之中。
<br/>　　诸气者，皆属于肺！
<br/>　　一呼一吸，与天地相通！
<br/>　　随着众多庚金之气的灌入，这一次，在肺之神藏之中，出现的是一道白色的道门。
<br/>　　随着白色道门凝聚为实质，无尽庚金之气汇聚为一柄白金色巨剑，透露着锐利锋芒之气。
<br/>　　白金色巨剑一剑斩在白色道门之上，道门就此大开，无尽庚金之气涌入道门，凝聚成一尊身穿白色帝袍的先天神邸，他面色一样与元始有七八分相像，手持白金色巨剑，像是一尊金皇。
<br/>　　肺之神藏道宫，达到巅峰，步入圆满！
<br/>　　当元始成功进入这个境界后，他感觉到浑身毛孔都舒张开来，像是与天地相连在一起，调动天地精气的速度快了不知多少倍。
<br/>　　之后，元始再次踏步向上，他依次经历了罡木层、罡土层、罡水层，直至将自己的五大神藏全部修炼圆满。
<br/>　　同时，在他体内的五大道宫之中，凝聚了五尊先天神邸，他们分别是：
<br/>　　掌控木之力的青帝木皇；
<br/>　　掌控火之力的赤帝火皇；
<br/>　　掌控土之力的黄帝土皇；
<br/>　　掌控金之力的白帝金皇；
<br/>　　掌控水之力的黑帝水皇。
