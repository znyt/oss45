        童筱颖第一次发现，自己真的很没用，完全就是昼景的拖油瓶。
<br/>
<br/>        就在她觉得自己无路可走的时候，脑海里忽然想起了一个人，于是立马掏出手机，给那个人打了通电话。
<br/>
<br/>        再见面的时候，童筱颖原本以为哥哥诺司峻的女朋友唐倩兮会对自己生疏了，没想到，她忽然一如既往地对她非常热情。
<br/>
<br/>        “筱颍，你终于决定要来我们会所驻唱了！我们老板费尔南可是一直在期待着你回归！”唐倩兮高兴不已地拉着童筱颖的手，激动不已地说。
<br/>
<br/>        自从上次筱颖在他们会所驻唱，客人反映她的歌声非常好听后，特别是大部分男客人，都希望能再次听她唱歌。
<br/>
<br/>        只可惜，她不唱了。
<br/>
<br/>        老板费尔南还多次跟她提起，让她这位小姑子，继续在他们会所驻唱。
<br/>
<br/>        本来，她有跟筱颖谈好，但是梁家后面又发生了那些事情，让筱颖驻唱的事情不了了之。
<br/>
<br/>        唐倩兮也是个识趣的女人，既然筱颖不愿意，她就不去打扰筱颖了。
<br/>
<br/>        只是没想到，这次是筱颖主动找她，想去他们会所驻唱，她当然求之不得，高兴还来不及。
<br/>
<br/>        童筱颖虽然仍旧一脸不情愿，但是事已至此，迫不得已地对唐倩兮陪着笑脸，道：“我有前提条件！”
<br/>
<br/>        “好说好说，只要你愿意去唱歌！什么事都好说！”唐倩兮二话不说就全部答应了。
<br/>
<br/>        “我工作的时间只到十一点半，另外，我只唱歌，不陪酒，不卖身！”童筱颖一脸认真地接着说道。
<br/>
<br/>        唐倩兮依旧犹如小鸡啄米般猛点头：“这事儿，我替我老板全都答应了！”
<br/>
<br/>        这一回，童筱颖主动请缨，唐倩兮将她带到老板费尔南面前时，费尔南便拿出了早已准备的合同，让童筱颖签。
<br/>
<br/>        这么好的声音，他可不想像上次那样，就让她唱了一次就跑了。
<br/>
<br/>        童筱颖认真地看完了合同后，不禁微微皱起了柳眉，回答道：“五年的时间太长了！”
<br/>
<br/>        “那就两年吧！”费尔南将指缝间夹着的雪茄，在烟灰缸上弹了弹。
<br/>
<br/>        童筱颖看着坐在老板椅上的费尔南，抿了抿唇，提出了自己的意见：“一年，可以吗？”
<br/>
<br/>        “我这里的驻唱……”
<br/>
<br/>        “老板，你就答应吧！我家这小姑子，还是个在校生。”唐倩兮连忙帮着童筱颖说话。
<br/>
<br/>        费尔南深知童筱颖的利用价值很大，于是答应了，便让助理去修改了合同，跟童筱颖签了约。
<br/>
<br/>        从今以后的一年内，童筱颖每天晚上八点到十一点半之间的时间里，必须唱五首歌，每首歌一百元，一晚五百元。
<br/>
<br/>        不陪酒、不卖身，除了唱歌，不为任何客人做任何服务。
<br/>
<br/>        如果不是唐倩兮罩着她童筱颖，这样的合同，费尔南才不会签。
<br/>
<br/>        童筱颖本身就长得漂亮，不陪酒，还真有点儿可惜。
<br/>
<br/>        当天，童筱颖在会所里便以“小谢”的艺名，再次出道。
<br/>
<br/>        她驻唱时所穿的衣服，全部由唐倩兮提供给她，而为了避免“红颜祸水”，她必须戴上假面眼镜登台驻唱。
<br/>
<br/>        童筱颖第一晚的驻唱，刚开口没唱几句，就引来了台下一波小骚动。
<br/>
<br/>        口哨声、欢呼声、鼓掌声……
<br/>
<br/>        大家似乎都很喜欢她的声音。
<br/>
<br/>        费尔南坐在监控室里，看着童筱颖在台上的表现，以及听着她婉转动人的歌声，不禁跟候在一旁的唐倩兮感慨道：“你家小姑子真有当歌星的潜质！”
<br/>
<br/>        “怎么？你有认识的熟人吗？要不，捧我家这小姑子当歌手？”唐倩兮逮着机会，就在费尔南耳边煽风点火。
<br/>
<br/>        其实，她这也是为了筱颖好。
<br/>
<br/>        筱颖留在他们这大城市里，总比会西镇那穷乡僻野的地方去要好。
<br/>
<br/>        “我有意有什么用？你家这小姑子似乎对当歌星不感兴趣。”费尔南双手一摊，无奈地耸了耸肩。
<br/>
<br/>        也对，筱颖确实对当歌星没兴趣，如果有兴趣的话，她早就在第一次来这里驻唱的时候，就会跟她这准嫂嫂谈及这方面的想法了。
<br/>
<br/>        但是筱颖没有，筱颖看起来只想当一名普通的室内设计师，然后嫁一个普通的老公，从此相夫教子，当个贤妻良母。
<br/>
<br/>        唐倩兮从未去真正了解过，筱颖为什么会对自己只有如此简单的希冀。
<br/>
<br/>        这一切的一切，都源于，筱颖最初衷，对“家”的渴望。
<br/>
<br/>        她只不过是个平凡的女子罢了。
<br/>
<br/>        童筱颖一边唱着有关于“家”的歌，一边回忆着自己的历历往事。
<br/>
<br/>        她不求大富大贵，不求功名利禄，只求有一个温暖的家。
<br/>
<br/>        而她现在所做的事情，就是在守护着自己的这个家。
<br/>
<br/>        童筱颖的歌声里，唱出了对“家”渴望的真实感情，明明会所里的气氛是聒噪的，但因为她那忧伤的歌声，让大家也跟着她的歌声沉浸在一片忧郁之中。
<br/>
<br/>        她的工作任务完成后，在后台，她卸了妆，接着又让唐倩兮带她去了财会部把工资给结了。
<br/>
<br/>        童筱颖拿着五百块的工资，心里却怎么也高兴不起来。
<br/>
<br/>        因为，昼景曾经跟她说过，他不希望她再去这种地方来唱歌。更何况，她每天所赚的这几百块钱，压根就不够昼景偿还债务吧！
<br/>
<br/>        不过，她和昼景的生活费还是够了。
<br/>
<br/>        童筱颖凌晨十二点差五分的样子回到家时，傅昼景还未回来，于是她先去洗了个澡。
<br/>
<br/>        另一边，傅昼景加班到多晚，季思妍就陪他到多晚，连赵斌都回家了，她都还不回家，一直坐在自己的办公桌前，帮傅昼景一起赶设计稿。
<br/>
<br/>        季思妍看着傅昼景埋头工作的样子，恍神间，感觉自己回到了高中时代，坐在自己的座位上，看着那边的傅昼景埋头备考做试卷的样子。
<br/>
<br/>        这个男人是她的初恋，与此同时，她也是他的初恋。
<br/>
<br/>        可结果，他们两之间，却彼此错过了彼此五年。
<br/>
<br/>        季思妍画着画，不经意间地一手托腮，微微仰头，双眸深邃，仔细地端详着傅昼景，脑海里思忖自己和他的事情。
<br/>
<br/>        办公室里灯火通明，但此刻就她和他两个人。
