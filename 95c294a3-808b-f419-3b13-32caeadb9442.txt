　　“灭神！”
<br/>　　忽然，一道极为冷酷的声音在嫦羲耳边响起。
<br/>　　不好！嫦羲心中暗叫一声，想要躲避，但奈何她的身体已经跟不上她的思维了。
<br/>　　又是一记耀光闪烁，一个噬神虫妖神轰出致命一击，嫦羲一个不慎，便让神光穿过她身体，直接轰击到她的元神之中，那神光带有极强的吞噬性，想要毁灭她的元神。
<br/>　　噬神虫是一种专门吞噬修士元神灵魂的虫族，凶名赫赫，但奈何这种东西孕育出来的条件极为苛刻，所以极为稀少，能够成长起来的那就更少了。但不可否认这一族对元神的攻击手段，的确是有独到之处，但它也有一个致命的缺点，便是不能让人察觉，若是被人提前察觉了，那他的攻击手段就会被大幅度削弱。
<br/>　　元神攻击，最为致命，灵魂是一个人的根本，若是灵魂有损，谁知道会发生些什么。幸运点，元神受损，修为到退，还有可能变为白痴，不幸者，有可能立即死亡。
<br/>　　“嗖——嗖——嗖——”
<br/>　　关键时刻，三道箭光闪过，就在嫦羲快要被灭杀之际，寒芒一现，快到也不可思议，电光火石之间，便见三个妖神倒地不起。顺眼看去，只见在他们的喉咙之中，都插着一根乌黑的长箭。
<br/>　　“什么——”
<br/>　　剩下两人大惊，刚要喊出来，但话还没有说完，他们眼中便再次闪现两道乌光，他们心道不好，想要闪避。可惜，刚刚心有所感，还没来得及有所行动，他们便被另外两道乌光射穿。
<br/>　　临死之前，他们只有一个念头，那就是怎么会有这么快？
<br/>　　最后，他们终于看清来人的影像了。那是一个长发披散，手持长弓，身披兽皮的年轻小伙，这还不重要，最重要的是，他们能够清晰感受到，那人是一个大巫。
<br/>　　巫族，那可是妖族的死敌。
<br/>　　“原来，我们死得不冤啊。”这是两妖最后的念头。
<br/>　　大巫并没有理会倒下的妖族，而是快步来到女子身前，将她扶起，同时，他也发现了这女子的容貌，简直是惊为天人，那一刻，他心神险些失守了，实在是这女子太漂亮了，不似人间拥有。
<br/>　　但随即，他便将这些思绪杂念抛之脑后，因为他发现眼前这位女神已经奄奄一息了，他有些担忧的急声叫道：
<br/>　　“姑娘，你没事吧？”
<br/>　　“我——”
<br/>　　嫦羲还没有将话说完，便昏死了过去。
<br/>　　“姑娘！姑娘……”
<br/>　　耳边再次传来那大巫的呼叫声，可惜，她早已听不见了。
<br/>　　……
<br/>　　“这……这是哪里？”
<br/>　　不知过了多久，嫦羲缓缓睁开眼睛，她只觉得自己头部十分疼痛，脑海之中，有一种被撕裂的感觉。她的脸色苍白无比，眉头紧皱起来，若是让人看到她如今这幅模样，肯定会让人心疼无比，有一种怜惜的冲动。
<br/>　　她带着茫然的的目光，打量着这间古朴的石堆房子。房间中十分简单，除了一张木床外，就只有一些弓箭，几张兽皮。
<br/>　　“你醒了！”
<br/>　　这时，一个英俊的青年走了进来，看见她醒过来，顿时脸上露出喜意，急忙上前问道。
<br/>　　“你是谁？这里是哪里？”嫦羲目光涣散，轻声问道。
<br/>　　“我叫后羿，姑娘，你是谁，为什么会被妖族追杀？”后羿回答后接着问道。
<br/>　　“我是……”嫦羲刚要回答，却发现自己什么都不记得了。
<br/>　　“我是谁？我从哪里来？我到底是谁？啊……我的头好痛！”女子说着说着，突然间便抱头大声叫了起来。
<br/>　　“姑娘！姑娘，你冷静点！”
<br/>　　后羿见此，想到之前在她大战之中可能被伤到神魂，记不起自己的身份也是有可能的。他连忙上前将她扶住，大声道。
<br/>　　“既然你想不起自己是谁了，那便不要再多想了，也许你以后能够想起来！”
<br/>　　在后羿的安慰下，女子渐渐恢复了平静，只是她的目光仍然十分茫然，她记不起自己的身份。
<br/>　　“既然你不知道自己的名字，那便重新取一个，如何？”后羿对着女子说道。
<br/>　　“好啊！”
<br/>　　“我看你娇娥动人，美丽非凡，就仿佛九天之上的仙女。娥，意指美丽的女子，不如就叫嫦娥吧！”后羿说道。
<br/>　　“嫦羲，嫦娥，嫦……为什么这个名字我有些熟悉呢？”女子眉头轻皱，低声呓语。
<br/>　　“嫦娥，你没事吧？”后羿有些担忧地看着眼前这个女子。
<br/>　　“我很好！”嫦娥突然将目光移到后羿身上，她觉得，其他一切都不重要了。
<br/>　　“嫦娥，你好有伤在身，好好躺着！我去给你煎药。”后羿有些担忧的对着嫦娥说道。
<br/>　　“好！”
<br/>　　……
<br/>　　天庭之上，太阳神宫中，只有帝俊、太一两兄弟在。帝俊深深地叹了一口气。东皇太一又怎会不知道自家兄弟心中所想。
<br/>　　“兄长勿忧，天庭只要不得罪几位圣人，想来圣人也不会无聊到找你我兄弟的麻烦。”
<br/>　　帝俊叹了一口气，缓缓道：“哎，圣人高高子在上，我们刚想对人族下手，却又发生太清圣人在人族中立下人教一事，这就相当于日后人族后面站着一位圣人，这该如何是好。唉……若是当初你我兄弟能在紫霄宫中争得一席圣位，我天庭也出一位圣人，如今又何来如此烦恼？”
<br/>　　东皇天一闻言，也不由得一阵沉默，虽然帝俊所言有些消极，但却也属实。他当即沉声道：
<br/>　　“既然如此，大哥，那我们便索性夺了那红云的圣位，他并不是道祖的弟子，我们还有机会。”
<br/>　　帝俊眼神闪烁一下，沉默片刻，终究是帝王，心性决绝，很快就下定决心道：
<br/>　　“五庄观终究是有镇元子镇守其中，不是那么容易得手的。不过，你可派人前去紧紧盯着五庄观，若是红云出了那五庄观，我们便前去阻截！”
<br/>　　“好，大哥！不过，我们还得防着其他人，洪荒之中不可能只有我们兄弟盯住那圣位！”
<br/>　　“也是，这件事就交给你去办了。”帝俊沉声道。
<br/>　　“好！大哥放心！”
<br/>　　幽冥血海深处，一头血红长发飘散的冥河道人正坐在自己的宫殿之中，他此刻一脸呈深思之状。
<br/>　　“女娲成圣时创造出一个全新的种族，西方二为圣人是立教成圣，甚至，就连三清成圣后都立了教派，可见这立教也是圣路之中极为关键的一环。”
<br/>　　“若是我也能够创造出一个种族，并借此立教，我能否成圣？”冥河越说越在理，越想越激动，他觉得此法可行。
<br/>　　“可是，又该怎样创造一个新的种族呢？”冥河眉头紧皱，再次陷入沉思中。
<br/>　　突然，他看见远处血海之中不断挣扎着的冤魂，眼前不由得为之一亮。
