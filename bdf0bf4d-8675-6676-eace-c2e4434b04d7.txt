　　待燕赤霞离去，顿时阴风刮起。
<br/>　　只见一群穿着花花绿绿长裙，莺歌燕舞的女子飘然落在兰若寺前，望着寺内，娇笑连连的探讨起来。
<br/>　　“姐妹们，那可恶的牛鼻子道士离开了，终于有机会下手了。”
<br/>　　“今日，僧舍中有两位青年，其中一位是世家公子，长得好生俊俏，另一位是书生，你们谁去试试？”
<br/>　　“又来一位书生，百无一用是书生。”
<br/>　　女子们捂嘴轻笑，美眸中却泛着寒芒，在她们眼中，无论是世家公子哥，或者是书生，都不是好东西。
<br/>　　当然，敢来这里的世家公子哥，她们也是第一次见，佩服他胆子不小的同时，心中也非常好奇。
<br/>　　至于书生，她们见的可就多了。
<br/>　　数十年间，来兰若寺的书生不可谓不多，基本都是人面兽心，最后无一不是死在她们手中。
<br/>　　这时，为首女子眼中闪过一抹嫉妒，回头看向最后面，容貌绝美的白裙女子。
<br/>　　“小倩，不如你去试试，你在姥姥手下，出手的次数最少，倒是磨练你的好机会。”
<br/>　　“琴姐姐，不如就算了吧，姥姥正在闭关，也没说让我们出来害人。”
<br/>　　聂小倩咬着粉唇，美眸中满是挣扎，哀求出声，楚楚动人。
<br/>　　聂小倩是一个拥有极美容貌的女鬼，生前只活到十八岁，死后葬在浙江金华城北的荒凉古寺旁，不幸被妖怪千年树妖胁迫害人。
<br/>　　对于这种事情，她于心不忍，但在千年树妖的淫-威下，又不敢不从。
<br/>　　虽修炼鬼道数十年，也不过是刚刚入门的小鬼，跟千年树妖这等强大的妖怪比起来，差距不是一星半点。
<br/>　　“聂小倩，他们擅自闯入姥姥的地盘，死有余辜，难道你敢对姥姥不敬？”
<br/>　　闻言，聂小倩娇躯一颤，她自然知道后者是故意刁难她，想到树精姥姥的可怕，只得走上前来。
<br/>　　“你们，还有谁愿意去的？”
<br/>　　话语刚落，其中一名绿裙女子走了出来，女子小巧玲珑，相貌柔美，一看便是江南水乡女子。
<br/>　　“小蝶愿意和小倩姐姐一同去。”
<br/>　　说罢，两女相视一眼，身影诡异的消失，眨眼间已经出现在寺庙的僧舍门口。
<br/>　　小蝶望着燃着灯火的僧舍，看向聂小倩，笑嘻嘻的说道：“小倩姐姐，你选哪个？”
<br/>　　可能是对于小蝶的印象不错，聂小倩道：“我去对付那个世家子弟，你去对付书生，这种书呆子比较好对付。”
<br/>　　“小倩姐姐，你可要小心，必要的时候，千万不要手下留情。”
<br/>　　...
<br/>　　...
<br/>　　此时，长明灯将僧舍照的非常明亮，慕清霄静坐在木板上，宛如老僧入定。
<br/>　　“咚咚——”
<br/>　　就在这时，低沉的敲门声响起，慕清霄剑眉一皱，缓缓睁开星目，其内精芒一闪即逝。
<br/>　　“谁？”
<br/>　　“小女子是燕道长的侍女，见公子深夜未眠，特地给公子送些酒水果肉。”
<br/>　　门外，清脆婉转的女子声响起，宛如黄鹂鸣唱，非常的悦耳。
<br/>　　闻言，慕清霄眼角狠狠的抽搐了一下，他是第一次听说，牛鼻子道士身边会带着待女，真是长见识了。
<br/>　　若是让燕赤霞听到，不知道到是否会气的七窍流血而亡。
<br/>　　若他真是世家子弟，恐怕会信以为真，此刻，嘴角却是牵起一丝莫名的笑意：“有劳姑娘，门掩着没关，请进吧。”
<br/>　　“嘎吱——”
<br/>　　话语刚落，木门便被推开，只见一名身穿白裙，腰系白绫，身子妙曼的女子走了进来。
<br/>　　女子容貌绝美，脚步轻盈，青丝高高挽起，其上插着银簪，黑发如瀑直达腰际，迈着莲步。
<br/>　　见慕清霄盯着自己看个不停，聂小倩装作羞涩，捂着俏脸。
<br/>　　望着聂小倩好一会，慕清霄才收回目光，这并不是他第一次见到女鬼。
<br/>　　当初，在古剑奇谭位面的时候，他在自闲山庄也遇到过女鬼叶沉香，确实与平常女子没有任何区别，而且自带清冷的气质。
<br/>　　慕清霄打量聂小倩的时候，聂小倩也在打量着他，暗道：“果真如姐妹们所说，这位公子生的好生俊俏。”
<br/>　　“小姐如何称呼？”
<br/>　　“小女子姓聂，名小倩，公子唤我小倩即可。”
<br/>　　听慕清霄询问，聂小倩也没有隐藏姓名，反正过了今晚，眼前的青年不是化作一堆枯骨，便是再无瓜葛。
<br/>　　不过，她不相信，有男人会在自己的绝美容貌下，无动于衷。
<br/>　　说罢，便转身将僧舍木门关上，手中捧着托盘，其中有几样精致的小菜与一壶美酒。
<br/>　　当然，如此低级的幻术，怎么可能迷幻的住他，在慕清霄看来，无论小菜或者美酒，不过是几块碎石罢了。
<br/>　　将托盘放在慕清霄的面前，聂小倩才认真的打量起慕清霄。
<br/>　　面前的青年身穿白衫，一尘不染，肌肤如玉，简直令天下女子都嫉妒，一看就是娇生惯养。
<br/>　　黑发披散在身后，面如冠玉，盘坐在木板上，举止语气都透露着高贵，看来家世不小。
<br/>　　聂小倩小的时候便生在都城，本身就拥有大家闺秀的气质，但在眼前的面前，却宛如萤火皓月。
<br/>　　如此高贵姿态，死了当真可惜。
<br/>　　不过皮囊在好，终究是个凡人，若是真动了不该动的心思，就只能让他化作一堆枯骨长埋于此。
<br/>　　“公子不吃吗？”
<br/>　　将托盘摆在身前，聂小倩走到慕清霄身侧，柳眉微皱，隐约能嗅到他身上一股奇特的清香。
<br/>　　慕清霄低头看了眼身前的几块石头，真的不忍心揭穿她：“先前在镇上吃饱了，暂时吃不下。”
<br/>　　“长夜漫漫，既然公子不吃，不如让小倩服侍您就寝可好？”
<br/>　　说着，聂小倩的娇躯已经靠了上来，莲藕般的玉臂搂在他的颈部，臻首垂下，吐气如兰。
<br/>　　粉唇轻启，香气扑面而来，令得慕清霄鼻子有些痒痒的，看了眼怀中的佳人，不知道在想什么。
<br/>　　感受到怀中传来的柔软触感，慕清霄鬼使神差的说道：“这样真的好吗？你不是燕道长的侍女吗？”
<br/>　　说起来，他的妻子不少，但鬼他真没碰过，不知道是什么感觉，顿时令他心中充满了探索的欲.望。
