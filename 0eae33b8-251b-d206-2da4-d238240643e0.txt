　　关于如何自保这是李承乾一直在思考和进行的事情，身为一个来自后世的四有青年，再怎么样也不会把自己置身于一个绝地之中。
<br/>　　所以打从数年前开始，小李就一直致力于把老头子架空，团结一切可以团结的力量。
<br/>　　表面上看李承乾一直在折腾，但现实却是老头子的手下大多数都与他有着大量的资金往来，换句话说他的手里虽然有大量的资金，但这些钱基本上都是那些勋贵们的家当，如果有一天老头子真的想要把他弄死，老头子的那些手下第一个便会出来反对。
<br/>　　当然了，这也仅仅是一个方面，在其他方面李承乾也不是没有准备，只不过那都是后备的计划，不到万不得以的时候并不足以实施。
<br/>　　而此时的秦府之中，老秦在送走了李承乾之后，也陷入了犹豫。
<br/>　　不可否认，李承乾提出的建议对他来说有着很深的诱惑，可是他又不得不顾及李二的想法，老秦很清楚自己就是老秦左右手的问题，可如果与李承乾弄出这样不清不楚的关系，会不会影响秦家在皇帝眼中的地位呢？
<br/>　　可是老李总会死的，若是等到老李驾鹤西游了，再去找李承乾，那还能来得及么？
<br/>　　站队需尽早的道理谁都懂，可是需要选择的时候你就会发现，其实真的很难选择。
<br/>　　……
<br/>　　倭国，战局已经进入对峙的阶段，没有了唐军的支持归化人根本无力前突，只能固守。
<br/>　　而倭人的一方因为前期损失过大，此时也无力再发起反击。
<br/>　　当然，更多的原因是不敢反击，毕竟唐军虽然不参与进攻了，但却并没有撤回去，如果他们现在反击天知道会不会惹恼了这些杀神，到时候若是再参与进战争，那可真是灭国之战了。
<br/>　　山背大兄王，不，现在应该叫倭皇了，仗打到这个样子舒明这个前任倭皇难辞其咎，最后只能禅位于山背以换取自己未来的平安。
<br/>　　山背大兄作为新任的倭皇，坐在一间由庙宇改造的王宫之中，看着手下的几个虾兵蟹将不知在想些什么，只是那时而严肃，时而紧张，时而又带着微笑的脸让所有人看着有些心惊。
<br/>　　“皇上，那些归化人已经去大唐讨了封赏，现在他们已经建国，国名大和，同时又请了大唐一位公主遥领了国主之位，您看我们应该如何应对。”曾经的王府老管家斋藤见众人都不敢说话，犹豫了半天走了上来。
<br/>　　不得不说，这正是一人得道鸡犬升天的真实写照，山背大兄当了皇帝，连他的管家也都混了三品的位置。
<br/>　　“这本来就是意料之中的事情，那些归化人现在看唐人就像看到亲爹一样，发生这种事情还值得奇怪？”某位打不了胜仗，但嘴皮子还比较利索的家伙撇嘴说道。
<br/>　　“你根本就不知道事情的重点，所以请你把嘴闭上。”斋藤瞪了一眼那将军，没好气的说道：“脑子里只有肌肉的家伙，如果你们能打几场胜仗，何至于让皇上住在这样的环境里。”
<br/>　　“打胜仗？说的容易要钱没钱，要粮没粮，你去外面看看我们的勇士现在吃的都是什么东西！还有脸跟我们提打胜仗！武器呢？装备呢？拿我们的一身肉去堵枪眼儿么。”
<br/>　　说到打仗，虽然那将军从来没打过胜仗，但却依旧振振有词，说的某管家哑口无言。
<br/>　　“够了，都不要吵了！”山背大兄终于是听不下去了，狠狠一跺脚说道：“明天就给朕……给朕派一个特使去大唐军营送贺礼。”
<br/>　　“啥？”下在一群人几乎都愣住了。
<br/>　　本来还以为皇上发那么大的火能提出点什么有建设性的意见，比如来一场反击，打下两座城给归化人一个警告，结果没想到竟然是……去送礼。
<br/>　　有些无法接受这个现实的大臣苦着脸，纠结着问道：“皇，皇上，您，您没开玩笑吧？”
<br/>　　“朕像是在开玩笑么？”山背大兄板着脸反问道。
<br/>　　“那，那为什么我们要去送贺礼呢？如果不是大唐我们何至于落到现在这个地步！”
<br/>　　“因为如果不想处境更糟就只能这样作，这就是大唐所谓的花钱买平安，懂么？
<br/>　　不懂！所有人都不懂仗打成这个样子还有什么平安可言！但是却没人再说什么，任由那斋藤领了圣旨，转身出去。
<br/>　　完犊子了，看来皇上都对未来不报希望了，以后就等着灭国吧！所有的朝臣如是想着。
<br/>　　可是，就在众人失望的时候，却听山背大兄说道：“物部氏，你们还有多少战士？”
<br/>　　战士？物部氏族长愣了一下，随后想了想说道：“回皇上，大概还有五万人左右。”
<br/>　　“大伴氏，你们呢？”山背大兄又看向另一个人。
<br/>　　“大概三万左右。”大伴氏族长说道。
<br/>　　“藤原。”
<br/>　　“八万左右。”
<br/>　　“很好，朕的手下还有十万勇士，我们并不是没有一战之力。”‘啪’的一声，山背大兄狠狠的一拍椅子扶手。
<br/>　　什么？下面众人再次愣住了，疑惑不解的看着坐在原本佛像那个位置的山北大兄。
<br/>　　“看什么，朕刚刚那样做只是为了麻痹那些唐人，只要他们不插手到这场战役里面，我们就可以打败归化人，到时候从归化人那里抢了武器和粮食，我们就可以继续扩大军队，最后从唐人手中把丢掉的土地全都抢回来，然后再把他们都杀光！”
<br/>　　卧槽，真的假的？啥时候皇上这么牛逼了？竟然想出这么好的办法来？
<br/>　　物部氏的族长在这个时候似乎想到了什么问题，看着山背大兄说道：“陛下，可是如果这样的话，斋藤……”
<br/>　　“成大事者不拘小节，相信斋藤一定会理解朕的一片苦心的。”山背大兄的眼中带着一丝疯狂。
<br/>　　在他看来眼下就是最好的反击机会，归化人在庆祝胜利，唐人在沾沾自喜，如果把这个机会错过了，等到他们再次安定下来，估计会出现再大的麻烦，到时候再想反击只怕难上加难。
