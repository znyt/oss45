　　这个戴着鸭舌帽穿着便装，出现在东海市网咖包厢里的男人不是别人，正是赵国邦！
<br/>　　赵国邦穿着便装出现在网咖包厢打CS局域网，也难怪慕容飞在刚看到他的时候，会忍不住大吃一惊。
<br/>　　正如赵天宇所说的那样，赵国邦是慕容飞的伯乐。慕容飞是赵国邦一手挑选出来，并且带进龙隐部队的，所以由赵国邦来跟慕容飞谈“追魂”行动是最合适的。
<br/>　　赵国邦对慕容飞刚才的回答并不意外，其实当时在军区师部侦察连看到这个有着很高绘画天赋和射击天赋的小子时，赵国邦心中就已经明白，他并不适合特种部队，更不适合龙隐部队。
<br/>　　当年赵国邦问及慕容飞在军队将来的打算时，慕容飞脱口而出要去特种部队，赵国邦就知道慕容飞会有这个回答，很大一部分原因是源于受到影视剧的影响。
<br/>　　《第一滴血》，兰博，点燃了太多人心中的热血，感受到血管中的血液在奔腾，幻想着自己也能变成像兰博一样的超级英雄。
<br/>　　尽管知道慕容飞其实不适合龙隐部队，可是慕容飞表现出来的天赋还是让赵国邦心动不已，当时跟“暗月”之间的较量已经展开，方牧南重伤退役，龙隐部队急需一批新鲜的血液，所以赵国邦最后还是把慕容飞给带走了，带进了龙隐部队。
<br/>　　“人生轨迹的选择就是这么奇妙。”赵国邦叹了口气，说道：“你可能自己也没想到，当初一句‘愿意’，就这么彻底改变了你的人生。”
<br/>　　“是啊。”慕容飞叹了口气，开玩笑似得说道：“如果给我一次重新选择的机会，我一定会选择拒绝。”
<br/>　　慕容飞说话的语气，看似是在开玩笑，可赵国邦知道这是慕容飞最真实的想法。
<br/>　　这个世界不会有月光宝盒，也没有人能像至尊宝一样，大喊一声“菠萝菠萝蜜”就能让时间倒流回到过去。
<br/>　　“你今年34岁了。”赵国邦对慕容飞说道：“还有一年，不，还有十个月你就要从作战部队强制退役了，想过有什么打算吗？”
<br/>　　“野狼，你千里迢迢从京城跑到这里来，不会就是为了找我打几局局域网CS，然后关心我作战部队退役后的生活吧？”慕容飞对赵国邦问道。
<br/>　　话说到这个份上，赵国邦也就不再瞒着什么了，拿起那份他从进入包厢就放在桌子上的“追魂”行动计划书，递给慕容飞，说道：“这是‘追魂’行动计划书，承影跟我说过了，这是你在战术中队的最后一次任务，完成以后就从战术中队退役，去红旗学校担任狙击战术教官。”
<br/>　　慕容飞起身接过赵国邦递过来的行动计划书，还没打开，只是看到名字便是笑了出来，说道：“‘追魂’行动，看名字这个行动就是为我量身打造的。我的代号叫‘无常’，无常的工作不就是招魂、追魂吗？”
<br/>　　“行动计划是暗影提出来，如你所说，之所以定名为‘追魂’行动，的确是把你作为了这次行动的核心执行者，或者说，是核心执行者的第一人选！”赵国邦看着正在阅读行动计划书的慕容飞，说道：“这次行动的危险程度很高，承影、暗影和老狐狸头也是商议了很长时间，之所以把你选择为核心执行者的第一人选，是因为你目前的情况是最合适的，如果换做其他人，我们就需要重新创造契机，这在时间上可能会来不及。我们跟‘暗月’交手了很多次，要说‘暗月’的核心人物，‘天使’我们也解决掉了不少，但是这还不足以伤及‘暗语’的根基。虽然不清楚小丑到底是不是‘暗月’真正的老大，但是如果能把小丑拿下，对‘暗月’来说也足以是伤筋动骨！”
<br/>　　“当然，如果你有顾虑也可以拒绝，最多我们放弃这次行动计划……”
<br/>　　“我答应！”
<br/>　　赵国邦的话还没说完，慕容飞就打断了他的话，直接表态自己愿意执行这次的“追魂”行动。
<br/>　　“你想清楚了？”赵国邦再次像慕容飞确认道：“这次行动危险程度很高，目前我们初步评定是A+，你很清楚，这意味着什么，稍有不慎，你就回不来了。而且你一旦进入执行阶段，过往的种种荣誉和荣耀都跟你没有任何关系了，而且在有一段时间中，你的身份会处于保密阶段，你清楚我是什么意思，你会跟以前的兄弟们……”
<br/>　　“我明白。”慕容飞合上行动计划书，看着赵国邦说道：“正如您所说，我是目前为止这个行动核心执行者的最佳人选，如果你们换人需要重新创造机会，这在时间是上一个很大的浪费，很有可能就会使这次行动计划流产，失去这次机会，我们很难再有下次机会，这是其一；其二，因为玛丽科娃！这个女人手上沾着太多的鲜血，因为千雪我跨过了我的心理障碍，可是这并不意味着我会忘记跟玛丽科娃之间的恩怨，血债必须要血来偿！如果说还有第三点，这是我的使命，只要我还在战术中队，这就是我的使命！使命，不容抗拒！”
<br/>　　慕容飞的三个理由，连赵国邦都忍不住为之动容，开口道：“不愧是我选出来，带进龙隐部队的兵，好兵！”
<br/>　　“野狼，在正式进入行动阶段之前，我也有几个要求。”慕容飞对赵国邦说道。
<br/>　　“你说，只要是在我能力范围内的，我一定会帮你办到。”赵国邦点头回答道。
<br/>　　“第一，我这次来东海市是为了千雪，我想在行动开始前，先陪她看完病。”慕容飞提出了自己的第一个要求。
<br/>　　“没问题，我可以答应你。”赵国邦点头答应了下来。
<br/>　　“第二，我想请你们务必对千雪保密。至少在我身份曝光前，请对她一定保密。”
<br/>　　赵国邦点头。
<br/>　　“第三。”慕容飞深吸一口气，说道：“如果我的行动失败，我希望能在行动失败后，能跟我父母说清楚情况。”
