　　上山的这个道士是山河真人，山顶这一块地方本来是他专属的修所，但现在他却没有资格上去了，因为现在山顶打坐修行的是的老祖宗青牛祖师。
<br/>　　山河真人不敢过去，离着老远就跪下了。
<br/>　　过了半柱香的时间，李浩然睁开了眼睛，周身那些紫色的如同花瓣一般的灵气全都汇聚在了左手上，直到完全被吸入。李浩然抬起手，转了转手腕，感受了一下手腕的变化，还算满意，然后头也不抬地问道：“何事？”
<br/>　　山河真人磕了个头，说道：“启禀祖师，今日又有人来求见老祖。”
<br/>　　“哦。”李浩然道：“来人不一般？”
<br/>　　自从法帖散发出去之后，最近一直有法术界的人前来打探消息，得知桃花山多了一个神秘强者之后，纷纷挑战。
<br/>　　李浩然出于自己的目的，来者不拒，将那些所谓的强者尽数打趴，这件事，山河真人和门派上下都是很爽的，毕竟桃花山本身只是一个不大不小的门派，这一次李浩然坐镇，也是桃花山扬名的机会，山河真人对李浩然言听计从，十分小心地伺候这位老祖宗。
<br/>　　“回祖师，这次来的人，非同寻常……”
<br/>　　“六大宗派的人？”
<br/>　　山河真人摇了摇头。
<br/>　　李浩然道：“既然不是，却有什么值得慌张？”
<br/>　　山河真人道：“祖师啊，这人虽然不是六大宗派之人，风头却甚至盖过他们，今天来的这位，是法术公会的恶一位高人，星月奴的大弟子鲲鹏，在法术界也是说一不二的人物，就算是几大宗派的宗师也要让他几分，他性格张狂，今日来，指明要跟祖师交手，不知祖师……”
<br/>　　“他已经来了。”李浩然看向山路，在转弯处有一片紫竹林，山河真人也转头看过去，什么也没看见。
<br/>　　一阵竹叶的响动，一个人拨开竹子走了出来。
<br/>　　是一个身材修长的黑脸男子，大步流星，一边嘴角微微翘起，神色中带着一丝凌然的傲气，看着就是张狂的人。
<br/>　　山河真人顿时有些不爽，起身说道：“鲲鹏师兄，这山顶是我门派禁地，你这么擅闯，似有不妥吧。”
<br/>　　“得罪，贫道等了半天，已经迫不及待要见一见你门中这位高人。”鲲鹏径直走过来，也不看山河真人，直接走到他身边，对李浩然拱了拱手，道：“不知高人怎么称呼？”
<br/>　　李浩然慢慢地走出亭子，并不回答，说道：“听说你们法术公会最近接管了人间法术界？”
<br/>　　“不敢，只是人间大乱，法术公会前来引领诸位，指正乱象。”鲲鹏答道，语气中带着一丝傲慢和得意。
<br/>　　李浩然道：“人间法术界历经数千年，一直没有灭绝，什么时候需要你们法术公会来插手了？”
<br/>　　鲲鹏笑了两声：“那也只是人间无才，否则怎么轮得我们法术公会？”
<br/>　　见李浩然不语，又说道：“阁下是桃花山之人？”
<br/>　　“我曾在人间。”李浩然说了句没头没脑的话，随即问道：“我听说星月奴门下六子，法力极深，你是其中之一？”
<br/>　　“贫道正是。”鲲鹏一笑，“我今日来桃花山，倒想领教前辈的人间法术。”
<br/>　　“你没资格。”李浩然淡淡说道。
<br/>　　鲲鹏脸色一变，道：“前辈是看不起我法术公会？”
<br/>　　山河真人一听，心中有些发紧，现如今这时节，就算是六大宗门，都不敢太得罪法术公会，鲲鹏又是法术公会的红人……
<br/>　　李浩然缓缓走过去，道：“轩辕氏中，人才辈出，但你不是。”
<br/>　　鲲鹏冷哼一声：“听前辈这口气，似乎很了解我们轩辕氏。”
<br/>　　李浩然抬起一只手，紫光乍现，在指尖环绕，说道：“我当年随我主周游太虚，开坛传道，轩辕氏族无不拜服，就算你师父星月奴，也不过是一个晚辈，你又算个什么东西？”
<br/>　　鲲鹏脸色大变，冷冷道：“前辈这牛皮吹得就大了，不过，前辈既然这般高高在上，居然有主子，可见也不过是个奴才，不知道前辈的主子是个什么人物？”
<br/>　　李浩然道：“我主，是天地大道之所在。”
<br/>　　老子为三清道祖，能给他当奴才，是世上所有修道之人可望而不可及的事。李浩然也懒得跟他多说，弹了一下手指，数道紫气飞射而出，在空中形成了一个“井”字，朝鲲鹏打了过来。
<br/>　　鲲鹏没想到他说打就打，立刻后撤半步，双手结印，念动咒语，袖子里飞出数十张灵符，在空中形成符阵，试图挡住紫气。
<br/>　　结果只是阻挡了差不多一个瞬间，便被紫气击落，继续朝他飞来。
<br/>　　鲲鹏一怔，双手结印，口中飞快念道：“孔雀明王，妙法乾坤，以我之名，护佑我身！着！”
<br/>　　一声鸣叫，从鲲鹏的头顶飞出了无数灵光，在上空形成了一只孔雀的形状，朝李浩然飞去。
<br/>　　“孔雀明王……”山河真人失声叫起来，没想到这个鲲鹏祖师还是道佛双修。
<br/>　　李浩然欺身而上，一只手在空中连画几下，形成了一个异体的“门”字，在手心浮现，朝那灵气所化的孔雀打了下去，鲲鹏也使出了十二分的法力，操控孔雀形体，朝山河真人手心的符文撞过去。
<br/>　　“轰”的一声巨响。
<br/>　　山河真人被余波扫到，急忙运气阻挡，等余波过去，定睛看去，李浩然一只手贴在鲲鹏的头顶正中间，掌心紫气隐现。
<br/>　　鲲鹏一脸懵，似乎不太明白发生了什么事，身边，灵气化形飞落，像是孔雀的羽毛。正是那只被他用灵气召唤而成的孔雀明王……
<br/>　　“孔雀明王法诀深不可测，不过你只学了皮毛。”李浩然声音很冷。
<br/>　　鲲鹏这才明白发生了什么事，满脸通红，但是不敢动一下，否则只要李浩然的手轻轻一按，他就要元神俱灭。
<br/>　　“前辈的手段，贫道领教了……”鲲鹏被逼出了这么一句求饶的话。
<br/>　　“你当我不敢杀你？”
<br/>　　鲲鹏怔了一下，道：“前辈杀我，便是自罪于法术公会……”
<br/>(本章完)
