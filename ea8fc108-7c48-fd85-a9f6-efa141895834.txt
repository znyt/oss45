　　踏入这山峰上的一刻，便是就足以感觉到一股巨大无比的压力。
<br/>　　就连楚风眠都是差一点，都被赶下道路上去。
<br/>　　不过很快，当楚风眠熟悉了一下这压力的时候，他便是发现，这压力虽然不下，但是极为稳定。
<br/>　　对武者来说，最多也只能够是让其多多花费一些力气，但是却远远无法阻止。
<br/>　　看来这也应该就是云羽天宫的考验之一，要承受这如此之大的压力，登上这山峰，才能够经历真正的考验。
<br/>　　“走！”
<br/>　　楚风眠观察了周围一阵，便是再度催动起这战龙宝车来，直接顺着道路，向着山峰上直接冲了过去。
<br/>　　这战龙宝车，乃是天级灵器，这种压力，虽然对战龙宝车来说有着一些影响，但是却微乎甚微，他的速度，比起大部分武者的速度要快得多。
<br/>　　一眼望去，这道路上的无数武者，都被这战龙宝车，远远的甩在后面，不少人都被直接超过。
<br/>　　“好，抢先一步，免得这云羽天宫的宝藏传承，落入到其他人的手中！”
<br/>　　楚风眠有些庆幸到。
<br/>　　也幸好这山峰上，还有这样一道考验，阻挡了大部分武者的速度，不然的话楚风眠想要追上来，也不是一件容易的事情。
<br/>　　这种远古宗门的传承，洞府，进入其中虽然运气也极为重要，但是谁若是第一个进入其中，好处绝不会少。
<br/>　　像是楚风眠就是在颜古龙帝洞府之中，抢先一步进入其中，才可以得到祖龙至尊剑，得到那血海。
<br/>　　要是当时楚风眠没有得到颜古龙帝洞府之中的无数宝藏的话，也许现在的楚风眠都是死了一万次了。
<br/>　　这次楚风眠也必须要第一个，进入到这云羽天宫之中去。
<br/>　　楚风眠的嚣张举动，如今已经是引得了道路上不少武者的敌视，尤其是那些被楚风眠超越了武者，一个个眼神中都是露出杀意。
<br/>　　他们自然也知道，这种远古洞府，谁要是能够抢先一步，就是抢占先机。
<br/>　　楚风眠都感觉到了，无数杀意，已经是笼罩在了楚风眠的头上。
<br/>　　至少有着七八人，已经是盯上了楚风眠，要将楚风眠置于死地。
<br/>　　“小子，滚下去！”
<br/>　　突然之间一声爆喝，就在楚风眠的前方响起，随着一声巨大无比的轰鸣声，一道巨大无比的手掌，顿时向着楚风眠轰击过来。
<br/>　　在这手掌之中，蕴含有着无尽的黑暗，这一道巨大无比的手掌，不止是要将楚风眠轰下去，更是要将楚风眠身后的无数武者，全部打落下去。
<br/>　　“古月震天掌，此人是古月峡谷的弟子，古鹏，在古月峡谷弟子中排列前三，曾经我跟他也有过一次交手。”
<br/>　　寒月漓看到这掌印到来，便是一眼认出这出手的武者到底是谁，出声道。
<br/>　　“古月峡谷的弟子，也敢来对我出手，找死？”
<br/>　　楚风眠眼神冰冷，他都没有出手将别人打下去，这古鹏倒是不知天高地厚，敢来向楚风眠出手。
<br/>　　面对这巨大无比的掌印，楚风眠冷哼一声，同样大手一挥，无数远古战龙精血便是凝聚起来，化为了战龙之爪，同样轰击过去。
<br/>　　“砰！”
<br/>　　两股力量相对，这掌印顿时全部破碎，四分五裂，楚风眠冷喝一声，战龙之爪更是向着前方轰击了过去。
<br/>　　只见一道身影，从这山峰上，突然掉落了下去，他的胸口都被打碎，受了重伤，正是那古鹏。
<br/>　　这古鹏，被楚风眠一掌之下，直接从山峰上打落了下来，如今直接掉到了山峰之下。
<br/>　　“此人倒是没死，让他捡了一条命。”
<br/>　　楚风眠看到古鹏直接掉落下去，冷哼一声道。
<br/>　　他也不想再去追杀这古鹏，对楚风眠来说，还是这云羽天宫更加重要，至于这古鹏，在让楚风眠碰到他的时候，在杀也不迟。
<br/>　　“走！”
<br/>　　战龙宝车，猛然加速，直接向着道路上方，驰骋而上，这一路上的其他武者，看到楚风眠刚刚出手，一个个的杀意也离开了楚风眠的身上。
<br/>　　就连古鹏都被楚风眠一掌打了下去，他们在对付楚风眠，不是都在自寻死路，便是只能眼睁睁的看着楚风眠，超越他们。
<br/>　　就在楚风眠冲上去的不久，又有十几名武者，速度极快，连连超越无数人，冲了上去。
<br/>　　这些人，乘坐一方天舟，数人合力之下催动，来抵挡威压，速度比起武者压快得多。
<br/>　　“荒古门的人！”
<br/>　　看到这一群人的到来，其他武者纷纷退到一边去，不敢挡路。
<br/>　　“那小子走的倒是真快，他那坐下宝车，应该就是九魔城的天魔宝车，没想到居然是落到了此人的手中。”
<br/>　　在这天舟上，一名荒古门弟子冷笑着开口道。
<br/>　　“据说九魔城的人，都被此人灭门了，这小子倒是狂妄，放言要杀我们所有七大宗门的人。”
<br/>　　“圣君不用在乎这小人的话，一个北域小子，得到些奇遇就自以为慌张了，碰到圣君此人也不过就是不堪一击的货色罢了，倒时候圣君出手，轻易便是足以将此人斩杀便是。”
<br/>　　又一名荒古门弟子不屑的开口道。
<br/>　　“他那坐下宝车，也迟早是圣君的。”
<br/>　　“天魔宝车，的确是至宝，此人的手中，据说还有着龙族至宝，祖龙至尊剑，杀了他倒是可以得到大大的好处。”
<br/>　　荒圣君冷冷的开口道。
<br/>　　“追，这小子逃不远，他既然想要得到这云羽天宫的宝藏，我们总能碰到，倒时候我就杀了此人！”
<br/>　　一边说着，一边这天舟便是超越无数武者，直接冲了上去。
<br/>　　另一边，楚风眠也察觉到了，在他的后面，这一群武者的存在。
<br/>　　不过现在楚风眠也没有心思，去多注意他们，如今楚风眠一路驾驭宝车，已经是快要抵达这山峰山顶。
<br/>　　这道路上的压力，也越来越强，达到了顶点。
<br/>　　一般实力稍弱的武者，甚至都通过这压力，进入到山峰之上。
<br/>　　但是这种压力，在楚风眠的面前，却是算不得什么。
<br/>　　“冲！”
<br/>　　楚风眠冷喝一声，他坐下的战龙宝车，便是猛然冲了上去，直接打破这压力，直接冲破了这压力大道。
