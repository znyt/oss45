　　眼见苏莫和扬天威两人飞上了星空之中，其它围观的武者，亦是不愿意错过这等好戏，纷纷向星空之上飞去。
<br/>　　云悠悠见此，眸中闪过迟疑之色，她并不太想等，但是苏莫还没等她说回答，便已经飞上了星空。
<br/>　　“神女，我们也去观战吧！”杨天成面露温柔的微笑，对云悠悠说道。
<br/>　　他可不想错过这次大战，亲眼看着苏莫被打败，这可是让人非常兴奋的事情，他当然不想错过。
<br/>　　“嗯！”云悠悠微微点了点头，她也是有些好奇，苏莫能不能打败杨天威，半柱香的时间而已，她可以等。
<br/>　　随即，云悠悠身形飘飞而起，向星空之上飞去。
<br/>　　扬天成亦是急忙跟了上去。
<br/>　　……
<br/>　　无边无际，黑暗的星空之中，苏莫和杨天威长身而立，远处大量的武者围观。
<br/>　　“苏莫，你扬言半柱香打败我，我倒要看看，你的实力是不是向口气那么大！”
<br/>　　杨天威遥望苏莫，朗声大喝，他的手中长刀刀芒千丈，恐怖无比，仿若是握着擎天巨刃。
<br/>　　此刻，杨天威的眼眸之中，带着淡淡的戏谑之色，因为他压根就不相信，苏莫有打败他的实力。

<br/>　　咻！
<br/>　　这一剑没有任何的花哨，只是简简单单的一剑，但威力不可谓不大。
<br/>　　霎时之间，一道万丈剑气飚射长空，裹挟惊天动地的杀机，向杨天威袭杀而去。
<br/>　　“雕虫小技！”
<br/>　　杨天威见此，顿时不屑一笑，但他的心中却是有些惊讶，因为苏莫这一剑，很明显拥有武尊境三重武者的实力。
<br/>　　虽然这个实力，还不足以和他抗衡，但以苏莫的修为来说，已经非常的逆天了。
<br/>　　“碎！”
<br/>　　随即，杨天威出刀了，一刀直劈而出，一道恐怖的白色刀气，如同九天闪电，狠狠的斩在了三色剑气之上。
<br/>　　轰隆隆！
<br/>　　爆响震天，剑气和刀气相击，纷纷爆裂，将虚空炸穿，恐怖的空间乱流混杂着毁灭的冲击破，横扫九天。
<br/>　　“杀！”
<br/>　　面对毁灭般的冲击破，苏莫不退反进，身上光芒耀眼，如同金刚浇铸的身体上，伴随着白光、琉璃之光璀璨无比，直接向杨天威冲了过去。
<br/>　　与此同时，苏莫催动了十几种剑之战魂，以及他的吞噬战魂。
<br/>　　他可没有时间，和对方慢慢的试探，现在，他要展开最强的战力，击败乃至击杀对方。
<br/>　　嗡嗡~~~
<br/>　　无上的剑威冲天而起，如同浩荡九天的雷霆，让得整个天地都为之颤抖，整片空间不断的出现裂纹。
<br/>　　恐怖的吞噬之力，伴随着吞噬奥义，如同席卷的汪洋，瞬间充斥整片空间。
<br/>　　惊人的一幕出现了，强大的吞噬力量，让得整片空间仿佛陷入了停懈之中，随即无尽的冲击破逆流，纷纷向苏莫汇聚而去。
<br/>　　“斩！”
<br/>　　惊天的暴喝声响起，苏莫如同闪电一般，已经急速的接近了杨天威。
<br/>　　他一剑挥出，如同霹雳长空，一道无比惊艳的三色剑气，再次向杨天威斩击而去。

<br/>　　所有观战之人，见此无不心神剧震，因为苏莫这一刻展现的实力，已经远远的超远了修为层次，简直恐怖到了极致。
<br/>　　“怎么可能？”扬天成面色一呆，口中倒吸了一口凉气，因为苏莫这一击，貌似比杨天威的战力都只强不弱。
<br/>　　“厉害！”云悠悠也是杏眼大睁，心灵感受到了强烈的冲击，距离上次太阴族内的会武，不过才过去了短短两三个月的时间，苏莫的实力，已经不知道提升了几倍。
<br/>　　“有点意思！”天狼眼眸微微一眯。
<br/>　　“这么强？”乙薇脸上现出震惊之色，她作为道宗真传弟子第一人，自然早就听闻过苏莫的妖孽之处。
<br/>　　但今日一见，似乎苏莫比传闻之中还要更加的妖孽。
<br/>　　而此刻，在苏莫攻击之下的杨天威，最是震惊。
<br/>　　“什么！”杨天威脸色一变，心中剧震，因为在苏莫这一剑之下，他感觉到了极度危险的气息；尤其是那种欲将一起都吞噬的力量。
<br/>　　这让他震惊不已，这就是苏莫的实力吗？怎么可能会这么强大？
<br/>　　在这一剑之下，杨天威甚至生出一种抵挡不住的感觉。
<br/>　　“吼！”
<br/>　　眼见恐怖的剑气，裹挟滔天威势袭杀而来，杨天威顿时暴吼一声，震动天地。

<br/>　　“这就是杨天威的特殊体质！”
