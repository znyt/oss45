　　泰山之顶，伏羲从证道天皇，到突破半步混元，之后又在极短时间内突破极道帝境，速度之快，让很多人都没来得反应过来。
<br/>　　看着从虚空之中走下来的伏羲，诸神眼中都流露出惊骇的神色，他们怎么也想不到，伏羲刚刚突破半步混元，怎么一点预兆也没有，便突破混元大罗金仙了，这怎么可能？
<br/>　　诸圣也流露出难以置信的眼神，女娲的脸上更是十分惊喜，不过，同时，她心中也带着浓浓的不解，她不知道为何伏羲会如此快便突破混元。诸圣之中，也只有元始若有所思，好似知道什么。
<br/>　　元始将目光投向伏羲，眼中神光爆射，瞳孔深处，好似有无尽人影在其中，他们交织坐下，手捏印诀，好似在推衍着什么。
<br/>　　时光逆流，在元始的瞳孔深处，出现一幅幅模糊的画面，画面之内，有两道人影，仔细看去，隐隐可以看出这两道身影都是伏羲。
<br/>　　两道身影好似在交谈着什么，让人听之不明，也无法看到。但这一切都瞒不过元始，元始瞳孔深处神光爆闪，他以强大的神通拨开重重迷雾，听清了他们谈话的内容。
<br/>　　“嗯？两个伏羲，一个是穿越者，一个是洪荒伏羲？两朵相似的花？万我归一，超脱彼岸么？原来如此。”元始心中喃喃道，脸上却露出一副沉思的状态。
<br/>　　既然伏羲有两个，那么他呢，是不是也会有两个元始？这也不是不可能，这种事谁也说不准，诸天万界，无尽宇宙，出现其他元始，也不是不可能。
<br/>　　眼前的伏羲便是两个伏羲合一，导致修为突破，实力大增，自己是否也可以通过找到其他的“元始”，将他们合一，来提高自己的修为？
<br/>　　想到这里，元始就有些激动，若是此事可行，便可以节省元始许多时间，而且吸收其他“元始”的大道，对自己的元始大道一样有好处。
<br/>　　不过，很快元始便摇了摇头，此法的确可行，但单找到其他“元始”这一件事，就十分困难。不同的元始分布在不同的世界，想要找到他们是何等的困难，虽说两者之间存在某种微妙的联系，但这种联系小到可以忽略不计，是十分难寻找的。
<br/>　　不过，元始精通因果大道，两者之间的因果联系虽然微小，但只要元始愿意花费时间去寻找，还是可能找到的。
<br/>　　但这也十分麻烦，对于元始来说，与其浪费时间去找其他“我”，还不如用来修炼呢，但此事也不是不可行，只是需要元始仔细考虑一番。虽然此事十分重要，但孰轻孰重，元始还是知道的。
<br/>　　这些心思只是在元始一念之间闪过，看着走到祭坛上的伏羲，元始撤去眼中的神光，恢复了淡然的模样，就好像刚才所发生的事，与自己无关似的。
<br/>　　高台上的伏羲，在被元始看到的瞬间，他心底没由来的出现短暂的心悸现象，就好似自己被什么神秘人物盯住了似的，自己的一切秘密，都掩藏不住，好似被他人看去一样。他感到十分心惊，自己已经是混元大罗金仙了，可还是能有这种感觉，那就说明，窥视他的人，实力超乎想象。
<br/>　　心念一动，伏羲便知道了窥探自己的人是谁了，洪荒之中能有如此本事的人，本就是寥寥无几，能让自己一个混元大罗金仙依旧感到心惊胆战的人，那就更少了。
<br/>　　他不动声色的看了一眼元始，发现元始早已恢复了常态，脸色淡然，看不出丝毫的动手迹象。伏羲不但没有放松，反而更加警惕了，他不知道元始是不是真的什么也没看到，元始给他的感觉就好似一个普通人，越是这样，他就越是心惊。
<br/>　　元始目光与伏羲对视一眼，露出一个似笑非笑的表情，让伏羲不由得心中一惊，但他面上却不动声色。
<br/>　　接着，伏羲有看向其他圣人，可惜，圣人一个个都是天地间的最强的一批人，心境修为一个个早已修炼到不知有多远，喜怒不行于色，那是最基本的修为，他很难从圣人的脸上看到什么信息。
<br/>　　再说，就算圣人有什么心思，也不会让他看到。倒是女娲娘娘，脸上明显露出了明媚的笑容，伏羲看到她，便想起虚影最后的一句话，他向着女娲点了点头。
<br/>　　轰！
<br/>　　八卦韵律扩散，席卷苍穹，一道巨大的轰鸣声，从伏羲体内传出，传到四面八方，传遍整个人族。
<br/>　　“我等拜见天皇！”
<br/>　　这一刻，无数人族嘶吼，一个个顶礼膜拜，望着高台上的伏羲，露出敬畏之情。
<br/>　　“伏羲证道天皇，功德圆满，三月之内，必须到火云洞，镇压人族气运，方可保人族气运经久不衰，万世长存！”
<br/>　　就在此时，元始面无表情，淡淡出声道。
<br/>　　“谨遵圣主圣谕！”
<br/>　　听了此话，伏羲双眼一眯，拱手向着诸圣一拜，随后便向着人族飞去，他只有三个月的时间，三月之内，他需要交代好人族一些事情，方可安心坐镇火云洞修炼。
<br/>　　天皇伏羲走后，诸圣自然也就各自消失于虚空之中，却是回各自的山门去了。
<br/>　　然而，元始并没有回到昆仑山，而是身影一闪，来到一条无名小河边，在河边，他看到一个五岁大小的小孩。
<br/>　　小孩手持一柄木剑，在河边挥舞着，剑光森寒，剑刃凌厉，剑剑逼人，端是一手好剑法。显然，这个小孩并不简单，以如此年纪便修炼出如此凌厉的剑法，能简单才怪。
<br/>　　元始在高空中看了良久，才一步步地走了下来，走向那个小孩。
<br/>　　“你这是在练剑么？”元始开口问道。
<br/>　　“当然！”
<br/>　　小孩手持木剑，傲然回答道。说完，他才感觉不对劲，怎么自己的嘴不受自己控制，还没有看到来人，便不由自主的回答了出来。紧接着，他警惕的看着元始，问道：
<br/>　　“呃，你是谁？”
<br/>　　“你说呢？”
<br/>　　元始微微一笑，如沐春风，让人一下子忘却了所有的烦恼，生不出丝毫的恶感。
<br/>　　“你是仙人！你一定是！”
<br/>　　小孩眼睛一亮，顿时回答道。他最向往那些仙人飞天遁地，御剑逍遥，杀妖除魔了，这样的事，他做梦都想，可惜，他一直没有见到真正的仙人。如今这人竟然能够从天上走下来，那么一定是传说中的仙人。
<br/>　　“你说是，那就算是吧。”元始微笑着说道，接着他继续问道：
<br/>　　“你名为何？”
<br/>　　“公孙轩辕！”
