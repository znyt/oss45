        &lt;h3&gt;第三卷第二百一十八章  起疑&lt;/h3&gt;
<br/>
<br/>        潘青松又开始要忙碌起来了，新年的几天假期虽然是在医院度过，却是他难得的清闲的时间，他拒绝了一切应酬，安心的陪着陈子澜，也拒绝了一切来探访的人，潘禺随着陈子昂一起去了北京，现在就剩他们俩人在病房里。
<br/>
<br/>        陈子澜在床上躺着，无论是吃喝都得需要人照顾，经过他细心的调养，陈子澜脸色已经恢复如常，就是骨折好起来很慢。
<br/>
<br/>        每天他都会给陈子澜变着花样做吃的，俩个人在一起其乐融融，似乎又回到了旧时光。
<br/>
<br/>        “子澜，你知道吗，第一次见你的时候我就喜欢上了你，这一喜欢就一辈子。”
<br/>
<br/>        “我就知道你小子坏得很，躺地上耍赖。”陈子澜眼里都是笑。
<br/>
<br/>        潘青松轻轻的将陈子澜的秀发用小皮筋扎起来，扎成两个小辫子，一瞬间躺着的陈子澜变成了一个小萝莉。
<br/>
<br/>        “不耍赖怎么能被你注意？当时我们单位可是只有你一个大美女。”潘青松拿过镜子，“你看，你美不美？”
<br/>
<br/>        “哎呀，潘青松，你真讨厌，怎么能给我扎这样的小辫子？”陈子澜捂着眼睛笑。
<br/>
<br/>        “大妈秒变小萝莉，不行，我也要秒变小帅哥，这样才能跟上你的脚步。”潘青松哈哈大笑。
<br/>
<br/>        “得了吧，你还妙变小帅哥？在我心中你从来都不是小帅哥。”陈子澜笑得眼泪都出来了。
<br/>
<br/>        “对啊，我一直都是你的大帅哥，子澜，我就想不通，你比我小，你竟然能来当我们的导员，简直是刷新了我的三观，当时我们几个人都不是你的对手。”
<br/>
<br/>        “天赋加勤奋，没法比！”陈子澜笑着打击潘青松。
<br/>
<br/>        “我也很勤奋，不过是遇见你之后才开始勤奋的，就害怕到时候打不过老婆，动不动的就被暴揍一顿，每天顶着熊猫眼上班是很丢人的。”
<br/>
<br/>        “油嘴滑舌，我哪有暴揍你？”陈子澜看着潘青松，人生何其有幸啊，能够认识潘青松，这一生她也无憾了。
<br/>
<br/>        “哼，只要是你不想动的时候，我就是被压制的对象。”潘青松傲娇的转过身子。
<br/>
<br/>        “说得你好像是小媳妇一样的。”陈子澜伸手抓住潘青松的手，潘青松反手将陈子澜的手握在手心。
<br/>
<br/>        “那可不，我不是一直都是你的小媳妇？”
<br/>
<br/>        “潘青松，你不要一副小媳妇样了，受不了你了，羞不羞耻。”
<br/>
<br/>        “在你面前我宁愿一辈子是小媳妇，不羞耻。”潘青松柔情蜜意，但是今天反转角色让陈子澜还是有些不习惯。
<br/>
<br/>        “你今天这么特别？”
<br/>
<br/>        “哪里特别了，想一想这个新年竟然是我们俩个人最私密的新年，平时上班都是忙忙碌碌，不是陪家人，就是到处跑，难得我们两个人人在一起呢。”潘青松感觉自己的眼眶湿润。
<br/>
<br/>        “只是辛苦你了，天天的陪着我，哪里都不能去。”
<br/>
<br/>        “不，我觉得是我最幸福的时光，能天天和你呆在一起，是我求之不得的事情。平时你的心思都不在我这里，不是父母亲就是孩子，我呢，就是你的一个挂件，想起来了拿起来挂在身上，想不起就不知道放在哪里。”潘青松像是一个要糖吃的小孩。
<br/>
<br/>        “潘青松，你这个摆设挂件加花瓶，我现在想起你了，快过来，宠幸一下你。”潘青松压着嗓子学陈子澜说话。
<br/>
<br/>        陈子澜听得眼眶湿润，人生在世，那能只是为了自己而活？像潘青松说的，他们确实都是在为别人而活，为父母，为孩子，为了肩上的担子。
<br/>
<br/>        “你这样一说，我还得感谢这次车祸呢，要不然我们就没有这样安宁的生活。”陈子澜感慨万千。
<br/>
<br/>        “不过，青松，我真的很担心子昂去甸城的，我们去甸城基本寸步难行，好不容易出门了，还弄得车毁人亡，我怕子昂引火烧身，我回来的这一段时间也考虑了很多的事情，甸城不是我们看到的那样太平，一切可能都是假像。”
<br/>
<br/>        “你也别太纠结，子昂在社会上这么多年，她应该有能力处理一切事情。他们是生意人，哪里有机会去哪里，是正常的，而且他们做生意很难触及到那些阴暗面。潘禺和子昂一起去北京了，年轻人自然有年轻人的想法，你好好养病。”
<br/>
<br/>        “上班了你就该忙起来了，我这里有护工，你也不要太劳累，别太担心我，现在你可是我们家的顶梁柱呢。”
<br/>
<br/>        “子澜，和子悦一起的那个闫欢也去甸城了，子悦说他们也是过去投资，好像和子昂商量好了。”潘青松皱着眉头，有些纠结。
<br/>
<br/>        “你的意思是，子昂和闫欢有秘密？”陈子澜和潘青松夫妻同心，彼此一个眼神都能明白对方在想什么。
<br/>
<br/>        “是啊，这就是我想不通的地方，他们俩人按说没有太多的交集，为什么闫欢会要去甸城投资？而且子悦都拦不住，闫欢几年前失踪过，失踪的那几年一直是个迷。另外，闫欢会拿着他们很大的一部分资金去甸城。”
<br/>
<br/>        “子悦怀疑闫欢了？”
<br/>
<br/>        “子悦倒是没有怀疑闫欢，她说她们有现在的成就一大部分也是闫欢的功劳，只是她很奇怪，为什么闫欢那么想要去甸城，要说资源，这边很多市场都没有拓展，正常的选择是不会去那么远的地方的。”
<br/>
<br/>        “要不，你查查闫欢？”
<br/>
<br/>        “我查过了，闫欢失踪的那几年是个迷，能做到这些的不是一般人，明显的被人抹去了痕迹，所以我觉得闫欢的背景有些复杂，还有子悦说闫欢是张函给她送过来的。”
<br/>
<br/>        “怎么又和张函扯上关系了？我就觉得我们子昂和张函扯上关系就没好事情，这一辈子怕是牵扯不清了。那闫欢和张函是什么关系？”
<br/>
<br/>        “说是同学关系，高中的同学。”
<br/>
<br/>        “青松，我去甸城的那个村子，在几年前被人围剿过，但不是当地的部门，是突然从上面过去的人，据说当时带走了一个人之后直接撤离，后续的事情都是交给当地的部门解决的。你去问问子悦，闫欢是什么时候到她那里的？”
<br/>
<br/>        “子澜，还是你聪明！”潘青松微微一笑。
<br/>
<br/>        “如果时间对得上，那么张函就很复杂，我们要提醒子昂不要再和张函有任何来往了。”
<br/>
<br/>        “他们孩子都有了，不来往可能吗？但是我和张函接触的这几年，张函没有任何问题，相反，他的工作很认真，铁面无私，他是为数不多的我很尊敬的人。他和卫红茹并称黑白双煞，一个是黑包公，一个是白青天。而且，我发现，他在秘密的调查一个人，也在秘密的调查他的老丈人陆家，只是我到现在都不知道他要调查的人是谁。”
<br/>
<br/>        “先把闫欢和张函的关系搞清楚。然后拿着这件事情可以侧面询问张函。”
