　　这一道无形龙爪，笼罩诸天，从天而降，向着韩圣横扫而去。
<br/>　　楚风眠一出手的强横，便是超出了在场所有人的意料之中。
<br/>　　那韩圣，更是最惊讶的一个。
<br/>　　他刚刚含怒出手，施展出的六绝神拳，已经是六绝宗的绝学了，现在却是被楚风眠看似随意的一掌直接打爆了。
<br/>　　这一尊半步天人境的绝代强者，面对那无形龙爪的横扫而来，脸色都是露出了几分慌张，想要向后退去，躲入到无尽的虚空之中。
<br/>　　可就是当他的身形即将踏入到虚空之中的一阵，一股无形的力量，突然拉扯在了韩圣的身上，将他的身躯从虚空之中拉扯而出，被这无形龙爪横扫。
<br/>　　“噗！”
<br/>　　韩圣身上的衣衫都被打的破碎，从他的口中都是吐出来的大口的精血。
<br/>　　“就这点实力，也敢出手？”
<br/>　　楚风眠轻蔑的看了一眼韩圣，冷笑说道。
<br/>　　这些圣龙秘境的老古董居然是敢如此轻视楚风眠，那楚风眠必须要给他们一个教训才行。
<br/>　　就先拿这韩圣开刀。
<br/>　　这效果的确是达到了，那些圣龙秘境的老古董，看着吐血的韩圣，有着一种见了鬼的感觉。
<br/>　　不敢相信眼中的一切。
<br/>　　身为半步天人境的韩圣，在这一次交手之下，直接是落了下风，甚至是现在就被直接打的吐血。
<br/>　　“这是什么武技？”
<br/>　　“始祖九式？小子你已经参悟出了始祖九式的奥秘？”
<br/>　　那万神龙猛然看向楚风眠，震惊道。
<br/>　　他已经看得出，刚刚楚风眠施展出的那一招无形龙爪，正是始祖九式。
<br/>　　“不错，始祖九式我已经参悟而出，这圣龙秘境在我的手下，才算是正统，你们这些老古董，现在乖乖听话臣服，我还可以考虑不杀你们，如何？”
<br/>　　楚风眠嘴角露出几分冷笑之色说道。
<br/>　　在这开口的一刻，楚风眠身上的气息也是在节节攀升起来，随手可能出手。
<br/>　　刚刚楚风眠一招直接将韩圣打的吐血，已经是在这些圣龙秘境老古董的心中留下了凶狠的印象，一股威严，自然显现。
<br/>　　“小家伙，得到些机缘奇遇，就敢如此嚣张？还让我们臣服于你？云尊都不敢说这种话！”
<br/>　　听到楚风眠嚣张无比的话音，那圣院宗的大长老王长老也是怒喝一声道。
<br/>　　“不错，太嚣张了，让我们臣服？”
<br/>　　“简直做白日梦！”
<br/>　　其他几位老古董也是纷纷开口道。
<br/>　　以这些老古董心中的尊严，根本不会选择臣服在楚风眠的麾下。
<br/>　　在圣龙秘境之中诞生出的天才，都对他们无比恭敬，还从来没有人敢让他们臣服。
<br/>　　他们这些老古董，统御圣龙秘境万年的时间，还从来没有人敢来挑衅他们的威严。
<br/>　　“是吗？不愿臣服，那就只有死路一条，正好我杀了你们，吞噬了你们的精血，也可以增添我的力量，帮助我掌控圣龙秘境，也算是让你们为圣龙秘境做最后一点的贡献。”
<br/>　　楚风眠声音冰冷无比。
<br/>　　不臣服便是死。
<br/>　　至于跟这些老古董谈判？楚风眠可没有那个兴趣，他要掌控圣龙秘境，便是整个圣龙秘境要臣服在楚风眠的麾下。
<br/>　　任何人也不可以插手其中，这些老古董留着，要是跟楚风眠为敌，反倒是祸害，不如一网打尽。
<br/>　　“楚风眠，你太嚣张了！”
<br/>　　“杀了我们？就凭你一个小家伙？”
<br/>　　“看来我们太久没有出手了，一个小家伙，都敢来挑衅我们的威严！不杀了此人，我们的威严何在！规矩何在！”
<br/>　　法理古帝盯着楚风眠，眼神之中露出冰冷的杀意到，这些人中，他对楚风眠的杀意最强。
<br/>　　“不用在废话了，刺杀此子！”
<br/>　　那韩圣恢复了几分伤势，盯着楚风眠怒声喝道。
<br/>　　在这众目睽睽之下，他被楚风眠直接打吐血了，丢尽了脸面，现在只有杀了楚风眠，才可以缓解他心中的屈辱。
<br/>　　轰！
<br/>　　两道身影突然动了，强横的灵力爆发而出，向着楚风眠笼罩而来。
<br/>　　法理古帝，韩圣，两位半步天人境强者，同时出手了，纷纷施展出了压箱底的杀招，向着楚风眠轰击而来。
<br/>　　两股力量，一股蕴含毁灭之力，一股蕴含规矩法理之力，同时轰击在了楚风眠的面前，要将楚风眠直接轰杀。

<br/>　　“联手又能如何？”
<br/>　　楚风眠嘴角轻蔑一笑，突然一掌打出，旋即无形龙爪再度出现，向着法理古帝，韩圣二人横扫而去。
<br/>　　“小心！”
<br/>　　韩圣已经是吃过一次这无形龙爪的亏，急忙出言提醒道。
<br/>　　那法理古帝也是身形闪烁，重新躲入到了阵图之中，借助阵图之力，来抵挡这一招。

<br/>　　那法理古帝头顶之上的阵图，都是在疯狂的震动着，在这巨大龙爪的轰击之下，都是打出了无数裂痕。
<br/>　　砰！
<br/>　　躲在阵图之中的法理古帝，韩圣二人，一下子是被打飞了出去，二人都是吐出了一大口的鲜血。

<br/>　　他本来打算杀了楚风眠，来弥补他刚刚损耗的精血损失，可现在他又一次是被龙爪轰飞出去，又是吐出了一大口的精血，伤上加伤。
<br/>　　这韩圣身上的境界，都开始不稳了起来，在这样下去，他的境界可能都会掉落到九劫古帝。
<br/>　　而且一旦是境界掉落，那这韩圣可是一辈子都无法再度踏入半步天人境了。
