　　至少，短短一个月的时间，他的实力就已经提升到了一个令人不敢置信的地步。
<br/>　　就像眼前这些，所谓搜索队的成员，在自己没得到系统之前，哪一个在自己面前，不是心高气傲的存在。
<br/>　　但是现在，这些人都趴在地上，和狗一样请求自己的宽恕，他们的生死，也都在自己的一念之间。
<br/>　　如果要用词语来形容刘哔现在的感觉，扬眉吐气，这个词再也适合不过了。
<br/>　　“哦，这样啊，原来是个误会啊。”语气稍稍缓和了一下，刘哔将孙强的尸体丢到一边，从半空中缓缓降落了下来。
<br/>　　看到了生存的希望，众人心中立即大喜，连连开口，发出了一连串附和的声音。
<br/>　　脸上带着笑意，刘哔缓缓走到众人面前，随着他的接近，几人脸上的表情都有些惶恐。
<br/>　　毕竟他们所面对的，是一个凶恶无比的高阶丧尸，换成任何人，在这种环境下都会有很大的压力吧。
<br/>　　“但真不巧，我正好肚子有些饿了，就拿你们填填肚子吧。”在众人前方站定，刘哔猛然出手，拉出了一张用灵气丝线组成的大网，向跪伏在地上的几人笼罩而去。
<br/>　　虽然他的出手没有任何预兆，但众人还是立即反应了过来，毕竟先前，他们也都在一直戒备着。
<br/>　　但是实力之间的差距，却是没有那般容易弥补的，几人的激烈挣扎，并没给刘哔带来多大的影响。
<br/>　　猩红色的灵气大网缓缓收拢，几分钟的时间，就将场中的幸存者们搅成了碎块，一股浓郁的血腥气从空中扩散开来，让人不由得食欲大增。
<br/>　　但耽搁了这么一会，天色已经完全黑下来了，再继续拖延下去，刘哔有些担心自己真的进不去基地。
<br/>　　自己的弟弟还眼巴巴的等着他回去呢。
<br/>　　有些可惜的看了一眼地上的血食，刘哔摘下面具，仔细将上面的血迹擦掉之后，收入到了空间戒指之中。
<br/>　　随后，刘哔发动了车辆，向来时道路折返而去。
<br/>　　随着他的离开，此地已经逐渐安静了下来，大约半小时之后，从远处冲过来一个匆忙的身影，蹲在地上的一堆碎肉块旁边，似乎在寻找着什么。
<br/>　　片刻之后，这道身影从地上站起，顺手扛起孙强已经干枯的尸体，消失在了茫茫夜色之中。
<br/>　　……
<br/>　　紧赶慢赶的，刘哔总算是在关闭城门之间进入了基地，在经过几个小时的例行紧闭之后，刘哔心急火燎的回到了家里。
<br/>　　此时的时间已经接近午夜了，家里的环境有些漆黑，刘莽在床上侧躺着，似乎是听见了声音，翻转了一下身体。
<br/>　　“哥，你回来了。”明亮的眼睛在黑暗里亮晶晶的，特别瞩目。
<br/>　　轻轻‘嗯’了一声，刘哔将手中提着的麻袋放在地上，自己一屁股坐在了土炕上。
<br/>　　“这么晚了还不睡觉，等我啊？”半躺在床上，刘哔舒展了一下身体，转过头来，和刘莽互相对视着。
<br/>　　黑暗中的视线并不清楚，但他还是注意到了刘莽嘴角露出的一抹笑意。
<br/>　　“扯淡，饿的睡不着，怎么可能为了等你。”带着有些不屑的语气，刘莽从床上坐了起来，经车熟路的拿起刘哔所带回来的麻袋，从里面掏出一包过期的薯片拆开。
<br/>　　似乎是真的饿了，刘莽吃东西的动作很是着急，刘哔坐在一旁静静的看着，有些宠溺的嘱咐了一句：“慢点吃，里面有水，别不小心噎死了。”
<br/>　　含糊不清的嘟囔了一句，刘莽突然停止了动作，伸手将麻袋里面的食物倒出来一般，随后将麻袋扎了起来。
<br/>　　“明天给秦老太太送去一般吧，这一周她可是没少偷摸给咱家送东西吃。”语气中带着哀求，他征求似得看着刘哔。
<br/>　　莞尔一笑，刘哔也没有反对，对现在的他来说，这种食物要多少有多少。
<br/>　　得到同意之后，刘莽有些开心的一笑，将剩下的食物藏在土炕下面的暗格里，随手继续吃起了手中已经拆开的薯片。
<br/>　　他没有拿水，虽然有些噎得慌，但他知道这些东西有多么来之不易。
<br/>　　“一包薯片怎么够吃，多吃一些，不够的话，我明天继续出去找。”看着刘莽小心翼翼进食的场景，刘哔皱起了眉头。
<br/>　　虽然只是一包过期的薯片，但在刘莽手上，却是一副珍贵无比的样子。
<br/>　　吃薯片的动作一顿，刘莽表情凝固了下来：“别，这不是还有食物吗，够吃很长时间了，别处去冒险了。”
<br/>　　自己哥哥的实力，他心中再也清楚不过了。
<br/>　　想必就这些东西，应该都是冒了极大的风险，他不想哥哥做这种危险的事情。
<br/>　　甚至他有时候都在想，如果没有他的拖累，估计哥哥要比现在过得轻松许多。
<br/>　　看着刘莽担心的眼神，刘哔又如何不知道自己弟弟在想什么，但是很多东西，他也没办法去和刘莽说明。
<br/>　　并不是信任的问题，就算任何人出卖自己，刘莽也是绝对不可能背叛的。
<br/>　　在这个世界上，刘莽是唯一值得自己去相信的人。
<br/>　　这不仅仅是血缘关系，事实上很多亲兄弟都会为了一瓶水而反目成仇，但他们不同。
<br/>　　两个人在末世一起生存了十年，一起度过了无数个困境，虽然也会发生争吵，可是出现问题的时候，谁都没有过想要扔下对方逃跑的想法。
<br/>　　甚至刘莽的右手臂，也是为了给自己挡住一头变异兽的攻击而断掉的。
<br/>　　但即便是再信任，有些事情，刘哔也不敢对刘莽全盘托出。
<br/>　　要知道，这个世界上的异能者，有很多手段，可以直接读取别人脑海中的记忆，出卖这种东西，很多时候都是迫不得已。
<br/>　　他无法接受，哪怕任何一点有可能将自己系统或者能力暴露出去的因素，这可是他安身立命的根本所在。
<br/>　　对此，刘哔面对自己弟弟的劝阻，也只能无奈的张了张口，最终却也还是什么都没有说，只是轻轻拍了拍刘莽的后背。
<br/>　　时间已经很晚了，该睡觉了，否则的话，怕自己的身份暴露出去，谨慎起见，刘哔也没有像往常一般修炼。
