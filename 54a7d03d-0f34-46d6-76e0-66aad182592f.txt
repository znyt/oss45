　　苏莫的声音如雷霆降世，传遍全场，滚滚涌荡。
<br/>　　他不知道会武能不能一次性挑战多人，但想必只要都愿意，也没有人会阻拦。
<br/>　　整个武斗场，所有人都是瞪圆了眼眸，不可思议的望着苏莫。
<br/>　　卧槽！
<br/>　　这个苏莫，这是蹬鼻子上脸了，说他狂妄这还真的是狂妄了，居然要一次性挑战四人，以一战四。
<br/>　　之前苏莫虽然自大，但有强大的实力依仗，众人倒是不认为苏莫多狂，但现在，此言一出，那就是真的太狂了！
<br/>　　真枯半圣、冯长老等道宗的长老，则是脸色一变，心中暗骂苏莫白痴。
<br/>　　以苏莫的实力，只要正常挑战，打败所有对手都不成问题，到时候轻松获得第一名。
<br/>　　但现在，苏莫居然要一次性挑战四大妖孽，这不是白痴吗？
<br/>　　明明有得到第一名的机会，却是做出这么不明智的事情，一对四，就算苏莫的战力再强，也是败多胜少啊！
<br/>　　而此刻，最为愤怒的人，莫过于炎澜天、卓剑豪等四人了。
<br/>　　他们四人，哪一个不是妖孽，均是武帝境之内的巅峰人物，现在被苏莫如此藐视，俱是心中怒火滋生。
<br/>　　“简直是不可理喻！”
<br/>　　“嚣张狂妄之徒！”
<br/>　　“你是白痴吗？我们四人会联手对付你！”
<br/>　　“呵呵，你还没有这个资格！”
<br/>　　炎澜天四人冷笑连连，但是却并没有上场，笑话，他们四人都是妖孽天才，心高气傲，怎么可能联手去对付苏莫？
<br/>　　如此的话，就算是胜了，也胜之不武！
<br/>　　苏莫面色淡淡瞥了四人一眼，随即转头望向阴泽长老，道:“阴泽长老，请问会武之中，能否一次性挑战四人？”
<br/>　　苏莫的话音朗朗传出，让得所有人都是面色古怪。
<br/>　　这个苏莫居然请示阴泽长老，这是要来真的啊，真是让人无法理解。
<br/>　　阴泽长老面色淡淡，听闻苏莫之言，嘴角露出了一丝笑容，道：“如果他们愿意，本长老不反对！”
<br/>　　本来按照会武的规则，这种一次性挑战多人的事情，根本是不允许的，不然谁能能这样做了，规则也就废了。
<br/>　　但现在，阴泽长老却是开了特例，因为他也想看看，苏莫的战力到底达到了什么地步。
<br/>　　他感觉苏莫的实力，只是展露了冰山一角，完全没有展现最强的战力。
<br/>　　苏莫闻听阴泽长老之言，微微一笑，继续问道：“那敢问阴泽长老，我若是一举击败了他们四人，能否一次性获得四分？”
<br/>　　本来苏莫也没想过要一次性击败炎澜天四人，但既然四人都跳出来了，他就想着一次性解决了，免得一次又一次的挑战这么麻烦。
<br/>　　“可以获得四分！”
<br/>　　阴泽长老点了点头，略一沉吟，又道：“不过，你若是败了，就要被直接淘汰，不会再有参加的资格！你可明白？”
<br/>　　阴泽长老心中有自己的考虑，他若是直接同意了苏莫的要求，难免会让人非议，认为他偏袒苏莫。
<br/>　　所以，他便加了一个条件，败了就要被淘汰。
<br/>　　这虽然是一个条件，看起来完全是苏莫需要付出的代价，但实际上却是刺激炎澜天四人，阴泽长老也希望四人能与苏莫一战。
<br/>　　果然，炎澜天四人闻言，眸光一闪，苏莫若是被淘汰了，那他们获得第一的机会，就能大大的增加了啊！
<br/>　　一旦获得第一，好处可是非常的大！
<br/>　　苏莫的实力之强，这是毋庸置疑的，只要苏莫不被淘汰，他们能得到第一的机会，非常的小。
<br/>　　炎澜天四人互相对视了一眼，每个人均是从其它人眼中看到了一丝迟疑之色，他们本来是不可能一起联手挑战苏莫的，但若是真的能将苏莫淘汰出局，倒是可以拉下脸面一战！
<br/>　　毕竟，相比于第一名的奖励，丢掉一点面子也无不可！
<br/>　　“明白！”
<br/>　　听了阴泽长老的话，苏莫微微的点了点头。
<br/>　　淘汰？
<br/>　　这是不可能的事情！
<br/>　　随即，苏莫再次转头看向炎澜天四人，口中顿时响起一声暴喝：“你们四个废物，如果是男人的话，就下来一战！”
<br/>　　苏莫的喝声，化为滚滚雷霆，轰然炸响，震荡了整个武斗场。
<br/>　　他是故意为之，激怒炎澜天四人应战，既然可以一战解决四人，一战得到四分，那还等什么？
<br/>　　听闻苏莫的喝声，炎澜天四人的脸色彻底的黑了下来，一瞬间便难看无比。
<br/>　　他们四人都是妖孽，自修炼武道以来，就是天才中的天才，何时被人骂过废物？
<br/>　　一瞬间，四人都怒了，心中怒火汹涌，无法抑制。
<br/>　　“简直是找死！”
<br/>　　“你必败无疑！”
<br/>　　“可恶，今日不打败你，我就不姓卓！”
<br/>　　“混账东西，你会后悔的！”
<br/>　　炎澜天、灵枫、卓剑豪、雷云皓四大妖孽天才，纷纷怒喝出声，而后纷纷身形一动，齐齐向战台上飞了过去。
<br/>　　嗖！嗖！嗖！嗖！
<br/>　　四人身影闪烁，瞬间便降临在了战台之上，和苏莫相隔数千里而立。
<br/>　　四人一字排开，个个身上气息强大，恐怖的气势如同山呼海啸，威不可挡。
<br/>　　整个武斗场瞬间死寂了下来，所有人都是暗暗心惊，居然真的要一挑四了！
<br/>　　这一下，真的是巅峰对决了，苏莫还能如此强势吗？
<br/>　　一挑四，就算苏莫的战力再强大，也很难胜利了吧！
<br/>　　虽然之前的苏莫，表现的非常强势，打败幽狼和扬天成，都是没有什么悬念，显得颇为轻松。
<br/>　　但是一次性对上四人，若是没有远超四人的实力，根本不可能战胜四人。
<br/>　　道宗的人全部都是心有忐忑，这一战，苏莫若是胜了，那会武第一名就是板上钉钉的事情了，但若是败了，就彻底完了，直接被淘汰出局。
<br/>　　“这小子！”冯光的脸色亦是凝重异常，心中暗骂苏莫鲁莽。
<br/>　　此时，除了道宗之人忐忑外，还有不少人心中忐忑，比如幽狼、扬天成等被苏莫击败的人。
<br/>　　当然他们不是怕苏莫败了，而是完全相反，怕苏莫胜了！
<br/>　　“苏莫，你必败无疑！”扬天成面色有些狰狞，眸中杀机凌厉，心中狂吼。
<br/>　　他恨不得当场斩杀苏莫，以泄心头之恨，但是这是不可能的事情，现在他只希望苏莫惨败，被淘汰出局。
