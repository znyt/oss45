　　“不对劲。”萧逸皱着眉。
<br/>　　“罡风破散，却吸力不消。”
<br/>　　萧逸自语着，脑海中思索着关于这片极寒裂缝之事。
<br/>　　千万年前，这里便是险地，便已经是裂缝。
<br/>　　裂缝，深不见底，如同一个深渊。
<br/>　　但那时的罡风，并不如现今这般恐怖。
<br/>　　萧逸在冰尊者的记忆中，得到了不少信息。
<br/>　　当然了，时间太久远的，非常模糊。
<br/>　　只有近数百年的，比较清晰。
<br/>　　但这并不妨碍萧逸推测出以前的事情。
<br/>　　因为，这近数百年来，冰尊者，每日都在做着同样的事——修炼武魂力量。
<br/>　　不错，修炼武魂力量。
<br/>　　冰尊者，曾是一位魂师。
<br/>　　魂师，作为武者的其中一个职业，自然也是存在修炼的。
<br/>　　萧逸从记忆信息中，得到一本功法，名唤‘魂念诀’。
<br/>　　这是一本属于魂师的修炼法诀。
<br/>　　同样，品阶不明。并非冰尊者所创，却是冰尊者一直修炼之法。
<br/>　　同时，还有一门名唤‘冰暴噬魂’的魂技。
<br/>　　冰尊者的记忆信息，多有残缺。
<br/>　　唯独冰尊圣纹诀、魂念诀、冰暴噬魂，这三样修炼之法，极其完整。
<br/>　　毫无疑问，这三样，都是他所修炼的法门，亦是他拥有的最强大的法门。
<br/>　　冰尊圣纹诀，自不必多说。
<br/>　　而魂念诀。
<br/>　　萧逸早就猜测，冰尊者的灵识中，绝对带有武魂力量。
<br/>　　那抹幽光，已是虚无之物，根本无法再有武道修炼。
<br/>　　唯一能修炼的，只剩下魂师的手段。
<br/>　　而冰尊者这数百年来，一直在做的，便是通过灵识，施展‘冰暴噬魂’。
<br/>　　利用这一门魂技，不断释放冰暴。
<br/>　　不断增加着极寒裂缝内的罡风强度。
<br/>　　萧逸猜测着，既然这数百年都是如此。
<br/>　　很可能以前的岁月，这千万年来，冰尊者一直都在这样做。
<br/>　　这才让得极寒裂缝之内的罡风，强到这个地步。
<br/>　　千万年都在做着同样一件事，萧逸不得不佩服冰尊者的毅力。
<br/>　　但同时，这又是冰尊者无可奈何之举。
<br/>　　他没有其它攻击手段。
<br/>　　他只剩下‘冰暴噬魂’这一魂技。
<br/>　　他必须保证自己拥有足够的实力去击杀他的‘猎物’。
<br/>　　之后才能考虑是否‘夺舍’。
<br/>　　千万年时间，他将极寒裂缝的罡风，都化为了自己可掌控的力量。
<br/>　　“原来如此。”萧逸心头思索一番，已明白了过来。
<br/>　　刚才自己的一剑，劈散了罡风。
<br/>　　但仍旧有一股吸力，将自己吸回洞中。
<br/>　　这股吸力，来自于极寒裂缝中铺天盖地的‘魂力’。
<br/>　　简单来说，极寒裂缝中的罡风，已经不再是正常的罡风了。
<br/>　　而是带有‘魂力’属性的‘冰暴罡风’。
<br/>　　萧逸走到洞穴边缘，看了一眼，脸色愈发凝重。
<br/>　　冰尊者这老怪物，千万年来，在极寒裂缝中不断施展‘冰暴噬魂’。
<br/>　　这里的冰暴罡风，特别是‘风眼’位置范围之内的强度，怕是已经超越了地极境的范畴。
<br/>　　难怪极寒裂缝会是极寒之地的第一险地。
<br/>　　冰尊者能操控这里头的所有罡风，恐怕就是当年的暴雪庄主那等层次的至强者来了，都远不是他的对手。
<br/>　　萧逸暗暗皱眉，若想靠自身武力，逃出极寒裂缝，似乎成了不可能之事。
<br/>　　“魂力攻击，无形诡异。”
<br/>　　“但冰尊者的灵识毕竟已经消失了，这里的冰暴罡风成了无主之物。”
<br/>　　“若是懂得魂师之手段，或许能逃出去也不一定。”
<br/>　　萧逸自语着。
<br/>　　话音落下，萧逸没有浪费时间，修炼起‘魂念诀’来。
<br/>　　同样的，‘魂念诀’对萧逸而言，并没有太大的难度。
<br/>　　萧逸一边修炼着，一边内视体内小世界情况。
<br/>　　不多时，体内两大武魂周遭，出现了一股诡异的力量。
<br/>　　这便是魂力。
<br/>　　萧逸稍稍对比了一下，这种力量的层次，大概和修罗之力差不多。
<br/>　　时间渐渐过去，两大武魂周遭的魂力，越发浓郁。
<br/>　　同时，萧逸也在做着思索。
<br/>　　上古时期，魂师、炼药师、阵法师等等，各种强大的职业并存。
<br/>　　武者，武道修炼为主。
<br/>　　各种职业为辅。
<br/>　　那时的武者，必是个个战力强悍。
<br/>　　那时的炎龙大陆，也必是精彩纷呈。
<br/>　　只可惜，到了现在，只剩下炼药师一门传了下来。
<br/>　　那种精彩的世界，让得萧逸这个向来不安分的人，有着些许狂热。
<br/>　　“呼。”半个时辰后，萧逸停下了修炼，并收敛了脑海中的多余想法。
<br/>　　现今最重要的，还是逃离此地。
<br/>　　“嗯？”萧逸忽然眉头一皱。
<br/>　　小世界内，原本已经修炼出一定的‘魂力’了。
<br/>　　魂力，分别覆盖在两大武魂之上，光芒闪烁，甚至奇异。
<br/>　　可忽然，冰鸾剑武魂周遭的魂力，瞬间消失。
<br/>　　“魂力也能作为你的修补力量？”萧逸自语一声，微微有些惊讶。
<br/>　　不过幸好，控火兽周遭的魂力，并未消失。
<br/>　　否则，萧逸真不知道还有什么办法能逃离极寒裂缝。
<br/>　　“喝。”萧逸低喝一声。
<br/>　　瞬间通过控火兽，使用‘魂力’。
<br/>　　这只是魂念诀中最简单的一种使用魂力的办法。
<br/>　　一道无形的波纹，陡然间从萧逸体内迸发。
<br/>　　半晌，这股无形的力量无影无踪。
<br/>　　萧逸看了眼一旁的洞穴墙体。
<br/>　　这些墙体，是厚厚的冰层。
<br/>　　萧逸微微伸手弹了弹。
<br/>　　霎那间，咔…咔…咔…
<br/>　　冰层掉落一大片。
<br/>　　冰层表面，看似完好无损；但实则里头数米之深的冰层，早已化为了齑粉。
<br/>　　“魂师的攻击手段，果真诡异。”萧逸吃了一惊。
<br/>　　下一秒，萧逸身影一闪，跃出洞穴。
<br/>　　顺带一提，风雪剑主的尸身，早被他放入了乾坤戒。
<br/>　　他要再次尝试能否离开了。
<br/>　　事实上，‘魂念诀’他并未完成参悟，里头许多魂师手段，他也并未完成修习。
<br/>　　但他不想再耽误时间。
<br/>　　一出洞穴，果然，无数凛冽罡风吹袭而来。
<br/>　　一股吸力，要将他吸回洞中。
<br/>　　这时，萧逸已明显感觉到，这周遭存在的‘魂力’。
<br/>　　“破。”萧逸低喝一声。
<br/>　　一阵无形力量，陡然间从身上爆发，抵消了冰暴罡风中的魂力。
<br/>　　萧逸的身影，快速飞跃。
<br/>　　堪堪在魂力耗尽前，飞出了近千米范围。
<br/>　　刚才短短的修炼时间，他修炼出的魂力，本就不多。
<br/>　　但，已经足够了。
<br/>　　千米范围外，已不再是‘风眼’位置。
<br/>　　吸力，以及罡风强度，大大减弱。
<br/>　　紫炎火翼，猛地一震，冲天飞起。
<br/>　　万米高度，对他而言，不过转瞬之间。
<br/>　　“冰承天，你跑不掉的。”萧逸大喝一声。
<br/>　　一道火光，瞬间从极寒裂缝冲出。
<br/>　　随后，光芒往着某个方向疾速飞行。
<br/>　　那个方向，正是极寒之地前往冰武王国的必经方向。
