<br/>?    凌仙看过陈明瞬间突破三层，已经惊为天人，要知道以他的悟性，冠绝整个南境，修了三年，还是没有修成，而那个妖王，一瞬间便是三层。
<br/>    
<br/>        凌仙回到自己的院子中，开始打算想办法拜陈明为师。
<br/>    
<br/>        忽然身旁响起一个苍老的声音，“你想要拜那人为师？”
<br/>    
<br/>        凌仙看向身旁，却是发现一个浑身黑衣，半边头发白，半边头发黑的老人，不过这个老人倒是浑身如同虚幻一般，就像是一个灵魂。
<br/>    
<br/>        凌仙看向老人的面容和样子，忽然想起了自己家祖师的画像，“你是，我凌家祖师！”
<br/>    
<br/>        老人笑着点点头，道：“不错，我便是你的祖宗，不过我仙躯已经损坏，只能通过你手上的这枚戒指上的吸收你的法力才能维持自己的存在。”
<br/>    
<br/>        凌仙道：“那我突破到入道境的时候，也是你吸收了我的法力。”
<br/>    
<br/>        老人点点头，道：“不错。”
<br/>    
<br/>        凌仙心知是自己的老祖害的自己变成如今这番模样，有苦说不出，“老祖你害的我好惨。”
<br/>    
<br/>        老人笑道：“你懂什么，你之前修行的功法，在整片大陆之上，虽然算得上是不错的功法，但是在仙人领域，根本不值一提，唯有道经，才是正途！”
<br/>    
<br/>        “曾经我成就仙人之躯，飞升仙界，便是见过有人以超脱境的修为，搏杀仙人，而那人，修行的便是道经。”
<br/>    
<br/>        凌仙心中一喜，道：“那既然老祖想要我修行道经，老祖可能教我道经？”
<br/>    
<br/>        老人露出一抹自信的微笑，然后道：“当然，不能！”
<br/>    
<br/>        两人都没有注意到的是，一只漆黑的乌鸦，在此时落在了窗台之上。
<br/>    
<br/>        凌仙道：“那老祖你还要我修行道经。”
<br/>    
<br/>        老人道：“天道是公平的，想要得到回报，便需要付出，以我的悟性，根本无法参透全部的道经，但是以你的悟性，未必不能，只是需要的时间有些长而已。”
<br/>    
<br/>        凌仙喜道：“那我的悟性，参悟道经需要多久。”
<br/>    
<br/>        老人扫了凌仙一眼，道：“以你的悟性，怎么也得要上个两三百年吧。”
<br/>    
<br/>        凌仙的脸瞬间就苦了，“等于说我还没修成，就已经老死了......”
<br/>    
<br/>        老人挥了挥手，然后道：“不然，你不是有更好的师父吗？”
<br/>    
<br/>        凌仙想到陈明修炼道经的那一幕，“老祖，你说的是旁边的那位妖王？”
<br/>    
<br/>        老人笑了笑，道：“他并不是妖王，而是一个人，不对，应该说，是一位仙，一位轮回仙。”
<br/>    
<br/>        凌仙惊讶道：“仙，轮回仙？那是什么境界？”
<br/>    
<br/>        老人道：“仙人本就已经是夺取天地造化于一身，只存在于传说中，但是轮回仙面前，仙人就如同蝼蚁一般渺小，只有仙王巅峰的存在，才能被称作轮回仙。到了他们这个境界，众生如同蝼蚁，天地不过蜉蝣，岁月不过眨眼，而他们想要更强，便需要重新走一遍修行路，感悟天地间的大道，若是我没有猜错的话，你见过的这位存在，应该就是一位轮回仙。没有想到，我活着的时候，仙人之躯，都没有见过轮回仙，甚至连画像，足迹都没有见过，如今死了之后，却是见到了一位真正的轮回仙！”
<br/>    
<br/>        凌仙的小嘴微张，他怎么也没有想到，曾经见过的这位妖王，居然会是这般伟岸不可言语的存在！
<br/>    
<br/>        “那我怎么可能入得了他的眼，此间的差距，根本不可估量！而且我道宫以碎，如何再次踏上修行路。”
<br/>    
<br/>        老人伸手拍了拍凌仙的头，身体是虚幻的，并没有拍到，尴尬地一笑，然后道：“痴儿，你怎么这么傻，这般掌握了天地伟力的伟岸存在，哪里还会在意天赋这些东西，他们收徒，看得是心性，唯有心诚，才能入得了他的法眼。而且轮回仙此行乃是为了感悟天地大道，人情冷暖，就要亲身去体会，你未必没有这个机会。”
<br/>    
<br/>        凌仙目光逐渐变得坚毅起来，“无论有多难，我都要拜他为师，重新踏上修行路！”
<br/>    
<br/>        陈明挥动翅膀，回到了自己的行宫之中，莫名其妙当了一位轮回仙，仙人的脑补能力，不去写小说还真是屈才了！
<br/>    
<br/>        不过，戒指里面的老爷爷，我喜欢这个设定！
<br/>    
<br/>        第二天一早，天色有些阴沉，陈明看着浓厚的铅云，应该是要下雪了，不知道卓青瑶这个小妮子会不会冻着，吃的好不好，睡得怎么样，会不会想念我的坑，唉，为人父母，就是得操这些心啊。
<br/>    
<br/>        索性今天无事，要等着凌仙这个小家伙来拜师，得找点活动消磨一下时间，最近无聊的时候把纸牌弄出来了，瞬间火遍了整个山头，这可比骰子有意思多了，大大小小的大妖都会玩，就这么愉快地决定了，找熊大熊二来斗·地主吧！陈明推开门，却是看见了凌仙跪在了门前，眼皮微微一颤，“你跪在这里做什么？”
<br/>    
<br/>        凌仙想起老祖的告诫，行走在人间感悟天地的轮回仙最为忌讳的就是别人点破他的身份，连忙道：“凌仙，想要拜妖王大人为师！请妖王大人收下弟子！”
<br/>    
<br/>        俗话说的好啊，得不到的永远在骚动，太容易得到的必定不会珍惜，陈明也不会直接答应，自然是要将他‘轮回仙’的排面拉满啊！
<br/>    
<br/>        扫了一眼凌仙，然后道：“跪着就有用吗？我可不会收你为徒，既然你喜欢跪着，那就跪着吧。”
<br/>    
<br/>        陈明叫来了熊大熊二，在大厅中升起了火，三个人开始斗·地主，陈明最近也是手痒，好久没玩过斗·地主了，有些回味以前的日子的意思，上手之后和熊大熊二玩得不亦乐乎。
<br/>    
<br/>        门没关，不时还看得到跪在门口的凌仙，三个时辰过去，天色越来越暗，寒风凛冽，冷风刺骨，天空之中飘起一朵朵雪花来，熊大熊二看了一眼凌仙，“妖王大人，冻坏的肉好不好吃啊？”
<br/>    
<br/>        陈明疑惑道：“你们两个憨货问这个做什么？”
<br/>    
<br/>        熊大指了指门口的凌仙，陈明扫了一眼，和凌仙对视了片刻，确定自己给出了自己这辈子以来最深邃的眼神，然后道：“他喜欢跪在门口，就让他跪着吧。”
<br/>    
<br/>        这一次斗·地主足足玩了三天，陈明有输有赢，倒是也没输太多，而且玩的这么小，对于他来说九牛一毛。
<br/>    
<br/>        天地之间吗，一片银装素裹，恍然一片白色的世界，大雪在地面踮起一尺多高，而跪在门口的凌仙，已经是被冻成了一个雪人，他不过只是闻道境一层，甚至还没到，和凡人区别并不大，而且还是一个少年，他的身躯已经失去了知觉，我快要死了吧。
<br/>    
<br/>        不过他看向陈明的眼神依旧坚定，如同有着火焰在双瞳之中燃烧。
<br/>    
<br/>        不过现在，这火焰快要熄灭了，因为他可以感觉到，自己已经开始迷糊了，他要失去意识了，他快要死了。
<br/>    
<br/>        这个时候，在他的眼中，忽然出现一片黑色的羽毛，而后是黑色羽毛织成的长袍，耳中传来缥缈的声音，“自今日起，你便是我门下二弟子！”
